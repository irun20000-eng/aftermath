01. 개념 구성 (Conceptual Foundation)

1 등비수열의 합 아이디어

첫째항이 \(a\), 공비가 \(r\)인 등비수열의 첫째항부터 제\(n\)항까지의 합 \(S_n\)은 다음과 같다.

\(S_n = a + ar + ar^2 + \cdots + ar^{n-1}\)

양변에 공비 \(r\)을 곱하면

\(r S_n = ar + ar^2 + \cdots + ar^{n-1} + ar^n\)

두 식을 변끼리 빼면 가운데 항들이 모두 소거된다!

\((1-r)S_n =\) \(a(1-r^n)\)

2 등비수열의 합 공식

위에서 얻은 식 \((1-r)S_n = a(1-r^n)\)의 양변을 \(1-r\)로 나누어 공식을 완성한다.

\(r \neq 1\) 일 때

\(S_n = \frac{a(1-r^n)}{1-r}\)

또는 \(S_n = \frac{a(r^n-1)}{r-1}\)

\(r = 1\) 일 때

\(S_n =\) \(na\)

⭐ 도전 문제 1 — Q4 Sₙ 등비 변환 본질 확장

📖 출처: 수능특강 수학Ⅰ · 등차·등비 · 기본 8

난이도 ★★★★ (학습지 max+3, 12분) · 핵심: 등차중항 + 부호 + Sₙ-c 등비 변환 (수능 29·30번급)

등비수열 \(\{a_n\}\)에 대하여 \(a_1 a_2 < 0\)이고, 세 수 \(a_2, a_1, a_3\)이 이 순서대로 등차수열을 이룬다. 수열 \(\{a_n\}\)의 첫째항부터 제\(n\)항까지의 합을 \(S_n\)이라 하자. 수열 \(\{S_n - 1\}\)이 등비수열일 때, \(S_7\)의 값을 구하시오.
📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 수Ⅰ 단독 풀이 흐름 (4단계):
  1. 등차중항 → r 후보: \(2 a_1 = a_2 + a_3 = a_1 r + a_1 r^2\) ⇒ \(r^2 + r - 2 = 0\) ⇒ \((r-1)(r+2)=0\) → \(r = 1\) 또는 \(-2\)
  2. 부호 조건으로 r 결정: \(a_1 a_2 = a_1^2 r < 0\) ⇒ r < 0 ⇒ \(r = -2\)
  3. \(S_n - 1\)이 등비 → \(a_1\) 결정:
    • \(S_n = \dfrac{a(1-(-2)^n)}{3} = \dfrac{a}{3} - \dfrac{a}{3}(-2)^n\)
    • \(S_n - 1 = \dfrac{a-3}{3} - \dfrac{a}{3}(-2)^n\) — 형태 \(A r^n + c\)에서 상수항 \((a-3)/3 = 0\)이어야 \(A r^n\) 등비 형태
    • ⇒ \(a_1 = 3\). 그러면 \(S_n - 1 = -(-2)^n\) → 공비 \(-2\)의 등비수열 ✓
  4. \(S_7\) 계산: \(S_7 = \dfrac{3(1-(-2)^7)}{3} = 1 - (-128) = \boxed{129}\)

핵심: 학습지 Q4("Sₙ에서 첫째항부터 등비")의 본질을 4단으로 확장 — 등차중항 → 부호 → Sₙ-c 변환 → 상수항 소거. \(S_n - c\)가 등비 ⟺ \(S_n\)이 \(A r^n + c\) 꼴.

※ 자세한 풀이는 수능특강 해설지 기본 8 참조.

⭐ 도전 문제 2 — Q3 인수분해 심화

📖 출처: 수능특강 수학Ⅰ · 등차·등비 · 유제 9

난이도 ★★★ (학습지 max+2, 8분) · 핵심: 1+r+r²+r³=(1+r)(1+r²) 인수분해 + 다중 조건 연립

공비가 \(1\)보다 큰 등비수열 \(\{a_n\}\)의 첫째항부터 제\(n\)항까지의 합을 \(S_n\)이라 하자. \(2 S_4 = 5(a_2 + a_3)\), \(S_5 = a_6 - 1\)이 성립할 때, \(S_6\)의 값은?
📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 수Ⅰ 단독 풀이 흐름 (3단계):
  1. 인수분해로 r 결정:
    • \(2 S_4 = 2 a (1 + r + r^2 + r^3) = 2 a (1+r)(1+r^2)\)
    • \(5(a_2 + a_3) = 5 a r (1 + r)\)
    • \((1+r) \neq 0\)로 약분 → \(2(1 + r^2) = 5 r\) ⇒ \(2 r^2 - 5 r + 2 = 0\)
    • \((2r-1)(r-2) = 0\) → \(r = 2\) (\(r > 1\))
  2. \(S_5 = a_6 - 1\)에서 첫째항 결정:
    • \(S_5 = a \cdot (2^5 - 1)/(2-1) = 31 a\), \(a_6 = 32 a\)
    • \(31 a = 32 a - 1\) ⇒ \(a = 1\)
  3. \(S_6\) 계산: \(S_6 = \dfrac{1 \cdot (2^6 - 1)}{2 - 1} = \boxed{63}\)

핵심: 학습지 Q3의 \(1 - r^6 = (1-r^3)(1+r^3)\) 인수분해 발상을 \(1 + r + r^2 + r^3 = (1+r)(1+r^2)\)로 확장. 등비합 형태에서 인수분해는 항상 도구상자에 두자.

※ 자세한 풀이는 수능특강 해설지 유제 9 참조.
