00. 핵심 개념 구조화 (Review)

이번 중단원에서 배운 핵심 공식과 원리들을 다시 한 번 기억 속에서 꺼내어 완성해 보자.

\(\Sigma\)의 성질과 주의점

\(\sum_{k=1}^{n} (ca_k + b_k) =\) \(c\sum a_k + \sum b_k\) | 단, \(\sum_{k=1}^{n} c =\) \(cn\)

자연수의 거듭제곱의 합
  • \(\sum_{k=1}^{n} k^2 =\) \(\frac{n(n+1)(2n+1)}{6}\)
  • \(\sum_{k=1}^{n} k^3 =\) \(\left\{ \frac{n(n+1)}{2} \right\}^2\)
여러 가지 수열의 합 (소거의 원리)
  • 부분분수: \(\frac{1}{AB} =\) \(\frac{1}{B-A} \left(\frac{1}{A} - \frac{1}{B}\right)\)
  • 무리식: \(\frac{1}{\sqrt{B}+\sqrt{A}}\) \(\Rightarrow\) 분모와 분자에 \(\sqrt{B}-\sqrt{A}\) 를 곱하여 유리화!

01. 실전 훈련 (Textbook Training)

미래엔 교과서 중단원 마무리 문항 (난이도순 배열)

[Type: Σ 기호 표현] 기본
1. 다음을 기호 \(\Sigma\)를 사용하여 나타내시오. (교과서 P.151 중단원 01번)

(1) \(2 + 7 + 12 + \cdots + 42\)

(2) \(6 + 10 + 14 + \cdots + 122\)

(3) \(\dfrac{1}{1^3} + \dfrac{1}{2^3} + \dfrac{1}{3^3} + \cdots + \dfrac{1}{10^3}\)

[Type: Σ의 성질 활용] 기본
2. \(\displaystyle\sum_{k=1}^{20} a_k = 10\)이고 \(\displaystyle\sum_{k=1}^{20} b_k = 12\)일 때, 다음 식의 값을 구하시오. (교과서 P.151 중단원 02번)

(1) \(\displaystyle\sum_{k=1}^{20} (2a_k - 5b_k)\)

(2) \(\displaystyle\sum_{k=1}^{20} (a_k + 3b_k - 2)\)

[Type: 거듭제곱 합 공식] 기본
3. 다음 식의 값을 구하시오. (교과서 P.151 중단원 03번)

(1) \(\displaystyle\sum_{k=1}^{20} (3k - 1)\)

(2) \(\displaystyle\sum_{k=1}^{15} (k-1)(k+3)\)

(3) \(\displaystyle\sum_{k=1}^{10} (k^3 - k + 10)\)

[Type: 부분분수 소거] 기본
4. \(\displaystyle\sum_{k=1}^{30} \frac{1}{(k+1)(k+2)}\) 의 값을 구하시오. (교과서 P.151 중단원 04번)
[Type: 연립방정식 활용] 표준
5. 두 수열 \(\{a_n\}\)과 \(\{b_n\}\)에 대하여 \(\displaystyle\sum_{k=1}^{15}(b_k - a_k) = 7\), \(\displaystyle\sum_{k=1}^{15}(a_k + 2b_k) = 11\)일 때, \(\displaystyle\sum_{k=1}^{15} a_k\)의 값을 구하시오. (교과서 P.152 중단원 05번)
💡 전략: \(\sum a_k = A\), \(\sum b_k = B\)로 놓으면, 두 조건이 \(A\)와 \(B\)에 대한 연립방정식이 된다.
[Type: 상수 결정] 표준
6. \(\displaystyle\sum_{k=1}^{20} (2k + a) = 500\)을 만족시키는 상수 \(a\)의 값을 구하시오. (교과서 P.152 중단원 06번)
[Type: 이차방정식 근의 활용] 표준
7. 이차방정식 \(x^2 - 4x - 3 = 0\)의 두 근을 \(a\)와 \(b\)라 할 때, \(\displaystyle\sum_{k=1}^{7} (k+a)(k+b)\)의 값을 구하시오. (교과서 P.152 중단원 07번)
💡 전략: \((k+a)(k+b) = k^2 + (a+b)k + ab\)로 전개한 후, 근과 계수의 관계 \(a+b = 4\), \(ab = -3\)을 대입하면 \(k\)에 대한 식으로 정리된다.
[Type: k²·(k+1) 합] 표준
8. 다음 식의 값을 구하시오. (교과서 P.152 중단원 08번)

\(1^2 \times 2 + 2^2 \times 3 + 3^2 \times 4 + \cdots + 8^2 \times 9\)

[Type: 무리식 유리화 소거] 표준
9. 다음 식의 값을 구하시오. (교과서 P.152 중단원 10번)

\(\dfrac{1}{\sqrt{1}+\sqrt{3}} + \dfrac{1}{\sqrt{3}+\sqrt{5}} + \dfrac{1}{\sqrt{5}+\sqrt{7}} + \cdots + \dfrac{1}{\sqrt{79}+\sqrt{81}}\)

02. 고난도 도전 (Advanced Challenge & Trap Analysis)

"틀려야 는다!"
[Type: 역수 수열의 합] 발전·서술형
10. 수열 \(\dfrac{1}{1},\ \dfrac{1}{1+2},\ \dfrac{1}{1+2+3},\ \dfrac{1}{1+2+3+4},\ \cdots\)의 첫째항부터 제\(n\)항까지의 합이 \(\dfrac{20}{11}\)일 때, 자연수 \(n\)의 값을 구하는 풀이 과정과 답을 쓰시오. (교과서 P.152 중단원 11번)

🚨 선생님의 함정 경고

이 수열의 제\(k\)항은 \(a_k = \frac{1}{1+2+\cdots+k} = \frac{2}{k(k+1)}\) 이다.
여기서 \(a_k\)를 구하고 끝이라고 착각하는 학생들이 매우 많다!
문제를 다시 읽어보자. "첫째항부터 제\(n\)항까지의 합"을 구하라고 했다. 즉, \(a_k\)를 다시 \(\Sigma\) 기호 안에 넣고 계산해야 한다.
\(\frac{2}{k(k+1)} = 2\left(\frac{1}{k} - \frac{1}{k+1}\right)\)로 부분분수 분해 후 소거!

[Type: Sn과 일반항의 관계] 발전
11. 수열 \(\{a_n\}\)의 첫째항부터 제\(n\)항까지의 합 \(S_n\)이 \(S_n = n^2 + 2n\)일 때, 다음에 답하시오. (교과서 P.153 중단원 13번)

(1) 수열 \(\{a_n\}\)의 일반항을 구하시오.

(2) \(\displaystyle\sum_{k=1}^{30} \frac{1}{a_k a_{k+1}}\) 의 값을 구하시오.

🚨 선생님의 함정 경고

(1)에서 \(a_n = S_n - S_{n-1}\) (단, \(n \geq 2\))를 이용하되, \(n = 1\)일 때도 성립하는지 반드시 확인!
(2)에서는 (1)의 결과를 대입한 후 부분분수 분해가 핵심이다.
