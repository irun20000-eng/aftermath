01. 실전 훈련 (Semester-final Comprehensive)

💡 2학기 학습한 단원 I (함수의 극한·연속), II (미분), III (적분)의 핵심을 종합한 14문항. 객관·단답 1~10번 + 서술형 11~14번. 비상 교과서·익힘책 P.148~P.159 기본·도전 문제 + 학기 종합 출제.

📘 단원 I. 함수의 극한과 연속 (Q1~Q3)

극한 계산 · 미정형 · 연속·미분가능 · 사잇값 정리

[I. 극한 — 미정형 \(\dfrac{0}{0}\)]
1. 극한값 \( \displaystyle \lim_{x \to 1} \dfrac{\sqrt{x + 3} - 2}{x^2 - 1} \) 의 값을 구하시오. (학기말 종합 — 단원 I 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: 분자 유리화: \( \dfrac{(\sqrt{x + 3} - 2)(\sqrt{x + 3} + 2)}{(x^2 - 1)(\sqrt{x + 3} + 2)} = \dfrac{x - 1}{(x - 1)(x + 1)(\sqrt{x + 3} + 2)} = \dfrac{1}{(x + 1)(\sqrt{x + 3} + 2)} \). \(x \to 1\): \( \dfrac{1}{2 \cdot 4} = \dfrac{1}{8} \).
[I. 함수의 연속 — 미정 상수 결정]
2. 함수 \( f(x) = \begin{cases} \dfrac{x^2 + a x - 6}{x - 2} & (x \neq 2) \\ b & (x = 2) \end{cases} \) 이 \(x = 2\)에서 연속이 되도록 하는 실수 \(a, b\)의 값을 구하시오. (학기말 종합 — 단원 I 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: 연속 ⇔ \( \lim_{x \to 2} f(x) = f(2) = b \). 분모 \(\to 0\), 극한이 유한 \(\Rightarrow\) 분자 \(\to 0\): \( 4 + 2a - 6 = 0 \Rightarrow a = 1 \). 그러면 분자 \(= x^2 + x - 6 = (x - 2)(x + 3) \). 극한 \( = \lim_{x \to 2} (x + 3) = 5 = b \). 답: \(a = 1, b = 5\).
[I. 사잇값 정리 — 실근의 존재]
3. 방정식 \( x^3 - 4 x^2 + 1 = 0 \)이 열린구간 \( (0, 1) \)과 \( (3, 4) \) 각각에서 실근을 적어도 하나씩 가짐을 사잇값 정리를 이용해 보이시오. (학기말 종합 — 단원 I 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: \( f(x) = x^3 - 4 x^2 + 1 \). 다항함수이므로 모든 실수에서 연속. 함숫값: \( f(0) = 1 \gt 0 \), \( f(1) = 1 - 4 + 1 = -2 \lt 0 \). 사잇값 정리에 의해 \((0, 1)\)에서 \(f(x) = 0\)인 \(x\) 존재. \( f(3) = 27 - 36 + 1 = -8 \lt 0 \), \( f(4) = 64 - 64 + 1 = 1 \gt 0 \). 사잇값 정리에 의해 \((3, 4)\)에서 실근 존재.

📗 단원 II. 미분 (Q4~Q7)

미분계수 · 접선 · 극값 · 그래프 · 속도와 가속도

[II. 미분계수 정의 활용]
4. 다항함수 \( f(x) \)에 대하여 \( f(2) = 3 \), \( f'(2) = 4 \)일 때, 극한값 \( \displaystyle \lim_{x \to 2} \dfrac{x^2 f(2) - 4 f(x)}{x - 2} \) 의 값을 구하시오. (학기말 종합 — 단원 II 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: 분자: \( x^2 f(2) - 4 f(x) = 3 x^2 - 4 f(x) \). \(x = 2\) 대입 시 \( 12 - 12 = 0 \). 변형: \(3 x^2 - 4 f(x) = 3(x^2 - 4) + (12 - 4 f(x)) = 3(x - 2)(x + 2) - 4(f(x) - 3) = 3(x - 2)(x + 2) - 4(f(x) - f(2))\). 따라서 분수 \( = 3(x + 2) - 4 \cdot \dfrac{f(x) - f(2)}{x - 2} \to 3 \cdot 4 - 4 \cdot 4 = 12 - 16 = -4 \).
[II. 접선의 방정식 — 외부점]
5. 점 \( (0, 2) \)에서 곡선 \( y = x^3 + 1 \)에 그은 접선의 방정식을 모두 구하시오. (학기말 종합 — 단원 II 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: 접점 \((t, t^3 + 1)\), 접선 기울기 \(3 t^2\). 접선식: \( y - (t^3 + 1) = 3 t^2 (x - t) \). 점 \((0, 2)\) 통과: \( 2 - (t^3 + 1) = 3 t^2 (0 - t) \Rightarrow 1 - t^3 = -3 t^3 \Rightarrow 2 t^3 = -1 \Rightarrow t^3 = -\dfrac{1}{2} \Rightarrow t = -\dfrac{1}{\sqrt[3]{2}} \). 기울기 \( 3 t^2 = 3 \cdot \dfrac{1}{\sqrt[3]{4}} = \dfrac{3}{\sqrt[3]{4}} \). 접선: \( y = \dfrac{3}{\sqrt[3]{4}} x + 2 \) (단 한 개).
[II. 닫힌구간의 최대·최소]
6. 닫힌구간 \([-1, 3]\)에서 함수 \( f(x) = x^3 - 3 x^2 + 1 \) 의 최댓값과 최솟값을 구하시오. (학기말 종합 — 단원 II 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: \( f'(x) = 3 x^2 - 6 x = 3x(x - 2) = 0 \Rightarrow x = 0, 2 \). 후보 함숫값: \( f(-1) = -1 - 3 + 1 = -3 \), \( f(0) = 1 \), \( f(2) = 8 - 12 + 1 = -3 \), \( f(3) = 27 - 27 + 1 = 1 \). 최댓값 \(1\) (\(x = 0\) 또는 \(x = 3\)), 최솟값 \(-3\) (\(x = -1\) 또는 \(x = 2\)).
[II. 속도와 가속도]
7. 수직선 위를 움직이는 점 P의 시각 \(t\)에서 위치가 \( x(t) = t^3 - 6 t^2 + 9 t \)일 때, 점 P가 처음으로 운동 방향을 바꾸는 순간의 가속도를 구하시오. (학기말 종합 — 단원 II 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: \( v(t) = x'(t) = 3 t^2 - 12 t + 9 = 3(t - 1)(t - 3) \). 운동 방향 전환 시각 = \(v(t) = 0\)이고 부호 변화: \(t = 1\) (\(+ \to -\)) — 첫 번째, \(t = 3\) (\(- \to +\)). \(a(t) = v'(t) = 6 t - 12 \). \(t = 1\): \(a(1) = -6\).

📕 단원 III. 적분 (Q8~Q10)

부정적분 · 정적분 · 넓이 · 속도와 거리

[III. 정적분 — 절댓값]
8. 정적분 \( \displaystyle \int_{-1}^{2} |x^2 - x|\,dx \) 의 값을 구하시오. (학기말 종합 — 단원 III 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: \(x^2 - x = x(x - 1) \). 부호: \(x \lt 0\)에서 \(+\), \(0 \lt x \lt 1\)에서 \(-\), \(x \gt 1\)에서 \(+\). 구간 분할: \( \int_{-1}^{0} (x^2 - x)\,dx + \int_0^1 (x - x^2)\,dx + \int_1^2 (x^2 - x)\,dx \). \( I_1 = \left[\dfrac{x^3}{3} - \dfrac{x^2}{2}\right]_{-1}^{0} = 0 - \left(-\dfrac{1}{3} - \dfrac{1}{2}\right) = \dfrac{5}{6} \). \( I_2 = \left[\dfrac{x^2}{2} - \dfrac{x^3}{3}\right]_0^1 = \dfrac{1}{2} - \dfrac{1}{3} = \dfrac{1}{6} \). \( I_3 = \left[\dfrac{x^3}{3} - \dfrac{x^2}{2}\right]_1^2 = \left(\dfrac{8}{3} - 2\right) - \left(\dfrac{1}{3} - \dfrac{1}{2}\right) = \dfrac{2}{3} + \dfrac{1}{6} = \dfrac{5}{6} \). 합 \( = \dfrac{5}{6} + \dfrac{1}{6} + \dfrac{5}{6} = \dfrac{11}{6} \).
[III. 두 곡선 사이의 넓이]
9. 두 곡선 \( y = x^2 \)과 \( y = -x^2 + 4x \)로 둘러싸인 도형의 넓이를 구하시오. (학기말 종합 — 단원 III 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: 교점: \(x^2 = -x^2 + 4x \Rightarrow 2 x^2 - 4x = 0 \Rightarrow 2x(x - 2) = 0 \Rightarrow x = 0, 2\). \(0 \lt x \lt 2\)에서 \(-x^2 + 4x \gt x^2\) (예: \(x = 1\): \(3 \gt 1\)). 넓이 \( = \int_0^2 [(-x^2 + 4x) - x^2] dx = \int_0^2 (-2 x^2 + 4x) dx = \left[-\dfrac{2 x^3}{3} + 2 x^2\right]_0^2 = -\dfrac{16}{3} + 8 = \dfrac{8}{3} \).
[III. 적분으로 정의된 함수]
10. 함수 \( \displaystyle f(x) = \int_1^x (t^2 - 4) dt \) 의 극솟값을 구하시오. (학기말 종합 — 단원 III 핵심)
🎯 풀이 전략 보기
💡 전략 힌트: \( f'(x) = x^2 - 4 = (x - 2)(x + 2) = 0 \Rightarrow x = \pm 2 \). 부호: \(x \lt -2\)에서 \(+\), \(-2 \lt x \lt 2\)에서 \(-\), \(x \gt 2\)에서 \(+\). 따라서 \(x = -2\)에서 극대, \(x = 2\)에서 극소. \( f(2) = \int_1^2 (t^2 - 4)\,dt = \left[\dfrac{t^3}{3} - 4t\right]_1^2 = \left(\dfrac{8}{3} - 8\right) - \left(\dfrac{1}{3} - 4\right) = \dfrac{7}{3} - 4 = -\dfrac{5}{3} \). 극솟값 \( = -\dfrac{5}{3} \).

서술형 문제 [11~14] — 단원 간 융합

아래 문제는 풀이 과정을 자세히 작성하시오. 단원 I·II·III가 융합된 종합 문항입니다.

[서술형 · I+II 융합 — 연속·미분가능 조건]
11. 함수 \( f(x) = \begin{cases} x^2 + a x + b & (x \leq 1) \\ c x + 2 & (x \gt 1) \end{cases} \) 이 \(x = 1\)에서 미분가능하고, \( f(-1) = 4 \)일 때, 실수 \(a, b, c\)의 값을 구하시오. (학기말 종합 — I·II 융합 서술형)
🎯 서술형 풀이 전략 보기
💡 전략: ① \(x = 1\)에서 연속: \( f(1^-) = 1 + a + b \), \( f(1^+) = c + 2 \). 같음: \(1 + a + b = c + 2 \Rightarrow a + b - c = 1\) ... (가).
② \(x = 1\)에서 미분가능: 좌도함수 \(f'_-(x) = 2x + a \Rightarrow f'_-(1) = 2 + a\). 우도함수 \(f'_+(x) = c\). 같음: \(2 + a = c \Rightarrow c = a + 2\) ... (나).
③ \( f(-1) = 1 - a + b = 4 \Rightarrow b - a = 3 \Rightarrow b = a + 3\) ... (다).
(가)에 (나), (다) 대입: \( a + (a + 3) - (a + 2) = 1 \Rightarrow a + 1 = 1 \Rightarrow a = 0 \). 따라서 \(b = 3\), \(c = 2\).
검증: \(f(x) = x^2 + 3\) (\(x \leq 1\)), \(f(x) = 2x + 2\) (\(x \gt 1\)). \(f(1^-) = 4\), \(f(1^+) = 4\) ✓. \(f'_-(1) = 2\), \(f'_+(1) = 2\) ✓. \(f(-1) = 1 + 3 = 4\) ✓.
서술 채점 포인트: (i) 연속 조건식 도출, (ii) 좌·우 미분계수 일치 조건 도출, (iii) 추가 조건 \(f(-1) = 4\) 식 도출, (iv) 세 조건 연립 풀이 \(\to a, b, c\) 유일 결정.
[서술형 · II 중심 — 방정식 실근 개수]
12. 방정식 \( x^3 - 3 x^2 + a = 0 \) 이 서로 다른 세 실근을 갖도록 하는 실수 \(a\)의 값의 범위를 구하시오. (학기말 종합 — II 중심 서술형)
🎯 서술형 풀이 전략 보기
💡 전략: ① \(a\) 분리: \( x^3 - 3 x^2 = -a \). 보조함수 \( g(x) = x^3 - 3 x^2 \)와 수평선 \(y = -a\)가 세 점에서 만나는 조건. ② \(g'(x) = 3 x^2 - 6x = 3x(x - 2) = 0 \Rightarrow x = 0, 2\). 극대 \(g(0) = 0\), 극소 \(g(2) = 8 - 12 = -4\). ③ 세 실근 조건: \(-4 \lt -a \lt 0 \Rightarrow 0 \lt a \lt 4\).
서술 채점 포인트: 보조함수 정의, 극값 좌표 명시, 극값 사이 수평선 교점 3개 조건.
[서술형 · II+III 융합 — 접선 + 넓이]
13. 곡선 \( y = x^2 \) 위의 점 \( (2, 4) \)에서의 접선을 \(\ell\)이라 하자. (학기말 종합 — II·III 융합 서술형)
(1) 접선 \(\ell\)의 방정식을 구하시오.
(2) 곡선 \(y = x^2\), 접선 \(\ell\), \(y\)축으로 둘러싸인 도형의 넓이를 구하시오.
🎯 서술형 풀이 전략 보기
💡 전략: (1) \( y' = 2x \), \(x = 2\)에서 기울기 \(4\). 접선: \(y - 4 = 4(x - 2) \Rightarrow y = 4x - 4\).
(2) 접선 \(\ell\)이 \(y\)축과 만나는 점 \( (0, -4) \). 영역: 곡선 \(y = x^2\) (위), \(y\)축 (왼쪽), 접선 \(\ell\) (아래). 영역 범위 \(x \in [0, 2]\). \(0 \leq x \leq 2\)에서 \(x^2 \geq 4x - 4\) (왜냐하면 \(x^2 - (4x - 4) = (x - 2)^2 \geq 0\)). 넓이 \( = \int_0^2 [x^2 - (4x - 4)] dx = \int_0^2 (x - 2)^2 dx = \left[\dfrac{(x - 2)^3}{3}\right]_0^2 = 0 - \left(-\dfrac{8}{3}\right) = \dfrac{8}{3} \).
서술 채점 포인트: 접선식 도출, 위·아래 곡선 판별 (제곱 차이 ≥ 0), 정적분 계산.
[서술형 · II+III 융합 — 속도와 거리]
14. 원점을 출발하여 수직선 위를 움직이는 점 P의 시각 \(t\)에서 속도가 \( v(t) = t^2 - 6 t + 8 \) 일 때, 다음 물음에 답하시오. (학기말 종합 — II·III 융합 서술형)
(1) 점 P의 시각 \(t = 5\)에서의 위치를 구하시오.
(2) 점 P가 출발 후 \(t = 5\)까지 움직인 거리를 구하시오.
🎯 서술형 풀이 전략 보기
💡 전략: \(v(t) = (t - 2)(t - 4)\). 부호: \(0 \lt t \lt 2\)에서 \(+\), \(2 \lt t \lt 4\)에서 \(-\), \(t \gt 4\)에서 \(+\).
(1) 위치 \( x(5) = \int_0^5 v(t)\,dt = \int_0^5 (t^2 - 6t + 8)\,dt = \left[\dfrac{t^3}{3} - 3 t^2 + 8 t\right]_0^5 = \dfrac{125}{3} - 75 + 40 = \dfrac{125}{3} - 35 = \dfrac{125 - 105}{3} = \dfrac{20}{3} \).
(2) 움직인 거리 \( = \int_0^5 |v(t)|\,dt \). 구간 분할:
\(\int_0^2 v\,dt = \left[\dfrac{t^3}{3} - 3 t^2 + 8 t\right]_0^2 = \dfrac{8}{3} - 12 + 16 = \dfrac{8}{3} + 4 = \dfrac{20}{3}\).
\(\int_2^4 v\,dt = \left[\dfrac{t^3}{3} - 3 t^2 + 8 t\right]_2^4 = \left(\dfrac{64}{3} - 48 + 32\right) - \dfrac{20}{3} = \dfrac{64}{3} - 16 - \dfrac{20}{3} = \dfrac{44}{3} - 16 = \dfrac{44 - 48}{3} = -\dfrac{4}{3}\). 절댓값 \( = \dfrac{4}{3} \).
\(\int_4^5 v\,dt = \left[\dfrac{t^3}{3} - 3 t^2 + 8 t\right]_4^5 = \left(\dfrac{125}{3} - 75 + 40\right) - \left(\dfrac{64}{3} - 48 + 32\right) = \dfrac{20}{3} - \left(\dfrac{64}{3} - 16\right) = \dfrac{20}{3} - \dfrac{64}{3} + 16 = -\dfrac{44}{3} + 16 = \dfrac{-44 + 48}{3} = \dfrac{4}{3}\). 양.
합 \( = \dfrac{20}{3} + \dfrac{4}{3} + \dfrac{4}{3} = \dfrac{28}{3} \).
서술 채점 포인트: \(v(t)\) 인수분해 + 부호표, 위치 정적분 계산, 움직인 거리 \(= \int|v|\,dt\) 구간 분할 + 부호 처리.
