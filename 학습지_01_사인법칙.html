🤔 1. 흥미 유발 (Open Question)

우리가 산의 높이나 강 건너편까지의 거리를 직접 자로 재지 않고도 알아낼 수 있을까? 오늘 배울 법칙이 바로 그 마법 같은 일을 가능하게 해준다. 직각삼각형이 아닐 때도 삼각비를 쓸 수 있을지 고민해 보자.

🔙 2. 지난 학습 복습 (Review)

중학교 때 배운 '원주각'을 기억하는가? 한 원에서 호의 길이가 일정할 때 원주각의 크기는 항상 일정하며, 지름의 원주각은 항상 \(90^\circ\) 이다. 이 성질이 오늘 증명의 핵심 무기가 된다!

01. 사인법칙 (The Law of Sines)

핵심 공식

\[ \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R \]

(\(R\)은 \(\triangle ABC\)의 외접원의 반지름)

🛠 공식 변형 Toolkit

Type A: 변의 길이 구하기

\(a =\) \(2R \sin A\)

\(b =\) \(2R \sin B\)

Type B: 사인값 구하기

\(\sin A =\) \(\frac{a}{2R}\)

\(\sin B =\) \(\frac{b}{2R}\)

Type C: 길이의 비 = 사인비

\(a : b : c =\)

\(\sin A : \sin B : \sin C\)

02. 수준별 훈련 (Level Training)

Level 1: 기초 훈련
1. \(\triangle ABC\)에서 \(a=10\), \(A=45^\circ\), \(B=60^\circ\)일 때, 변 \(b\)의 길이를 구하시오.
2. \(\triangle ABC\)의 외접원의 반지름 길이가 \(R=8\)이고, \(A=30^\circ\)일 때, 변 \(a\)의 길이를 구하시오.
Level 2: 응용 훈련
3. \(\triangle ABC\)에서 \(A:B:C = 1:2:3\)일 때, \(a:b:c\)를 구하시오.
💡 Hint: 세 각의 합은 \(180^\circ\)이다. 비례 배분을 통해 각 \(A, B, C\)의 크기를 먼저 구한 뒤, Type C 공식을 적용해보자.
4. 원에 내접하는 \(\triangle ABC\)에서 \(\sin A + \sin B + \sin C = \frac{3}{2}\) 이고, \(\triangle ABC\)의 둘레의 길이가 \(12\)일 때, 외접원의 반지름 길이 \(R\)을 구하시오.
Level 3: 심화 훈련 (함정 주의)
5. \(\triangle ABC\)에서 \(b=2\), \(c=2\sqrt{3}\), \(B=30^\circ\)일 때, 각 \(C\)의 크기를 모두 구하시오.
⚠️ 핵심 분석 (틀려야 는다!): \(\sin C\)의 값이 양수일 때, \(C\)는 예각이 될 수도 있고 둔각이 될 수도 있다. \(0^\circ < C < 180^\circ\) 범위에서 만족하는 각이 2개 나올 수 있음을 명심하자.

03. 형성평가 (Final Check)

6. \(\triangle ABC\)에서 다음 관계식이 성립할 때, 이 삼각형은 어떤 삼각형인지 말하시오.
\[ a \sin A = b \sin B \]
💡 전략: 변의 길이(\(a, b, c\))에 대한 식으로 모두 바꾸어 보자. (Type B 변형 공식 활용)

⭐ 도전 문제

삼각형 \(ABC\)에서 \(\angle A = 60^\circ\), \(a = 7\), \(b = 8\)일 때, \(\sin B\)의 값을 구하시오.
💡 전략: 사인법칙 \(\dfrac{a}{\sin A} = \dfrac{b}{\sin B}\)에서 \(\sin B\)를 \(a\), \(b\), \(\sin A\)로 표현해보자.
