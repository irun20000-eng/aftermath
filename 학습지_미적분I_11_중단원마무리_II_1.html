01. 실전 훈련 (Textbook Mastery)

💡 비상 미적분 I 교과서 중단원 마무리(P.64~P.66) 14문항을 그대로 학습지화. 기초 1~3 · 기본 4~12 · 도전 13·14 (서술형 킬러)

[기초-평균변화율 직접 계산]
1. 함수 \( f(x) = 2x^2 - 3x \)에서 \(x\)의 값이 \(0\)에서 \(2\)까지 변할 때의 평균변화율을 구하시오. (교과서 P.64 중단원 기초 01번)
🎯 풀이 전략 보기
💡 전략 힌트: 평균변화율 \(= \dfrac{f(2) - f(0)}{2 - 0}\). \(f(2) = 8 - 6 = 2\), \(f(0) = 0\). 답: \(\dfrac{2 - 0}{2} = 1\).
[기초-접선의 기울기 = 미분계수]
2. 곡선 \( y = -x^2 + 7x - 2 \) 위의 점 \((1, 4)\)에서의 접선의 기울기를 구하시오. (교과서 P.64 중단원 기초 02번)
🎯 풀이 전략 보기
💡 전략 힌트: 접선의 기울기 \(= f'(1)\). \(f'(x) = -2x + 7\)이므로 \(f'(1) = 5\). 또는 정의식으로 \(\lim_{h \to 0}\dfrac{f(1+h)-f(1)}{h}\)를 직접 계산해도 같은 값.
[기초-도함수 공식 + 곱의 미분]
3. 다음 함수를 미분하시오. (교과서 P.64 중단원 기초 03번)
(1) \( y = 10 \)    (2) \( y = -4x^2 + 2x + 1 \)    (3) \( y = (x - 1)(x^2 + x + 6) \)    (4) \( y = (x - 2)(x + 1)(x + 3) \)
🎯 풀이 전략 보기
💡 전략 힌트: (1) 상수 \(\Rightarrow\) \(y' = 0\). (2) 항별 미분 \(\Rightarrow\) \(y' = -8x + 2\). (3) 곱의 미분 \(y' = (x^2+x+6) + (x-1)(2x+1) = 3x^2 + 5\). 검산: 전개 후 미분 결과와 일치. (4) 삼중 곱: \(y' = (x+1)(x+3) + (x-2)(x+3) + (x-2)(x+1) = 3x^2 + 4x - 5\).
[기본-평균변화율 조건으로 미정계수]
4. 함수 \( f(x) = x^2 + x + 1 \)에서 \(x\)의 값이 \(a\)에서 \(a + 3\)까지 변할 때의 평균변화율이 \(6\)이다. 이때 실수 \(a\)의 값을 구하시오. (교과서 P.65 중단원 기본 04번)
🎯 풀이 전략 보기
💡 전략 힌트: 평균변화율 \(= \dfrac{f(a+3) - f(a)}{3} = \dfrac{6a + 12}{3} = 2a + 4 = 6\) \(\Rightarrow\) \(a = 1\).
[기본-그래프 분석 + 기울기 비교]
5. 미분가능한 함수 \(y = f(x)\)의 그래프와 직선 \(y = x\)가 다음 그림과 같다. 두 점 \(\mathrm{A}, \mathrm{B}\)가 함수 \(y = f(x)\)의 그래프 위에 있을 때, 보기 중 옳은 것을 모두 고르시오. (단, \(0 \lt a \lt b\)) (교과서 P.65 중단원 기본 05번)
<보기> ㄱ. \( \dfrac{f(a)}{a} \lt \dfrac{f(b)}{b} \)
ㄴ. \( f'(a) \lt f'(b) \)
ㄷ. \( f(a) - a \gt f(b) - b \)
🎯 풀이 전략 보기
💡 전략 힌트: 그래프 해석. \(y = f(x)\)는 위로 볼록 증가(우상향, 기울기 점차 커짐) — "아래로 볼록"한 곡선. \(\dfrac{f(a)}{a}\)는 원점과 \(\mathrm{A}\) 잇는 직선의 기울기, \(\dfrac{f(b)}{b}\)는 원점과 \(\mathrm{B}\) 잇는 직선의 기울기.
ㄱ: 곡선이 아래로 볼록 + 원점 통과 \(\Rightarrow\) \(b\)가 더 멀수록 원점과 잇는 직선의 기울기가 더 커짐 \(\Rightarrow\) 참.
ㄴ: \(f'\)은 접선의 기울기. 곡선이 아래로 볼록 \(\Rightarrow\) 오른쪽으로 갈수록 접선 기울기가 점점 커짐 \(\Rightarrow\) \(f'(a) \lt f'(b)\). 참.
ㄷ: \(f(x) - x\)는 \(\mathrm{A}, \mathrm{B}\)에서 곡선과 직선 \(y = x\) 사이의 세로 거리. 그림에서 \(\mathrm{B}\)가 \(\mathrm{A}\)보다 \(y = x\)에서 더 멀리 떨어져 있음 \(\Rightarrow\) \(f(b) - b \gt f(a) - a\), 따라서 \(f(a) - a \gt f(b) - b\)는 거짓.
정답: ㄱ, ㄴ.
[기본-극한식 \(\to\) 미분계수 변환]
6. 함수 \( f(x) = 3x^3 - x + 2 \)에 대하여 \( \displaystyle \lim_{h \to 0}\dfrac{f(1 + 3h) - f(1)}{2h} \) 의 값을 구하시오. (교과서 P.65 중단원 기본 06번)
🎯 풀이 전략 보기
💡 전략 힌트: \(\dfrac{f(1+3h) - f(1)}{2h} = \dfrac{f(1+3h) - f(1)}{3h} \cdot \dfrac{3}{2} \to f'(1) \cdot \dfrac{3}{2}\). \(f'(x) = 9x^2 - 1\), \(f'(1) = 8\). 답: \(\dfrac{3}{2} \cdot 8 = 12\).
[기본-\(\dfrac{0}{0}\) 부정형 + 미분계수]
7. 미분가능한 함수 \(f(x)\)가 \( f(2) = 1, \; f'(2) = -6 \) 을 만족시킬 때, \( \displaystyle \lim_{x \to 2}\dfrac{f(x) - 1}{x^2 - x - 2} \) 의 값을 구하시오. (교과서 P.65 중단원 기본 07번)
🎯 풀이 전략 보기
💡 전략 힌트: \(x^2 - x - 2 = (x-2)(x+1)\). \(f(x) - 1 = f(x) - f(2)\). 식 \(= \dfrac{f(x) - f(2)}{x - 2} \cdot \dfrac{1}{x+1}\). \(x \to 2\) 극한 \(= f'(2) \cdot \dfrac{1}{3} = -6 \cdot \dfrac{1}{3} = -2\).
[기본-자기참조형 다항함수]
8. 다항함수 \(f(x)\)가 \( f(x) = 2x^3 - 3x f'(2) \) 를 만족시킬 때, \(f'(1)\)의 값을 구하시오. (교과서 P.65 중단원 기본 08번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(2) = k\)로 두고 \(f(x) = 2x^3 - 3kx\) \(\Rightarrow\) \(f'(x) = 6x^2 - 3k\). \(x = 2\): \(24 - 3k = k\) \(\Rightarrow\) \(k = 6\). \(f'(x) = 6x^2 - 18\), \(f'(1) = -12\).
[기본-구간 분기 함수의 미분가능성]
9. 함수 \( f(x) = \begin{cases} x^2 + ax + 3 & (x \geq 1) \\ 3x + b & (x \lt 1) \end{cases} \) 가 모든 실수 \(x\)에서 미분가능할 때, 실수 \(a, b\)의 값을 구하시오. (교과서 P.66 중단원 기본 09번)
🎯 풀이 전략 보기
💡 전략 힌트: 미분가능 \(\Rightarrow\) ① \(x=1\) 연속, ② 좌·우 미분계수 일치.
① 연속: \(1 + a + 3 = 3 + b\) \(\Rightarrow\) \(a - b = -1\).
② 미분계수: 우측 \(f'_+ = 2x + a\big|_{x=1} = 2 + a\), 좌측 \(f'_- = 3\). 일치: \(2 + a = 3\) \(\Rightarrow\) \(a = 1\). ①에서 \(b = 2\).
답: \(a = 1, b = 2\).
[기본-곱의 미분으로 \(g'(1)\) 구하기]
10. 미분가능한 두 함수 \(f(x), g(x)\)가 \( g(x) = (x^3 - x + 1) f(x) \) 를 만족시키고 \(f(1) = 1, f'(1) = 5\)일 때, \(g'(1)\)의 값을 구하시오. (교과서 P.66 중단원 기본 10번)
🎯 풀이 전략 보기
💡 전략 힌트: \(p(x) = x^3 - x + 1\), \(p'(x) = 3x^2 - 1\), \(p(1) = 1\), \(p'(1) = 2\). 곱의 미분: \(g'(1) = p'(1)f(1) + p(1)f'(1) = 2 \cdot 1 + 1 \cdot 5 = 7\).
[기본-나머지정리 + 양변 미분]
11. 다항식 \( x^{100} - 50 x^2 - a x + b \)를 \( (x + 1)^2 \)로 나누었을 때의 나머지는 \(x + 2\)이다. 실수 \(a, b\)의 값을 구하시오. (교과서 P.66 중단원 기본 11번)
🎯 풀이 전략 보기
💡 전략 힌트: \(P(x) = x^{100} - 50 x^2 - ax + b = (x+1)^2 Q(x) + (x+2)\).
① \(x = -1\) 대입: \(1 - 50 + a + b = 1\) \(\Rightarrow\) \(a + b = 50\).
② 양변 미분 후 \(x = -1\) 대입: \(P'(x) = 100 x^{99} - 100x - a\), \(P'(-1) = -100 + 100 - a = -a\). 우변 미분 \(= 2(x+1)Q + (x+1)^2 Q' + 1\), \(x = -1\)에서 \(= 1\). \(\Rightarrow\) \(-a = 1 \Rightarrow a = -1\). ①에서 \(b = 51\).
답: \(a = -1, b = 51\).
[기본-함수방정식 + 미분계수 정의]
12. 미분가능한 함수 \(f(x)\)가 모든 실수 \(x, y\)에 대하여 \( f(x + y) = f(x) + f(y) - 4xy \) 를 만족시키고, \(f'(0) = 7\)일 때, \(f'(1)\)의 값을 구하시오. (교과서 P.66 중단원 기본 12번)
🎯 풀이 전략 보기
💡 전략 힌트: ① \(x = y = 0\) 대입 \(\Rightarrow\) \(f(0) = 0\). ② \(f'(1) = \displaystyle\lim_{h \to 0}\dfrac{f(1+h) - f(1)}{h}\). 함수방정식에서 \(x=1, y=h\): \(f(1+h) - f(1) = f(h) - 4h\). \(\Rightarrow\) \(f'(1) = \lim_{h \to 0}\dfrac{f(h) - 4h}{h} = f'(0) - 4 = 7 - 4 = 3\).
[도전-극한 조건 + 곱의 미분 활용 (서술형)]
13. 미분가능한 함수 \(f(x)\)가 \( \displaystyle \lim_{x \to 2}\dfrac{f(x) - 2}{x - 2} = 1 \) 을 만족시킬 때, \( \displaystyle \lim_{x \to 2}\dfrac{x^2 f(x) - 4 f(2)}{x^3 - 8} \) 의 값을 풀이 과정과 함께 구하시오. (교과서 P.66 중단원 도전 13번)
🎯 서술형 풀이 전략 보기
💡 전략 (5단계):
① 조건식 분석: \(\lim_{x \to 2}\dfrac{f(x) - 2}{x - 2} = 1\). 분모 \(\to 0\)이고 극한이 실수 \(\Rightarrow\) 분자도 \(\to 0\) \(\Rightarrow\) \(f(2) = 2\). 그리고 좌변 \(= f'(2) = 1\).
② 목표식 분자에 \(\pm 4 f(x)\) 끼워 넣기: \(x^2 f(x) - 4 f(2) = x^2 f(x) - 4 f(x) + 4 f(x) - 4 f(2) = (x^2 - 4) f(x) + 4 [f(x) - f(2)]\).
③ 분모 인수분해: \(x^3 - 8 = (x - 2)(x^2 + 2x + 4)\).
④ 두 항 극한 분리: 첫 항 \(\dfrac{(x-2)(x+2) f(x)}{(x-2)(x^2+2x+4)} = \dfrac{(x+2)f(x)}{x^2 + 2x + 4} \to \dfrac{4 \cdot 2}{12} = \dfrac{2}{3}\). 둘째 항 \(\dfrac{4[f(x) - f(2)]}{(x-2)(x^2+2x+4)} = \dfrac{4}{x^2+2x+4} \cdot \dfrac{f(x) - f(2)}{x - 2} \to \dfrac{4}{12} \cdot 1 = \dfrac{1}{3}\).
⑤ 합: \(\dfrac{2}{3} + \dfrac{1}{3} = 1\). 답: \(1\).
채점 포인트: \(f(2) = 2, f'(2) = 1\) 도출 / \(\pm 4 f(x)\) 분리 / 인수분해 \(x^3 - 8\) / 두 항 극한 명시 / 최종 합산.
[도전-삼차함수 보조함수 + 인수분해 (서술형)]
14. 삼차함수 \(f(x)\)에 대하여 \( f(-1) = f(0) = f(1) = 2 \) 이다. \(f'(1) = 8\)일 때, \(f'(2)\)의 값을 풀이 과정과 함께 구하시오. (교과서 P.66 중단원 도전 14번)
🎯 서술형 풀이 전략 보기
💡 전략 (4단계):
① 보조함수: \(g(x) = f(x) - 2\)로 두면 \(g\)는 삼차함수이고 \(g(-1) = g(0) = g(1) = 0\) \(\Rightarrow\) 세 근 \(-1, 0, 1\).
② 인수분해: \(g(x) = a(x+1) \cdot x \cdot (x-1) = a(x^3 - x)\) (\(a\)는 상수). 따라서 \(f(x) = a(x^3 - x) + 2\).
③ 도함수와 조건 적용: \(f'(x) = a(3x^2 - 1)\). \(f'(1) = a(3 - 1) = 2a = 8\) \(\Rightarrow\) \(a = 4\).
④ 결론: \(f'(x) = 4(3x^2 - 1)\), \(f'(2) = 4(12 - 1) = 44\). 답: \(f'(2) = 44\).
채점 포인트: 보조함수 정의 / 인수분해 형태 명시 / 상수 \(a\) 결정 / 최종 값.
