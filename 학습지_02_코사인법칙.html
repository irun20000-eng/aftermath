00. 생각 열기 (Warm-up & Connection)

🤔 1. 흥미 유발 (Open Question)

바다 한가운데 두 개의 섬이 있다. 배가 없어서 건너갈 수 없는데, 두 섬 사이의 거리를 구하려면 어떻게 해야 할까? 내가 있는 곳에서 각 섬까지의 거리와, 두 섬 사이의 각도만 알면 잴 수 있다. '건널 수 없는 거리'를 재는 마법, 바로 코사인법칙이다.

🔙 2. 지난 학습 복습 (Review)

중학교 때 배운 피타고라스의 정리를 기억하는가? 직각삼각형에서 \(a^2 + b^2 = \) \(c^2\) 이다. 오늘 배울 코사인법칙은 바로 이 피타고라스 정리의 **확장판**으로, 직각이 아닌 모든 삼각형에 적용할 수 있는 강력한 무기이다!

01. 코사인법칙 (The Law of Cosines)

핵심 공식

\[ a^2 = b^2 + c^2 - 2bc \cos A \]

(나머지 두 변에 대해서도 \(b^2=a^2+c^2-2ac\cos B\) 등으로 동일하게 성립한다.)

🛠 공식 변형 Toolkit

Type A: 두 변과 끼인각을 알 때

남은 한 변의 길이를 구할 수 있다.

\(a =\) \(\sqrt{b^2 + c^2 - 2bc \cos A}\)

Type B: 세 변의 길이를 알 때

원하는 각의 크기(코사인값)를 구할 수 있다.

\(\cos A =\) \(\frac{b^2 + c^2 - a^2}{2bc}\)

Type C: 피타고라스와의 연결

만약 각 \(A = 90^\circ\) 라면, \(\cos 90^\circ =\) \(0\) 이므로,

결국 \(a^2 = b^2 + c^2\) 이 된다!

02. 수준별 훈련 (Level Training)

Level 1: 기초 훈련
1. \(\triangle ABC\)에서 \(b=3\), \(c=5\), \(A=60^\circ\)일 때, 변 \(a\)의 길이를 구하시오.
2. \(\triangle ABC\)에서 \(a=7\), \(b=3\), \(c=8\)일 때, 각 \(A\)의 크기를 구하시오.
Level 2: 응용 훈련 (사인법칙 융합)
3. \(\triangle ABC\)에서 \(a=5\), \(b=4\), \(\cos C = \frac{1}{5}\)일 때, 이 삼각형의 외접원의 반지름의 길이 \(R\)을 구하시오.
💡 힌트 보기
💡 Hint: 먼저 코사인법칙으로 변 \(c\)를 구하자. 그 다음, \(\cos C\)를 이용해 \(\sin C\)를 찾아낸 뒤에 사인법칙을 적용해야 한다!
4. \(\triangle ABC\)에서 세 변의 길이의 비가 \(a : b : c = 3 : 5 : 7\) 일 때, 이 삼각형의 가장 큰 내각의 크기를 구하시오.
💡 힌트 보기
💡 Hint: 길이가 가장 긴 변과 마주보는 대각이 항상 가장 큰 각이다. 변의 길이를 \(3k, 5k, 7k\)로 두고 식을 세워보자. (\(k\)는 약분되어 사라진다!)
Level 3: 심화 훈련 (함정 주의)
5. 세 변의 길이가 \(a=4\), \(b=5\), \(c=6\)인 \(\triangle ABC\)는 예각삼각형, 직각삼각형, 둔각삼각형 중 어떤 삼각형인지 판별하시오.
⚠️ 함정 분석 보기
⚠️ 핵심 분석 (틀려야 는다!): 모든 각의 코사인 값을 계산할 필요가 없다. 삼각형의 가장 큰 각(가장 긴 변의 대각)이 둔각인지 예각인지만 판별하면 된다. 가장 긴 변 \(c=6\)에 대한 \(\cos C\)의 **부호**만 확인해 보자! \(\cos C > 0\)이면 예각, \(\cos C < 0\)이면 둔각이다.

03. 형성평가 (Final Check)

6. \(\triangle ABC\)에서 다음 관계식이 성립할 때, 이 삼각형은 어떤 삼각형인지 말하시오. (단, \(a \neq b\))
\[ a \cos A = b \cos B \]
🎯 풀이 전략 보기
💡 전략: 이전에는 사인법칙 변형 공식을 대입했다면 이번엔 코사인법칙 변형 공식(Type B)을 대입하여 전체를 변의 길이(\(a, b, c\))에 대한 식으로 바꾸어 전개해 보자.

⭐ 도전 문제 1

📖 출처: 수능특강 수학Ⅰ · 사인법칙·코사인법칙 · 예제 3

난이도 ★★★★ (학습지 max+1, 6분) · 핵심: 조건식 → ±분기 → 모순 기각 → 직각 판별 → 넓이

\(\overline{AB} = 4\), \(\overline{BC} = 3\)인 삼각형 \(ABC\)에 대하여 다음 식이 성립할 때, 삼각형 \(ABC\)의 넓이는?
\[ (\sin B - 2\cos A \sin C)^2 + \cos^2 B = 1 \]
선지: ① \(\dfrac{3\sqrt{7}}{2}\)    ② \(2\sqrt{7}\)    ③ \(\dfrac{5\sqrt{7}}{2}\)    ④ \(3\sqrt{7}\)    ⑤ \(\dfrac{7\sqrt{7}}{2}\)
📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 수Ⅰ 단독 우회 풀이:
  1. 피타고라스 항등식: \((\sin B - 2\cos A \sin C)^2 = 1 - \cos^2 B = \sin^2 B\) → \(\sin B - 2\cos A \sin C = \pm \sin B\)
  2. + 경우: \(\cos A \sin C = 0\) → \(\angle A = 90°\) → \(\overline{BC}^2 = \overline{AB}^2 + \overline{AC}^2\)인데 \(9 < 16\)이라 모순
  3. − 경우: \(\sin B = \cos A \sin C\). 사인법칙 \(b = 2R\sin B,\; c = 2R\sin C\)로 변 환산 → \(b = c \cdot \cos A\). 코사인법칙으로 \(\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}\) 대입 → \(a^2 + b^2 = c^2\) → \(\angle C = 90°\)
  4. 빗변 \(c = \overline{AB} = 4\), 다리 \(a = \overline{BC} = 3\) → \(b = \overline{CA} = \sqrt{7}\) → 넓이 \(= \dfrac{1}{2} \cdot 3 \cdot \sqrt{7} = \dfrac{3\sqrt{7}}{2}\)
※ 원 해설은 덧셈정리(\(\sin B = \sin(A+C)\))를 사용하나, 위 우회 풀이로 코사인법칙·사인법칙·피타고라스 항등식만으로 완결 가능. 자세한 풀이는 수능특강 해설지 예제 3 참조.

⭐ 도전 문제 2

📖 출처: 수능특강 수학Ⅰ · 사인법칙·코사인법칙 · 기본 6

난이도 ★★★ (학습지 max 동일, 7분) · 핵심: 보조 삼각형 코사인법칙 2회 연쇄 + 사인법칙 외접원 R

이등변삼각형 \(\triangle ABC\)(\(\overline{AB} = \overline{AC} = 6\))의 변 \(\overline{AC}\) 위의 점 \(E\)와 변 \(\overline{BC}\) 위의 점 \(D\)에 대하여 \(\overline{AE} = \overline{DE},\; \overline{AD} = 3,\; \overline{BD} = 3\)일 때, 삼각형 \(DEC\)의 외접원의 반지름의 길이를 구하시오.
📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 수Ⅰ 단독 풀이 흐름:
  1. \(\triangle ADE\)에서 코사인법칙으로 \(\cos(\angle A)\) 결정 → \(\sin(\angle A)\) 도출
  2. \(\triangle ABC\)에서 코사인법칙으로 \(\overline{BC}\) 계산
  3. \(\triangle ABC\)에서 사인법칙으로 \(\sin(\angle ACB)\) 계산
  4. \(\angle DCE = \angle ACB\)이므로 \(\triangle DEC\)에서 사인법칙: \(2R = \dfrac{\overline{DE}}{\sin(\angle DCE)}\)

핵심: 원 해설은 반각 공식·삼각방정식을 거치지만, 위 흐름으로 코사인법칙 2회 + 사인법칙 2회만으로 완결 가능 (배각·반각·삼각방정식 불필요).

※ 도형 그림과 원 해설은 수능특강 해설지 기본 6에서 확인.
