C 도입 운영 가이드 — 단원 I-2 마인드맵

발문: "지난 두 차시에서 배운 핵심 한 줄을 적어 보자. 그 한 줄이 어떻게 14문항으로 펼쳐지는가?"

예상 반응 ①"연속이란 \(\lim = f(a)\) 한 줄."
피드백: 정확. 그 한 줄이 ① 3조건 점검 ② 사칙 ③ EVT/IVT로 가지를 친다. 칠판에 마인드맵으로 시각화.
예상 반응 ②"분모가 0이 안 되어야 한다."
피드백: 사칙의 핵심 조건. 오늘 Q3·Q6·Q11에서 반복 등장. "분자 \(\to 0\) → 미정계수 결정" 패턴 환기.
예상 반응 ③"부호가 바뀌면 실근이 있다."
피드백: IVT의 응용. 오늘 Q12·Q14에서 반복. "보조함수 정의 → 양 끝 부호 → IVT" 4단계 표준 답안 패턴.

05·06차시 연결: 정의 1조 + 사칙 4공식 + EVT 1개 + IVT 1개 = 7개의 도구. 14문항은 이 7개 도구를 각각 2번씩 두드린다. 풀이 중 막히면 "어느 도구를 쓸 차례인지" 자문하는 메타 전략을 권장.

D 개념 지도 노트 — 단원 I-2 압축 정리 · 14문항 매핑

UNIT I-2 SUMMARY
연속 ⇔ \( f(a)\,\exists\ \&\ \lim f\,\exists\ \&\ \lim f = f(a) \)  |  사칙: \(k f,\ f \pm g,\ fg\) 연속, \(\frac{f}{g}\)는 \(g(a) \neq 0\)에서.  |  EVT·IVT는 닫힌+연속 가정에서 작동.
📌 판서 시연 — 14문항 도구 매핑:
  1. 정의 점검 (Q1·Q5·Q6 그래프): 3조건 표를 그려라.
  2. 구간 연속성 (Q2): 정의역 분리 + 끝점 한쪽 극한 점검.
  3. 분모 영점 (Q3·Q11): 분자·분모 모두 다항이면 분모 \(= 0\)인 점만 불연속.
  4. 경계점 미정계수 (Q6·Q7·Q8·Q9): "분모 \(\to 0\) + 극한 실수 → 분자 \(\to 0\)" 한 줄로 미정계수 결정.
  5. 사칙 보기 판별 (Q10): 분수의 분모 영점 가능성만 의심.
  6. EVT (Q4): 다항 → 꼭짓점·끝점 비교.
  7. IVT 실근 (Q12·Q14): 보조함수 정의 → 양 끝 부호 → IVT 인용 4단계.
  8. 곱 함수 연속 (Q13): 한 인수 불연속 → 다른 인수가 불연속점에서 영점 (도전 05차시 D2의 일반화).
오개념 ①: "분모 \(\to 0\)이면 발산이다." → 분자도 \(\to 0\)이면 \(\frac{0}{0}\) 부정형, 약분 후 극한이 실수가 될 수 있음.
오개념 ②: "Q5의 \(\lim_{x \to 0} f(x)\)는 \(f(0)\)과 같다." → 그래프상 \(x = 0\)에서 함숫값이 정의되지 않거나 곡선 끝일 수 있음. 정의역 \((0, 5)\)에 주목.
오개념 ③: "Q14에서 \(f(-1) \gt 0\)일 수도 있다." → 만약 \(f(-1) \gt 0\)이면 \(f(-1)\)과 \(f(1) \lt 0\)의 부호가 달라 \((-1, 1)\)에 IVT 실근이 추가 발생 → 총 3실근 ≠ "오직 두 개" 조건 위배.

E 문항별 해설 카드 (14문항 — 기초 1~3 · 기본 4~12 · 도전 13·14)

기초1\(x=3\)에서 연속 판정
(1) \(\frac{2}{x+1}\)  (2) \(x+1\,(x\neq 3)\), \(-2\,(x=3)\)
(1) \(f(3) = \frac{2}{4} = \frac{1}{2}\), \(\lim_{x\to 3} = \frac{1}{2}\) → 일치 → 연속. (2) \(\lim_{x\to 3}(x+1) = 4 \neq f(3) = -2\) → ③ 깨짐 → 제거 가능 불연속.
정답: (1) 연속 (2) 불연속
기초2연속인 구간
(1) \(x^2 + x + 3\)  (2) \(\sqrt{2x - 4}\)
(1) 다항 → \((-\infty, \infty)\). (2) \(2x - 4 \geq 0\) ⇒ \(x \geq 2\). 끝점 \(x=2\)에서 우극한 \(= f(2) = 0\) → \([2, \infty)\).
정답: (1) \((-\infty,\infty)\) (2) \([2,\infty)\)
기초3유리식 연속성
\(f = x^2+1\), \(g = x^2-2x-3\). (1) \(\frac{f}{g}\) (2) \(\frac{g}{f}\)
(1) \(g = (x-3)(x+1) = 0\) ⇒ \(x = 3, -1\) 불연속, 그 외 연속. (2) \(f = x^2+1 \geq 1 \gt 0\) → 분모 \(\neq 0\) → 모든 실수 연속.
정답: (1) \(x = 3, -1\) 외 연속  (2) 모든 실수 연속
기본4닫힌구간 최대·최소
(1) \(x^2 + 2x - 5\) on \([-4, 1]\) (2) \(\sqrt{x-2}\) on \([3, 6]\)
(1) 꼭짓점 \(x = -1 \in [-4, 1]\), \(f(-1) = -6\) (최솟). \(f(-4) = 3, f(1) = -2\) → 최댓 \(3\). (2) 증가 → 최솟 \(f(3) = 1\), 최댓 \(f(6) = 2\).
정답: (1) 최댓 \(3\), 최솟 \(-6\) (2) 최댓 \(2\), 최솟 \(1\)
기본5그래프 보기 판정 (calc_07_q5.png)
열린구간 \((0, 5)\)의 그래프. ㄱ \(\lim_{x\to 1} f = 4\) ㄴ \(\lim_{x\to 0} f = f(0)\) ㄷ 불연속 \(x\)가 2개 ㄹ 극한 없는 \(x\)가 2개.
불연속·극한 후보점은 \(x = 1, 3, 4\). 표로: \(x=1\): 좌극한 \(=\) 우극한 \(=\) 곡선의 ○ \(= 4\), \(f(1) = \bullet = 1\). 극한 존재(4), 연속 아님 (4 \(\neq\) 1). \(x=3\): 좌극한 \(=\) 곡선의 ○ \(= 3\), 우극한 \(=\) 수평선 \(= 1\). 극한 미존재, 불연속. \(x=4\): 좌극한 \(=\) 수평선의 ○ \(= 1\), 우극한 \(=\) 우하향 곡선의 ○ \(= 3\). 극한 미존재, 불연속. ㄱ ✓ (좌·우극한 모두 4). ㄴ 정의역 \((0,5)\)에 \(x=0\) 미포함, \(f(0)\) 부재 → 진위 부적절(주의: 학습지 hint는 \(x=2\) 표기 오타로 보임, 실제 후보는 \(x = 1, 3, 4\)). ㄷ 불연속점 = \(\{1, 3, 4\}\) 3개 → ㄷ 거짓. ㄹ 극한 미존재 \(= \{3, 4\}\) 2개 → ㄹ ✓.
정답: ㄱ, ㄹ. (교과서 정답 동일.)
학습지 hint 오류 발견: "각 후보점 \(x = 1, 2, 4\)"라고 기재되어 있으나 그래프에는 \(x=2\)가 후보가 아니며 \(x=3\)이 후보. 학습지 본문 별도 PR로 정정 권장.
기본6분기식 미정계수 \(a, b\)
\(\frac{x^2+ax+b}{x-2}\,(x\neq 2)\), \(3\,(x=2)\) 모든 실수 연속.
\(x=2\) 연속 ⇒ \(\lim_{x\to 2}\frac{x^2+ax+b}{x-2} = 3\). 분모 \(\to 0\), 극한 실수 ⇒ 분자 \(\to 0\) ⇒ \(4 + 2a + b = 0\). 분자 \(= (x-2)(x-r)\) 꼴. \(\lim = 2 - r = 3\) ⇒ \(r = -1\). 분자 \(= (x-2)(x+1) = x^2 - x - 2\). 비교: \(a = -1, b = -2\).
정답: \(a = -1,\ b = -2\)
기본7경계점 미정계수 — \(a\)의 곱
\(3x^2-1\,(x\geq a)\), \(2-5x\,(x \lt a)\) 모든 실수 연속.
경계 \(x=a\): \(2 - 5a = 3a^2 - 1\) ⇒ \(3a^2 + 5a - 3 = 0\). 비에타: 두 근의 곱 \(= \frac{-3}{3} = -1\).
정답: \(-1\)
기본8근호 미정계수 + \(f(3)\)
\(x \neq 2\)에서 연속, \((x-3)f(x) = \sqrt{x+a} - 1\). \(f(3) = ?\)
\(x = 3\) 연속 ⇒ 양변 \(x \to 3\)에서 좌변 \(\to 0\) ⇒ 우변 \(\to 0\) ⇒ \(\sqrt{3+a} - 1 = 0\) ⇒ \(a = -2\). \(f(x) = \frac{\sqrt{x-2}-1}{x-3}\) (\(x \neq 3\)). 유리화 \(\frac{(\sqrt{x-2}-1)(\sqrt{x-2}+1)}{(x-3)(\sqrt{x-2}+1)} = \frac{1}{\sqrt{x-2}+1}\). \(x \to 3\) ⇒ \(\frac{1}{1+1} = \frac{1}{2}\). 연속이므로 \(f(3) = \frac{1}{2}\).
정답: \(f(3) = \frac{1}{2}\)
기본9극한 조건 → \(f(1)\)
\(x \neq 0\)에서 연속, \(\lim_{x\to 1}\frac{(\sqrt{x}-1)f(x)}{x^3-1} = 5\). \(f(1) = ?\)
\(\sqrt{x} - 1 = \frac{x-1}{\sqrt{x}+1}\), \(x^3 - 1 = (x-1)(x^2+x+1)\). 약분 후 \(\frac{f(x)}{(\sqrt{x}+1)(x^2+x+1)}\). \(x \to 1\) ⇒ \(\frac{f(1)}{2 \cdot 3} = \frac{f(1)}{6} = 5\) ⇒ \(f(1) = 30\).
정답: \(f(1) = 30\)
기본10사칙 보기 — 반드시 연속
ㄱ \(2f - g\) ㄴ \(\{f\}^2\) ㄷ \(\frac{f}{g}\) ㄹ \(\frac{2}{\{f\}^2+1}\)
ㄱ·ㄴ 사칙(실수배·차·곱) → 항상 연속 ✓. ㄷ \(g(a) = 0\) 가능 → 보장 X. ㄹ 분모 \(\geq 1 \gt 0\) → 항상 양수 → 항상 연속 ✓.
정답: ㄱ, ㄴ, ㄹ
기본11분수함수 최댓값 \(a\)
\([1, 3]\)에서 \(\frac{a}{|x-4|}\) 최댓 \(= 8\). \(a = ?\)
\(x \lt 4\)에서 \(|x-4| = 4-x\). \(f(x) = \frac{a}{4-x}\) 증가. 최댓값 \(x = 3\)에서 \(\frac{a}{1} = a = 8\).
정답: \(a = 8\)
기본12IVT 실근 구간 식별
\(2x^3 - 3x^2 - x + 1 = 0\) 세 실근. \((-1,0), (0,1), (1,2), (2,3)\) 중?
\(f(-1) = -3, f(0) = 1, f(1) = -1, f(2) = 3, f(3) = 25\). 부호 변화: ㄱ −→+, ㄴ +→−, ㄷ −→+. (ㄹ +→+ 없음.)
정답: ㄱ, ㄴ, ㄷ
도전(서술)13곱 함수 두 경계 연속 (서술형)
\(f = x^2+ax+b\), \(g(x) = x^2\,(|x|\leq 1)\), \(2\,(|x|\gt 1)\). \(fg\)가 모든 실수 연속. \(a, b = ?\)
\(g\)는 \(x = \pm 1\)에서 점프 불연속. \(f\)는 다항 → 연속. \(x = 1\) 연속: 좌극한 \(= f(1) \cdot 1^2 = f(1)\), 우극한 \(= f(1) \cdot 2\). 일치 ⇒ \(f(1) = 2 f(1)\) ⇒ \(f(1) = 0\) ⇒ \(1 + a + b = 0\). \(x = -1\) 연속: 좌극한 \(= f(-1) \cdot 2\), 우극한 \(= f(-1) \cdot (-1)^2 = f(-1)\). 일치 ⇒ \(f(-1) = 0\) ⇒ \(1 - a + b = 0\). 연립: 두 식 합 \(\Rightarrow 2 + 2b = 0 \Rightarrow b = -1\). 차 \(\Rightarrow 2a = 0 \Rightarrow a = 0\).
정답: \(a = 0,\ b = -1\). 서술 채점: ① 두 경계점 좌·우극한 식 명시 ② \(f(\pm 1) = 0\) 도출 근거 ③ 연립 풀이.
도전(서술)14IVT 두 그래프 교점 증명 (서술형)
\(f\) 연속, \(f(-3)f(-1) \lt 0\), \(f(1)f(3) \lt 0\), \(f(1) = -1\). \(f(x) = 0\)이 \((-3, 3)\)에 오직 두 실근. \(y = f(x)\)와 \(y = -2x^3\)이 \(-1 \lt x \lt 1\)에 교점 적어도 하나.
① 두 실근의 위치: \(f(-3)f(-1) \lt 0\) → IVT로 \((-3, -1)\)에 실근 1개. \(f(1) = -1 \lt 0\), \(f(1)f(3) \lt 0\) ⇒ \(f(3) \gt 0\) → \((1, 3)\)에 실근 1개. "오직 두 개"이므로 \((-1, 1)\)에는 \(f = 0\)이 없음. ② \(f(-1)\) 부호: 만약 \(f(-1) \gt 0\)이면 \(f(-1) \gt 0, f(1) = -1 \lt 0\) 부호 다름 → IVT로 \((-1, 1)\)에 실근 추가 → 모순. 따라서 \(f(-1) \lt 0\). ③ 보조함수 \(h(x) = f(x) - (-2x^3) = f(x) + 2x^3\). \(f\) 연속 + \(2x^3\) 연속 → \(h\) 연속. ④ \(h(-1) = f(-1) + 2(-1)^3 = f(-1) - 2 \lt -2 \lt 0\). \(h(1) = -1 + 2 = 1 \gt 0\). 부호 변화. ⑤ IVT에 의해 \(h(c) = 0\)인 \(c \in (-1, 1)\) 적어도 하나 ⇒ \(f(c) = -2c^3\) ⇒ 두 그래프 교점.
정답 (증명 완료). 서술 채점: ① 두 실근 구간 식별 ② \(f(-1)\) 부호 결정 ③ 보조함수 정의 ④ 양 끝 부호 ⑤ IVT 인용. 5점 만점에서 단계 누락마다 -1.

G 성찰지 운영 — 3유형 우수 답안 예 (중단원 단위)

유형 1: 단원 종합 통찰 "단원 I-2를 마무리하면서 \(\lim = f(a)\) 한 줄이 결국 사칙·합성·IVT·EVT 모든 정리의 출발점이라는 걸 알게 됐다. 흐름도를 그려보니 정리들이 모두 정의 한 줄에서 가지처럼 뻗어 나오고, 정리들의 '연속 가정'이 사실 이 정의를 끌어다 쓰는 것이었다. 14문항 풀이 중 어느 도구를 쓸 차례인지 자문하는 메타 전략이 효과적이었다."
유형 2: 자주 틀린 유형 분석 "구간별로 정의된 \(fg\)에서 경계점의 좌극한·우극한 식을 어느 구간 식으로 잡아야 할지 자주 헷갈렸다 (Q13). '경계점 표시 → 좌극한은 어느 구간 식 → 우극한은 어느 구간 식 → 일치 조건'을 표로 그려놓는 습관을 들였고, 그 다음부터 곱·합 어떤 형태가 와도 같은 절차로 풀린다. 자기점검 체크리스트 효과 큼."
유형 3: 다음 단원 확장 "다음 단원 미분에서 미분계수 \(\lim_{h\to 0}\frac{f(a+h)-f(a)}{h}\)를 보니 결국 \(h \to 0\)일 때 분자도 \(0\)으로 가는 \(\frac{0}{0}\) 부정형. Q8·Q9에서 익힌 유리화·인수분해가 그대로 쓰일 것 같다. '미분가능 ⇒ 연속' 명제도 결국 연속이 더 약한 조건이라는 얘기라, 이번 단원의 연속 정의가 미분의 기반 가정으로 이어진다는 가설을 세웠다."
