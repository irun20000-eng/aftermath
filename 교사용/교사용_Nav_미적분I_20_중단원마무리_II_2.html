한 줄 핵심: "\(f'(x)\)의 부호 → 증감 → 극값·최대최소·실근·부등식·속도"라는 단 하나의 흐름으로 이번 단원 전체 14문항을 꿰뚫을 수 있다. 도구를 꺼내는 판단력이 이 차시의 진짜 목표.

[12미적-02-04~08] II-2 도함수의 활용 종합 + 서술형 풀이 문장 구성 능력.
선수학습
12~19차시 전체. 접선의 방정식, 평균값 정리, 증감·극값, 최대·최소, 방정식·부등식 활용, 속도·가속도.

C 단원 회고 발문 (Warm-up 대체)

발문: "12~19차시에서 가장 자주 막혔던 한 가지 도구는? 그 도구가 필요한 '신호'는 무엇인가?"

예상 반응 ①"사차함수 극댓값 조건. 판별식 \(D\geq 0\)으로 통합해 틀렸다."
피드백: 신호 = "사차함수 + 극댓값". 도함수가 삼차식이므로 실근 1개·3개 분기를 먼저 확인. \(f'(x)=0\)이 서로 다른 세 실근 → \(f'\) 부호 변화표 → 극대 존재.
예상 반응 ②"닫힌구간 최대·최소에서 양 끝점을 후보에서 빼먹었다."
피드백: 신호 = "닫힌구간 + 최대·최소". 루틴: ① 영점(\(f'=0\)) ② 구간 양 끝점 ③ 모두 함숫값 계산 ④ 비교. 영점만 쓰면 무조건 감점.
예상 반응 ③"\(|f(x)|=k\) 그래프 개형을 못 그렸다."
피드백: 신호 = "절댓값 + 방정식 실근 개수". \(y=|f(x)|\)의 개형은 \(f\)의 음의 부분을 위로 접은 것. 꺾임점과 두 극댓값의 높이 비교로 \(k\)의 범위 결정.

대단원 예고: 다음 차시(21차시 대단원 평가 II)는 II-1(미분계수·도함수)과 II-2(도함수의 활용) 전 범위 종합. 평균변화율·미분계수 정의까지 포함.

D 개념 지도 노트 — 단원 종합 도구 트리 · 판서 · 오개념

DECISION TREE — II-2 도함수의 활용 도구 선택
접선 → 접점 미지수화  |  증가/감소 → \(f'(x) \geq 0\) (판별식)  |  극값 → \(f'=0\) 부호 변화표  |  닫힌구간 최대·최소 → 영점+양끝점 후보  |  실근 개수 → 보조함수 + \(y=k\) 교점  |  부등식 → 최솟값 \(\gt 0\)  |  속도·가속도 → 미분(위치→속도→가속도)  |  평균값 정리 → 평균속도=순간속도 존재
📌 판서 시연 — 13번 \(|f(x)|=k\) 5실근 접근:
  1. \(f(x)=2x^3-3x^2-12x+4\) 극값 계산: \(f'(x)=6(x-2)(x+1)=0\). 극대 \(f(-1)=11\), 극소 \(f(2)=-16\).
  2. \(y=|f(x)|\) 개형 스케치: 음수 부분 위로 접음. \(x=2\)에서 \(|f|=16\) (새로운 극대), \(f=0\)인 점에서 꺾임.
  3. \(y=k\) 수평선과 5개 교점 조건: 두 극댓값(11과 16) 사이 → \(11 \lt k \lt 16\).
  4. 경계 확인: \(k=11\)이면 4개, \(k=16\)이면 4개 → 부등호 엄격(strict).
오개념 ①: 닫힌구간 최솟값을 극솟값과 혼동 → 영점에서 극소라도 양 끝점과 비교 필수.
오개념 ②: \(f'(x)\geq 0\) 이면 증가라고만 외움 → \(f'=0\)인 점이 고립점(단일)이면 여전히 전체 증가.
오개념 ③: 사차함수 극댓값 조건에서 \(f'(x)=0\)의 근이 1개인 경우 극값 없음(접촉) vs. 3개인 경우 극대·극소 구분 — 반드시 부호 변화표 작성.

E 문항별 해설 카드 (14문항 종합)

기초1접선의 방정식 — 기울기 대입
곡선 \(y=x^3-3x^2+4\) 위의 점 \((1,2)\)에서의 접선의 방정식.
\(f'(x)=3x^2-6x\). 기울기 \(f'(1)=3-6=-3\). 접선: \(y-2=-3(x-1)\) → \(y=-3x+5\).
정답: \(y = -3x + 5\)
기초2속도와 가속도 — \(t=1\)
위치 \(x=2t^3-5t\). \(t=1\)에서 속도·가속도.
속도 \(v=6t^2-5\). \(v(1)=6-5=1\). 가속도 \(a=12t\). \(a(1)=12\).
정답: 속도 \(1\), 가속도 \(12\)
기초3외부 점에서 접선 → 도형 넓이
점 \((0,2)\)에서 곡선 \(y=-x^3+x+4\)에 그은 접선과 \(x\)축, \(y\)축으로 둘러싸인 넓이.
접점 \((t, -t^3+t+4)\). \(f'(t)=-3t^2+1\). 접선이 \((0,2)\) 통과: \(-t^3+t+4-2=(-3t^2+1)(0-t)\) → \(-t^3+t+2=3t^3-t\) → \(4t^3-2t-2=0\) → \(2t^3-t-1=0\) → \((t+1)(2t^2-2t+1)=0\). \(t=-1\) (실수근). 접점 \((-1, 4)\). 기울기 \(f'(-1)=-3+1=-2\). 접선 \(y=-2x+2\). \(x\)절편 \(1\), \(y\)절편 \(2\). 삼각형 넓이 \(=\dfrac{1}{2}\cdot 1\cdot 2=1\).
정답: 넓이 \(1\)
핵심: 외부 점 → 접선 문제는 접점을 미지수 \(t\)로 두고 "통과 조건" 연립. 힌트의 \(2t^3+2=0\)은 \(-t^3+t+4-(-3t^2+1)(-t)-2=0\) 정리 과정의 중간 단계.
기본4공통접선 — 두 조건 연립
두 곡선 \(f(x)=x^3+ax+b\), \(g(x)=cx^2+1\)이 점 \((1,2)\)에서 만나고 공통접선을 가질 때, \(a,b,c\).
조건 ①: \(f(1)=1+a+b=2\) → \(a+b=1\). 조건 ②: \(g(1)=c+1=2\) → \(c=1\). 공통접선: \(f'(1)=g'(1)\). \(f'(x)=3x^2+a\), \(g'(x)=2cx\). \(3+a=2c=2\) → \(a=-1\), \(b=2\).
정답: \(a=-1,\ b=2,\ c=1\)
기본5평균값 정리 — \(c=1\), \(a\) 결정
\(f(x)=x^3+x+1\), 구간 \([-1,a]\), \(c=1\). \(a\)의 값 (\(a\gt 1\)).
\(f(-1)=-1\), \(f'(1)=4\). \(\dfrac{f(a)-f(-1)}{a+1}=4\) → \(\dfrac{a^3+a+2}{a+1}=4\). 분자 \(=(a+1)(a^2-a+2)\) → \(a^2-a+2=4\) → \(a^2-a-2=0\) → \((a-2)(a+1)=0\). \(a\gt 1\)이므로 \(a=2\).
정답: \(a = 2\)
기본6전 구간 증가 조건 — 판별식
\(f(x)=x^3+ax^2+3x+1\)이 모든 실수에서 증가하는 \(a\)의 범위.
\(f'(x)=3x^2+2ax+3\geq 0\) (모든 \(x\)). 판별식 \(D/4=a^2-9\leq 0\) → \(-3\leq a\leq 3\).
정답: \(-3 \leq a \leq 3\)
기본7🖼 도함수 그래프 해석 → 극솟값 ★ 이미지 확인
최고차 계수 \(-1\)인 삼차함수 \(f(x)\)의 도함수 \(y=f'(x)\) 그래프(images/calc_20_q7.png)에서 극댓값이 7일 때 극솟값.
그림: \(f'(0)=0\), \(f'(2)=0\), 위로 볼록한 포물선(최고차 음수) → \(f'(x)=-3x(x-2)=-3x^2+6x\). 적분: \(f(x)=-x^3+3x^2+C\). \(f'\) 부호: \(x\lt 0\) 음, \(0\lt x\lt 2\) 양, \(x\gt 2\) 음 → \(x=0\) 극소, \(x=2\) 극대. 극댓값 \(f(2)=-8+12+C=4+C=7\) → \(C=3\). 극솟값 \(f(0)=C=3\).
정답: 극솟값 \(3\)
이미지 대조 포인트: \(f'(x)\)가 \(x=0\)과 \(x=2\)에서 0이 되고 위로 볼록한 포물선인지 PNG 확인. 힌트 검증 완료: 직접 계산으로 극솟값 3 확인.
기본8닫힌구간 최솟값 → \(a\)
\([0,3]\)에서 \(f(x)=2x^3-3x^2+a\)의 최솟값이 5. \(a\).
\(f'(x)=6x(x-1)=0\) → \(x=0,1\). 후보: \(f(0)=a\), \(f(1)=a-1\), \(f(3)=a+27\). 최솟값 \(=f(1)=a-1=5\) → \(a=6\).
정답: \(a = 6\)
기본9두 곡선 세 교점 → \(k\) 범위
\(y=x^3+4x^2\), \(y=-2x^2-9x+k\)가 서로 다른 세 점에서 만나는 \(k\)의 범위.
\(x^3+6x^2+9x=k\). \(g(x)=x^3+6x^2+9x\). \(g'(x)=3(x+1)(x+3)=0\) → 극대 \(g(-3)=0\), 극소 \(g(-1)=-4\). 수평선 \(y=k\)가 서로 다른 세 교점 ⇔ 극솟값 \(\lt k\lt\) 극댓값 → \(-4\lt k\lt 0\).
정답: \(-4 \lt k \lt 0\)
기본10부등식 항상 성립 → \(k\)
\(x\gt 0\)에서 \(f(x)\gt g(x)\). \(k\)의 범위.
\(h(x)=f(x)-g(x)=2x^3+9x^2-24x+(k-1)\). \(h'(x)=6(x+4)(x-1)\). \(x\gt 0\)에서 \(x=1\) 극소. \(h(1)=2+9-24+k-1=k-14\gt 0\) → \(k\gt 14\).
정답: \(k \gt 14\)
기본11속도 일치 순간 두 점 거리
\(x_P=\dfrac{2}{3}t^3+18t-20\), \(x_Q=6t^2-7\). 속도 같아지는 순간 거리.
\(v_P=2t^2+18\), \(v_Q=12t\). \(2t^2-12t+18=0\) → \((t-3)^2=0\) → \(t=3\). \(x_P(3)=18+54-20=52\), \(x_Q(3)=54-7=47\). 거리 \(=|52-47|=5\).
정답: \(5\)
기본12사차함수 극댓값 갖기 위한 \(a\) 범위
\(f(x)=x^4-4x^3-ax^2\)이 극댓값을 갖기 위한 \(a\)의 범위.
\(f'(x)=4x^3-12x^2-2ax=2x(2x^2-6x-a)\). 사차(최고차 양수) 함수가 극댓값을 가지려면 \(f'(x)=0\)이 서로 다른 세 실근 → \(2x^2-6x-a=0\)이 \(x=0\)이 아닌 두 실근. 판별식: \(D/4=9+2a\gt 0\) → \(a\gt -\dfrac{9}{2}\). 단 \(a=0\)이면 \(f'(x)=2x(2x-6)=0\) → 이중근, 극대 없음. 따라서 \(a\gt -\dfrac{9}{2}\)이고 \(a\neq 0\).
정답: \(a \gt -\dfrac{9}{2}\)이고 \(a \neq 0\)
핵심 포인트: \(a\gt 0\)이면 이차식 \(2x^2-6x-a=0\)의 두 근 모두 \(x=0\)과 다름(자동 성립). \(a\lt 0\)이면 \(a\gt -9/2\) 추가 확인 필요. \(a=0\) 경계는 이중근으로 극대 없음.
도전13⭐ 서술형 — \(|f(x)|=k\) 5실근 조건
\(f(x)=2x^3-3x^2-12x+4\). \(|f(x)|-k=0\)이 서로 다른 5개의 실근. \(k\)의 값의 범위. 서술 풀이.
① \(f'(x)=6x^2-6x-12=6(x-2)(x+1)=0\). 극대 \(f(-1)=-2-3+12+4=11\), 극소 \(f(2)=16-12-24+4=-16\). ② \(y=|f(x)|\) 개형: 음수 부분(극소 근방) 위로 접음 → \(x=2\) 근방에서 \(|f(2)|=16\) (새 극대). 꺾임점 \(f(x)=0\)인 세 실수근에서 높이 0. ③ \(y=k\)가 5개 교점 ⇔ \(|f|\)의 작은 극대(11)와 큰 극대(16) 사이. → \(11\lt k\lt 16\).
정답: \(11 \lt k \lt 16\). 서술 채점: ① 극값 도출 ② \(|f|\) 개형 설명(두 극대 11·16) ③ \(k\) 범위 근거. 3단계 모두 필수.
발표 유도: "\(k=11\)일 때 교점이 4개인 이유"를 학생에게 그래프로 설명하게 한다 — 꺾임점이 극대와 겹쳐 실근이 중복됨.
도전14⭐ 서술형 — 평균값 정리 + 구간 단속 실생활
제한 속도 80 km/h 도로, 15 km 구간을 10분(= \(\tfrac{1}{6}\) 시간)에 통과. 평균값 정리로 제한 속도 초과 순간 존재 증명. 서술 풀이.
① 시각 \(t\)(시간)에서 위치 \(x(t)\). \(x(t)\)는 \([0, \tfrac{1}{6}]\)에서 연속, \((0, \tfrac{1}{6})\)에서 미분가능 가정. ② 구간 평균속도 \(=\dfrac{x(1/6)-x(0)}{1/6}=\dfrac{15}{1/6}=90\) km/h. ③ 평균값 정리: 어떤 \(c\in(0,\tfrac{1}{6})\)가 존재하여 \(x'(c)=90\) km/h. 즉 순간속도 90 km/h인 시각 반드시 존재. ④ \(90\gt 80\) → 제한 속도를 초과한 순간이 반드시 있었다.
정답: 제한 속도 초과 순간 반드시 존재. 서술 채점: ① 가정(연속·미분가능) ② 평균속도 90 도출(단위 환산) ③ 평균값 정리 적용 ④ 결론. 4단계.
