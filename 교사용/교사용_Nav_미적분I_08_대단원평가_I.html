C 도입 운영 가이드 — 단원 I 마인드맵

발문: "극한과 연속을 한 줄로 묶는다면? 그 한 줄이 7차시 동안 어떤 가지로 펼쳐졌는가?"

예상 반응 ①"\(\lim_{x \to a} f(x) = f(a)\) 한 줄."
피드백: 정답. 극한이 함숫값과 같을 때를 연속이라 하니, 단원 I-1·I-2 전부가 이 한 줄에 압축. 마인드맵 시각화.
예상 반응 ②"부정형은 부정형끼리, 연속은 연속끼리."
피드백: 평가에서는 두 축이 한 문항에서 합류한다 (Q3·Q7·Q8·Q9). "부정형 해결로 미정계수 + 연속 조건으로 \(f\) 결정" 패턴.
예상 반응 ③"IVT는 단원 끝에 나온 정리라 잘 모르겠다."
피드백: Q13에서 보조함수 \(h = f - x^3\) 정의 → 양 끝 부호 곱 → 정수 \(a\) 개수 결정. 06차시 표준 절차의 직접 적용.

단원 I 전체 연결: 7차시 = 7개 도구(직관 정의·좌우극한·사칙·부정형·연속 3조건·EVT·IVT). 13문항은 이들을 조합. 풀이 중 "어느 도구 차례인가?" 자문하면 8할은 자동 해결.

D 개념 지도 노트 — 단원 I 압축 정리

UNIT I SUMMARY
극한: 직관 정의 + 좌·우극한 + 사칙 + 부정형 4종(\(\tfrac{0}{0}, \tfrac{\infty}{\infty}, \infty-\infty, 0\cdot\infty\)).  |  연속: \(\lim = f(a)\) 3조건 + 사칙 + EVT + IVT.
📌 13문항 도구 매핑:
  1. 극한 사칙 (Q1·Q2): 곱·합·차의 분리 + 정확한 계산.
  2. 부정형 \(\tfrac{0}{0}\) + 미정계수 (Q3·Q9): "분모 \(\to 0\) + 극한 실수 → 분자 \(\to 0\)" → 미정계수 결정.
  3. 도형 극한 (Q4): 연립 → 변수 분리 → 극한.
  4. 샌드위치 (Q5): \(x\)로 나누어 양 끝 함수 ⇒ 사이 함수 극한.
  5. 연속 정의 (Q6): 좌·우극한 + 함숫값 표 작성.
  6. 곱 함수 연속 (Q7·Q12): 한 인수의 영점이 다른 인수의 불연속을 흡수.
  7. 유리함수 + EVT (Q8·Q11): 분모 영점 조사 + 닫힌+연속 가정 점검.
  8. \(\infty\) 사칙 + 유계 (Q10): 발산하는 분모로 나누면 유계 함수는 \(\to 0\).
  9. IVT + 보조함수 (Q13): \(h = f - g\) + 양 끝 부호 곱 \(\lt 0\).
오개념 ①: "\(\sqrt{x^2} = x\)" — \(x \to -\infty\)일 때 \(\sqrt{x^2} = |x| = -x\). Q2 ⑤의 핵심 함정.
오개념 ②: "분수의 분모가 \(0\)이면 무조건 발산한다." — 분자도 \(0\)이면 \(\tfrac{0}{0}\) 부정형, 약분 후 실수 극한 가능.
오개념 ③: "Q8에서 \([-2, 0]\)이 닫힌구간이니 EVT로 최대·최소 모두 가진다." — \(x = -2\)에서 함수 정의 자체가 안 됨(분모 0). EVT 가정 "연속"이 깨짐.
오개념 ④: "Q13에서 \(h(-1) \cdot h(1) \leq 0\)도 충분하다." — 등호이면 끝점이 영점일 가능성, 열린구간 \((-1, 1)\) 내부 실근 보장 안 됨. 엄격 부등호 \(\lt 0\).

E 문항별 해설 카드 (객관식 1~8 + 서술형 9~13)

표준1극한 사칙 (\(a\) 결정 → 합)
\(\lim_{x\to 2} f = 3, \lim g = a, \lim fg = -6\). \(\lim(3f + 2g) = ?\)
곱 \(= 3 \cdot a = -6\) ⇒ \(a = -2\). 사칙: \(3 \cdot 3 + 2 \cdot (-2) = 9 - 4 = 5\).
정답: \(5\)
표준2옳지 않은 것 — ⑤ 부호 함정
5개 극한 중 옳지 않은 것은? (① 대입 ② 약분 ③ 유리화 ④ 최고차비 ⑤ 무한대 유리화)
① \(1 - 4 + 1 = -2\) ✓. ② \(\frac{(3x+1)(x-1)}{x-1} \to 4\) ✓. ③ 유리화 \(\frac{x}{x(1+\sqrt{1-x})} \to \frac{1}{2}\) ✓. ④ \(\frac{2}{1} = 2\) ✓. ⑤ \(\sqrt{x^2-2x}-x\) 유리화 \(\frac{-2x}{\sqrt{x^2-2x}+x}\). \(x \to \infty\)에서 \(\sqrt{x^2-2x} \approx x\) → 분모 \(\approx 2x\) → 극한 \(= -1\), 1이 아님.
정답: ⑤. (실제 극한 \(-1\).)
표준3미정계수 \(a, b\)
\(f(x) = ax + b\), \(\lim_{x\to 1}\frac{(x^2+1)f(x)}{x^4-1} = 2\). \(a, b\)?
\(x^4 - 1 = (x-1)(x+1)(x^2+1)\). \(x \to 1\) 분모 \(\to 0\), 극한 실수 → 분자 \(\to 0\) → \(f(1) = a + b = 0\). 즉 \(b = -a\) ⇒ \(f(x) = a(x-1)\). 약분 후 \(\frac{(x^2+1) \cdot a(x-1)}{(x-1)(x+1)(x^2+1)} = \frac{a}{x+1}\). \(x \to 1\): \(\frac{a}{2} = 2\) ⇒ \(a = 4, b = -4\).
정답: \(a = 4,\ b = -4\)
표준4원 + 곡선 도형 극한 (calc_08_q4.png)
원 \(x^2 + (y-a)^2 = a^2\) ∩ 곡선 \(y = -x^2 + a\) 제1사분면 점 \(P\). \(Q\) = \(P\)의 \(y\)축 발, \(C\) = 원의 중심 \((0, a)\). \(\lim_{a\to 0^+}\frac{\overline{OQ}}{\overline{OC}} = ?\)
\(\overline{OC} = a\). 곡선식 \(x^2 = a - y\)를 원에 대입: \((a-y) + (y-a)^2 = a^2\). 치환 \(u = a - y \geq 0\): \(u + u^2 = a^2\) ⇒ \(u^2 + u - a^2 = 0\). \(u = \frac{-1 + \sqrt{1 + 4a^2}}{2}\) (양근). \(y_P = a - u\), \(\overline{OQ} = y_P\). 비 \(\frac{y_P}{a} = 1 - \frac{u}{a}\). \(a \to 0^+\): \(\sqrt{1 + 4a^2} \approx 1 + 2a^2\) ⇒ \(u \approx a^2\) ⇒ \(\frac{u}{a} \approx a \to 0\). 따라서 비 \(\to 1\).
정답: \(1\). [학습지 hint "\(\frac{1}{2}\)"는 오류로 보임 — 실제 극한은 \(1\). 학습지 본문 정정 권장.]
판서 포인트: 도형 극한은 ① 두 식 연립 → ② 변수 분리 → ③ \(a \to 0^+\)에서 차수 비교 3단계. 학생들이 \(u\)의 차수가 \(a^2\)임을 놓치면 답이 틀리므로 차수 비교를 강조.
표준5샌드위치 정리
\(-3x^2 + 2x \leq f(x) \leq x^2 + 2x\). \(\lim_{x\to 0^+}\frac{\{f\}^2}{3x^2 - xf}\)?
양변 \(\div x\) (\(x \gt 0\)): \(-3x + 2 \leq \frac{f}{x} \leq x + 2\). 샌드위치 → \(\frac{f}{x} \to 2\). 분자\(\div x^2\) = \(\{f/x\}^2 \to 4\). 분모\(\div x^2\) = \(3 - \frac{f}{x} \to 1\). 비 \(\to 4 / 1 = 4\).
정답: \(4\)
표준6\(x=0\) 연속 보기
ㄱ \(|x|\) ㄴ \(\frac{x}{x^2-x}\) ㄷ \(\frac{1}{x}\,(x\neq 0), 1\,(x=0)\) ㄹ \(2x\,(x\geq 0), \sqrt{-x}\,(x \lt 0)\)
ㄱ \(|x|\)는 모든 실수 연속 ✓. ㄴ 분모 \(x(x-1) = 0\)에서 \(x=0\) → 정의 X → 불연속. ㄷ \(\lim_{x\to 0} \frac{1}{x}\) 존재 X → 불연속. ㄹ 좌극한 \(= \lim \sqrt{-x} = 0\), 우극한·함숫값 \(= 2 \cdot 0 = 0\) → 일치 → 연속 ✓.
정답: ㄱ, ㄹ
표준7곱 함수 + 미정계수
\((x^2-1) f(x) = ax^3 + bx^2 - ax - b\), \(f\) 모든 실수 연속, \(f(-1) = 1, f(1) = 2\). \(a, b\)?
우변 인수분해: \(ax(x^2-1) + b(x^2-1) = (x^2-1)(ax+b)\). \(x \neq \pm 1\)에서 \(f(x) = ax + b\). 연속 가정에 의해 \(f(\pm 1)\)도 \(\pm a + b\). \(f(-1) = -a + b = 1\), \(f(1) = a + b = 2\). 합 → \(2b = 3 \Rightarrow b = \frac{3}{2}\). 차 → \(2a = 1 \Rightarrow a = \frac{1}{2}\).
정답: \(a = \frac{1}{2},\ b = \frac{3}{2}\)
표준8EVT — \(\frac{3x-2}{x+2}\) 보기
ㄱ \([-4, 0]\) 최댓값 X  ㄴ \([-2, 0]\) 최솟값 O  ㄷ \([0, 2]\) 최솟값 O
\(f\)는 \(x = -2\)에서 분모 \(=0\) → 불연속(무한 발산). ㄱ \([-4, 0]\)은 \(x=-2\) 포함, 그 근방 \(\pm\infty\) 발산 → 최댓값 없음 ✓. ㄴ \([-2, 0]\)의 좌끝 \(x=-2\)에서 정의 자체가 안 됨 → 함수가 그 구간에서 정의되지 않음 → 최댓값·최솟값 모두 부재 → ㄴ 거짓. ㄷ \([0, 2]\)는 연속구간(분모 \(\neq 0\)). EVT 적용 → 최댓·최솟 모두 존재 ✓.
정답: ㄱ, ㄷ
서술9다항함수 결정 + 미정계수 \(k\)
\(\lim_{x\to 1}\frac{f(x)}{x-1} = 2\), \(\lim_{x\to\infty}\frac{f(x) - 3x^2}{2x - 1} = k\). (1) \(f(x) = ?\) (2) \(k = ?\)
② 식: 분모 1차, 극한 실수 \(k\) → 분자 1차 이하. \(f\)는 이차이며 최고차 \(3x^2\) 소거 — 즉 \(f(x) = 3x^2 + (\cdots)x + (\cdots)\). ① 식: 분모 \(\to 0\) + 극한 실수 → 분자 \(\to 0\) → \(f(1) = 0\). 따라서 \(f(x) = (x-1)(3x + c)\) 꼴. \(\lim_{x\to 1}\frac{(x-1)(3x+c)}{x-1} = 3 \cdot 1 + c = 2\) ⇒ \(c = -1\). \(f(x) = (x-1)(3x-1)\). (2) \(f(x) - 3x^2 = (x-1)(3x-1) - 3x^2 = 3x^2 - 4x + 1 - 3x^2 = -4x + 1\). \(\lim_{x\to\infty}\frac{-4x + 1}{2x - 1} = -2\). \(k = -2\).
정답: (1) \(f(x) = (x-1)(3x-1) = 3x^2 - 4x + 1\)   (2) \(k = -2\). 서술 채점: ① 차수 균형 → \(f\) 이차 + 최고차 \(3\) ② \(f(1) = 0\) 도출 ③ \(c\) 결정 ④ \(k\) 계산.
서술10\(\infty\) 사칙 + 유계로 나누기
\(\lim f = \infty\), \(\lim\{4f - 2g\} = 1\). \(\lim\frac{3g - f}{5f}\)?
\(h(x) = 4f - 2g \to 1\) (유계). \(f \to \infty\)이므로 \(\frac{h}{f} \to 0\). \(g = \frac{4f - h}{2} = 2f - \frac{h}{2}\) → \(\frac{g}{f} = 2 - \frac{h}{2f} \to 2 - 0 = 2\). \(\frac{3g - f}{5f} = \frac{3 \cdot \frac{g}{f} - 1}{5} \to \frac{3 \cdot 2 - 1}{5} = \frac{5}{5} = 1\).
정답: \(1\). 서술 채점: ① \(h\)가 유계임을 명시 ② \(\frac{h}{f} \to 0\) 근거 ③ \(\frac{g}{f}\) 도출 ④ 최종 계산.
서술11유리식 모든 실수 연속 → \(a\) 범위
\(f = 3x - 8, g = x^2 + ax + 9\), \(\frac{f}{g}\)가 모든 실수 연속. \(a\) 범위?
모든 실수 연속 ⇔ \(g \neq 0\)인 모든 실수 ⇔ \(g\)의 판별식 \(\lt 0\). \(D = a^2 - 4 \cdot 9 = a^2 - 36 \lt 0\) ⇒ \(-6 \lt a \lt 6\).
정답: \(-6 \lt a \lt 6\). 서술 채점: ① "분모 ≠ 0" 명시 ② 판별식 \(\lt 0\) 근거.
서술12그래프 + 곱 연속 (calc_08_q12.png)
\(f\)가 \(x=1\)에서 점프 (좌극한 \(L_1=2\), 우극한 \(L_2=1\), \(f(1) = 2\)). \(g(x) = (x-a)f(x)\) 연속. \(a\)?
그래프 읽기: \(x \leq 1\)에서 \(y = x+1\), \(f(1) = 2 = \bullet\). \(x \gt 1\)에서 ○가 \((1,1)\) → 수평선 \(y=1\). 좌극한 \(L_1 = 2\), 우극한 \(L_2 = 1\), \(f(1) = 2\). \(g\) 연속 ⇒ 좌극한 = 우극한: \((1-a) \cdot 2 = (1-a) \cdot 1\) ⇒ \((1-a)(2 - 1) = 0\) ⇒ \(a = 1\). 확인: \(a=1\)이면 \(g(1) = 0 \cdot f(1) = 0\), 좌극한 \(= 0 \cdot 2 = 0\), 우극한 \(= 0 \cdot 1 = 0\). 일치 ✓.
정답: \(a = 1\). 서술 채점: ① \(f\) 불연속점 식별 ② 좌·우극한 식 명시 ③ \((x-a)\) 영점 = 불연속점 결론.
서술13IVT — 정수 \(a\) 개수
\(f\) 모든 실수 연속, \(f(-1) = a, f(1) = a - 4\). \(f(x) = x^3\)이 \((-1, 1)\)에 반드시 실근. 정수 \(a\) 개수?
① 보조함수 \(h(x) = f(x) - x^3\) 정의. \(f, x^3\) 연속 → \(h\) 연속. ② \(h(-1) = f(-1) - (-1)^3 = a + 1\), \(h(1) = f(1) - 1 = a - 4 - 1 = a - 5\). ③ "반드시 실근" 충분조건: \(h(-1) \cdot h(1) \lt 0\) ⇒ \((a + 1)(a - 5) \lt 0\) ⇒ \(-1 \lt a \lt 5\). ④ 정수 \(a \in \{0, 1, 2, 3, 4\}\) → 5개.
정답: 5개. 서술 채점: ① 보조함수 정의 ② 연속 명시 ③ 양 끝 부호 곱 \(\lt 0\) (등호 X) ④ 정수 개수 셈.
주의: 부호 곱이 \(\leq 0\)이면 끝점 영점 가능 → 열린구간 내부 보장 X. 엄격 부등호 \(\lt 0\)이 결정적.

G 성찰지 운영 — 3유형 우수 답안 예 (대단원 단위)

유형 1: 단원 종합 통찰 — 극한 → 연속 흐름 "단원 I 전체를 정리하며 극한과 연속이 결국 '극한값 = 함숫값' 한 줄로 묶인다는 걸 알게 됐다. 이 한 줄이 IVT에서는 '부호 변화 → 실근 존재'의 전제로, EVT에서는 '닫힘+연속 → 극값 존재'의 전제로 쓰인다. 단원이 따로따로가 아니라 정의 한 줄을 중심으로 부채꼴처럼 펼쳐져 있다는 게 보였다."
유형 2: 실수 분석 — 자주 틀린 유형 "단원 I에서 가장 자주 틀린 건 ① \(|x|\) 같은 좌·우 분기 빠뜨리기, ② \(x \to -\infty\)에서 \(\sqrt{x^2}\) 부호 헷갈리기 두 가지. '① 좌·우 분기 확인 → ② 절댓값·근호 부호 처리 → ③ 약분/유리화 → ④ 연속 조건 점검' 4단계 체크리스트를 시험지 여백에 적어두고 풀고 있다. 비슷한 실수가 거의 없어졌다."
유형 3: 미분으로의 다리 "미분계수 \(\lim_{h\to 0}\frac{f(a+h)-f(a)}{h}\)를 보면 \(h \to 0\)일 때 분자도 \(0\)으로 가는 전형 \(\frac{0}{0}\) 모양. 이번 단원의 인수분해·약분이 그대로 쓰일 것 같다. '미분가능 ⇒ 연속'은 연속이 더 약한 조건이라는 얘기라, 단원 I의 연속 정의가 단원 II의 기반으로 들어간다는 가설을 세웠다."
