C 생각 열기 운영 가이드

발문: 곡선 위 두 점 \(\mathrm{A}, \mathrm{B}\)에서 \(f(\mathrm{A}) = f(\mathrm{B})\)일 때 그 사이 어딘가에 기울기 \(0\)인 접선이 반드시 존재할까? 어디일까?

예상 반응 ①"정점에서 기울기가 \(0\)이니까 정점이 답이다."
피드백: 정답에 가깝다. "\(f'(c) = 0\)이라는 점이 적어도 하나 존재"가 롤의 정리임을 명시. 정점이 유일한 게 아니라 "적어도 하나" 강조.
예상 반응 ②"\(f(\mathrm{A}) = f(\mathrm{B})\) 조건이 없으면 어떻게 되지?"
피드백: 좋은 질문. 그 경우 두 점을 잇는 직선의 기울기와 같은 접선이 존재 — 그게 평균값 정리. 롤은 그 특수 경우라는 연결을 미리 예고.
예상 반응 ③"미분 안 되면 어떻게 해?"
피드백: 핵심 질문. \(f(x) = |x|\)에서 \(x = 0\) 미분 불가 → 롤 정리 무너짐을 보여 줌. 3조건 모두 필수임을 강조하며 개념 2(반례 점검)로 직접 연결.

차시 12 → 13 연결: 차시 12에서 "두 곡선의 공통 접선" 조건이 ① 점 공통 ② 기울기 일치였다. 평균값 정리도 결국 "두 점을 잇는 직선과 평행한 접선의 존재"라는 같은 발상의 일반화.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE THEOREM
\( \dfrac{f(b) - f(a)}{b - a} = f'(c), \quad c \in (a, b) \)    (롤: \(f(a) = f(b) \Rightarrow f'(c) = 0\))
📌 판서 시연 5단계:
  1. 좌표평면에 \(\mathrm{A}(a, f(a))\), \(\mathrm{B}(b, f(b))\)와 두 점을 잇는 직선 색깔로 표시.
  2. 두 점 사이 어떤 점 \(c\)에서의 접선을 그려 직선 AB와 평행함을 시각화.
  3. 3조건 (① \([a, b]\)에서 연속 ② \((a, b)\)에서 미분가능 ③ 롤이면 \(f(a) = f(b)\)) 칠판 박스로 강조.
  4. 반례: \(f(x) = |x|\), \([-1, 1]\) → \(x = 0\) 미분 불가, 접선이 \(x\)축 평행한 점 없음.
  5. 따름정리 증명: \(h(x) = f(x) - g(x)\), \(h'(x) = 0 \Rightarrow h\)는 상수 → 부정적분 \(+ C\) 미리 예고.
오개념 ①: "평균값 정리는 \(c\)가 어딘지 알려준다." → 존재성만 보장. 어디·몇 개는 직접 풀어야 함 (L3 7번 함정).
오개념 ②: "롤 정리에서 \(f(a) = f(b)\)만 보면 된다." → 연속·미분가능 두 조건도 필수. 한 가지라도 빠지면 결론 무너짐.
오개념 ③: "\(f'(x) = 0\)이 한 점이라도 있으면 \(f\)는 상수." → "구간 전체"에서 \(f'(x) = 0\)이어야 상수. 한 점만이면 \(x^3\)처럼 변곡 가능.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11롤의 정리의 \(c\)
\(f(x) = x^2 - 4x + 3\), \([1, 3]\)에서 롤의 정리의 \(c\).
조건 확인: \(f(1) = 1 - 4 + 3 = 0\), \(f(3) = 9 - 12 + 3 = 0\) ✓. 연속·미분가능 ✓. \(f'(x) = 2x - 4\), \(f'(c) = 0 \Rightarrow c = 2 \in (1, 3)\) ✓.
정답: \(c = 2\).
L12평균값 정리의 \(c\)
(1) \(f(x) = -2x^2 + 4x\), \([0, 4]\)   (2) \(f(x) = x^3\), \([-3, 3]\)
(1) \(f(0) = 0\), \(f(4) = -16\). 평균변화율 \(= -4\). \(f'(x) = -4x + 4\), \(-4c + 4 = -4 \Rightarrow c = 2\). (2) \(f(-3) = -27\), \(f(3) = 27\). 평균변화율 \(= 9\). \(f'(x) = 3x^2\), \(3c^2 = 9 \Rightarrow c = \pm\sqrt{3}\). 둘 다 \((-3, 3)\)에 속함.
정답: (1) \(c = 2\)   (2) \(c = \pm\sqrt{3}\)
L13롤 정리 적용 불가 판정
\(f(x) = |x|\), \([-1, 1]\). 롤의 정리가 적용되지 않는 이유.
연속 ✓, \(f(-1) = 1 = f(1)\) ✓. 그러나 \(x = 0\)에서 좌미분 \(-1\), 우미분 \(+1\) → 미분 불가. 결론: 3조건 중 "\((a, b)\)에서 미분가능" 조건이 깨짐 → 정리 적용 불가. 실제로 \(f'(x) = \pm 1\)이 \(0\)이 되는 점 없음.
정답: \(x = 0\)에서 미분가능 조건이 깨져서 적용 불가.
L24\(c\) 주어졌을 때 미지수 결정
\(f(x) = x^3 + x + 1\), \([-1, a]\), \(c = 1\)일 때 \(a\)는? (\(a \gt 1\))
\(\dfrac{f(a) - f(-1)}{a - (-1)} = f'(1) = 4\). \(f(-1) = -1\). \(a^3 + a + 1 + 1 = 4(a + 1) \Rightarrow a^3 + a + 2 = 4a + 4 \Rightarrow a^3 - 3a - 2 = 0\). 인수분해: \((a - 2)(a + 1)^2 = 0\). 검증: \((2)(9) = 18 = 8 - 6 - 2 + 18 \cdots\) 다항 전개 OK. \(a \gt 1 \Rightarrow a = 2\).
정답: \(a = 2\).
L25구간 단속 — 평균값 정리의 현실 응용
제한 80 km/h, 15 km 구간을 10분에 통과. 과속 순간 있었나?
평균속도 \(= \dfrac{15\,\mathrm{km}}{10/60\,\mathrm{h}} = 90\,\mathrm{km/h}\). 위치 \(f(t)\)는 연속·미분가능. 평균값 정리 \(\Rightarrow f'(c) = 90\)인 \(c\) 존재 → 어느 순간 순간속도 \(90\) km/h. 제한 80 km/h \(\lt 90\) → 과속 순간 반드시 존재.
정답: 과속 순간이 반드시 있었다. 슬로건: "평균이 \(M\)이면 어느 순간도 \(M\)."
L26부등식 증명에 평균값 정리
\(n \geq 2\)에 대해 \(2(n - 1) \lt n^2 - (n - 1)^2 \lt 2n\)을 평균값 정리로 증명.
\(f(x) = x^2\)에 \([n - 1, n]\)에서 평균값 정리. \(f\)는 연속·미분가능, \(f'(x) = 2x\). \(\dfrac{n^2 - (n - 1)^2}{n - (n - 1)} = 2c\), \(c \in (n - 1, n)\). 즉 \(n^2 - (n - 1)^2 = 2c\). \(n - 1 \lt c \lt n \Rightarrow 2(n - 1) \lt 2c \lt 2n\). 대입: \(2(n - 1) \lt n^2 - (n - 1)^2 \lt 2n\). \(\blacksquare\)
정답: 평균값 정리에 의한 부등식 증명 완료. 패턴: "함수의 차분 = 미분계수 × 구간 길이", \(c\)의 위치 정보로 양변 묶기.
L37🚨 오답률 1위 — 카드 1·2·3 참/거짓
미분가능한 \(f\)가 \((0, 0), (2, 2), (4, 0)\) 지남. 카드 1: \(x\)축 평행 접선 존재 / 카드 2: 기울기 \(-1\) 접선 존재 / 카드 3: \((2, 2)\)에서 접선이 \(x\)축 평행.
카드 1 (참): \(f(0) = f(4) = 0\), \([0, 4]\) 롤 정리 → \(f'(c) = 0\)인 \(c \in (0, 4)\) 존재 ✓. 카드 2 (참): \([2, 4]\)에서 \(f(2) = 2, f(4) = 0\). 평균변화율 \(= \dfrac{0 - 2}{4 - 2} = -1\). 평균값 정리 → \(f'(c) = -1\)인 \(c \in (2, 4)\) 존재 ✓. 카드 3 (거짓): 카드 1은 "\(f'(c) = 0\)인 \(c\) 존재"만 보장 — \(c\)가 정확히 \(2\)인지는 모름. 반례: 세 점을 지나며 \(f'(2) \neq 0\)인 함수 쉽게 구성 (적당히 비대칭 4차).
정답: 카드 1·2 참, 카드 3 거짓. 핵심: 평균값·롤 정리는 존재성만 보장 — 위치·개수·특정점은 결론에 포함되지 않음.
발표 유도: "카드 3이 맞다고 생각한 학생 손!" → 손든 학생에게 "왜 정확히 \(x = 2\)에서일까?"를 직접 묻는다. "존재성 ≠ 특정성" 한 줄로 깨우치게 한다.
FC8형성평가 — \(f' = g' \Rightarrow f = g + k\) 증명
\([a, b]\)에서 \(f'(x) = g'(x)\)이면 \(f(x) = g(x) + k\) (상수)임을 평균값 정리로 설명.
\(h(x) = f(x) - g(x)\)로 두면 \(h\)는 \([a, b]\)에서 연속·미분가능. \(h'(x) = f'(x) - g'(x) = 0\) (모든 \(x \in (a, b)\)). 차시 내 따름정리 ("\(h'(x) = 0 \Rightarrow h\)는 상수")에 의해 \(h(x) = k\). 따라서 \(f(x) - g(x) = k \Rightarrow f(x) = g(x) + k\). \(\blacksquare\)
정답: \(h = f - g\) 트릭으로 차시 내 따름정리 적용. 의미: 부정적분 \(\int f\,dx = F(x) + C\)의 \(C\)는 평균값 정리에서 보장되는 상수 — 차시 22 부정적분의 정당화 근거.
CH1D1⭐ 수능특강 도함수의 활용 1 유제 3 [26009-0077]
(가) 모든 \(x \gt 0\): \(f(x) - f(0) \geq 2px\). (나) \(x \gt 0\)에서 \(f'(x)\)의 최솟값 \(4\). \(p\)의 최댓값?
STEP 1: 임의의 \(x \gt 0\)에 대해 \([0, x]\)에서 평균값 정리 → \(f(x) - f(0) = x f'(c)\), \(c \in (0, x)\). STEP 2: (가)와 결합. \(x f'(c) \geq 2px\), \(x \gt 0\)이므로 \(f'(c) \geq 2p\). 임의의 \(x \gt 0\)에 대응되는 \(c \in (0, x)\)에서 성립 → \(x \gt 0\)에서 \(f'\)의 최솟값 \(\geq 2p\). STEP 3: (나)에서 \(f'\) 최솟값 \(= 4 \geq 2p \Rightarrow p \leq 2\). STEP 4: \(p\) 최댓값 \(= 2\) (예: \(f(x) = 4x + C\)에서 등호 성립).
정답: \(p = 2\). 핵심: 전 구간 부등식을 평균값 정리로 \(f'(c)\) 조건으로 변환 → \(f'\)의 최솟값과 직결.
CH2D2⭐ 2005학년도 9월 모평 가형 28번 [4점]
다항함수 \(f\), \(f'(x) \geq 0\), \(f(0) = \tfrac{1}{5}\), \(f(1) = 1\). 보기 ㄱ·ㄴ·ㄷ 중 옳은 것.
ㄱ (참): \([0, 1]\) 평균값 정리 → \(f'(c) = \dfrac{1 - 1/5}{1} = \tfrac{4}{5}\)인 \(c \in (0, 1)\) 존재. ㄴ (참, 적분 활용 — 2학기 진도): 공식 \(\int_a^b f + \int_{f(a)}^{f(b)} f^{-1} = bf(b) - af(a)\)에 \(a = 0, b = 1\) 대입 → \(= 1 \cdot 1 - 0 = 1\). 차시 26~27 학습 후 재방문 권장. ㄷ (참): \(g = f \circ f\), \(g(0) = f(\tfrac{1}{5})\), \(g(1) = f(1) = 1\). 평균값 정리 → \(g'(c) = 1 - f(\tfrac{1}{5})\)인 \(c \in (0, 1)\) 존재. 그래프에서 \(0 \lt f(\tfrac{1}{5}) \lt 1\). \(g'(x) = f'(f(x))f'(x)\)는 연속이고, \(g(0) = f(\tfrac{1}{5})\), \(g(1) = 1\) 사이에서 평균값 정리·사잇값 결합으로 \(g'(c) = 1\)인 점 존재. (정확한 평가원 의도: \([0, 1]\)에서 평균변화율 \(= \dfrac{1 - f(\tfrac{1}{5})}{1}\)를 \(g'\)의 한 값으로 보고, 다른 적당한 구간에서 \(g'\) 가 \(1\)을 갖는 점을 보장.)
정답: ⑤ ㄱ, ㄴ, ㄷ. 핵심: 평균값 정리가 (i) 직접 적용 (ii) 적분과 결합 (iii) 연쇄법칙과 결합의 3중 통합. ㄴ은 차시 26 이후 재방문.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (존재성 정리의 본질) "롤·평균값 정리가 '\(c\)가 어디인지' 알려주는 줄 알았는데, 사실은 '그런 \(c\)가 존재한다'는 사실만 보장한다는 걸 알게 됐다. 카드 3 함정에 속을 뻔했지만, 결론은 '존재'까지일 뿐 위치는 직접 풀어야 한다는 점을 굵게 적었다."
유형 2: 문제해결력 (구간 단속 사례 정착) "구간 단속 예시가 강력했다. 15 km를 10분에 지났으면 평균 90 km/h. 평균값 정리에 의해 어느 순간 정확히 90 km/h였던 적이 있다는 결론. 마라톤 평균 페이스에도 그대로 쓰여서 '평균이 곧 어느 순간 실제로 일어난 값'을 슬로건으로 정리했다."
유형 3: 발전 가능성 (\(f' = 0 \Rightarrow\) 상수 → 부정적분 \(+C\)) "따름정리 \(f' = g' \Rightarrow f = g + k\)가 차시 22 부정적분의 \(+ C\)를 정당화한다는 걸 알게 됐다. \(C\)가 그냥 약속이 아니라 평균값 정리에서 보장되는 거라는 가설을 세웠다. 단원 III으로 이어지는 다리가 보였다."
