C 생각 열기 운영 가이드

발문: 이차방정식 \(x^2 - 2x - 3 = 0\)의 실근은 \(y = x^2 - 2x - 3\)과 \(x\)축의 교점. 삼차·사차로 확장하면? 매개변수 방정식 \(2x^3 - 6x = a\)의 실근 개수는 \(a\)에 따라 어떻게 달라질까?

예상 반응 ①"방정식 풀려면 인수분해해야 한다."
피드백: 정통적이지만 한계 있음 — \(x^3 - 3x + 1\)은 인수분해 불가. "실근 개수만 알아도 충분한 문제"가 많음을 강조 → 미적분의 새 도구 소개.
예상 반응 ②"\(a\)에 숫자를 차례로 대입해 보자."
피드백: 시행착오는 비효율. "매개변수 분리"라는 우아한 도구 예고 — \(a\)를 한쪽으로 빼면 (고정 그래프) + (움직이는 수평선)이라는 시각화 가능.
예상 반응 ③"극값 부호로 결정될 것 같다."
피드백: 정확한 직관! 차시 16 형성평가에서 \(0 \lt a \lt 4\)를 도출한 사고가 본 차시의 표준 4케이스 분류로 일반화됨을 예고.

차시 16·17 연결: 차시 16의 그래프 개형 + 차시 17의 최솟값 사고가 본 차시에서 방정식·부등식 해결 도구로 통합. 단원 II의 도함수 활용 시리즈 클라이맥스.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE 3 PRINCIPLES
\( \#\{\text{실근}\} = \#\{\text{그래프} \cap x\text{축}\},\quad f(x) = a \Leftrightarrow y = f(x) \cap y = a,\quad \forall x:\ f \geq 0 \Leftrightarrow \min f \geq 0 \)
📌 판서 시연 — 삼차함수 실근 개수 4케이스 표:
  1. 케이스 ① \(Mm \lt 0\): 극값 부호 반대 (예: \(M = 5, m = -3\)) → \(x\)축이 그래프를 3번 가로지름 → 실근 3개.
  2. 케이스 ② \(Mm = 0\): 극값 중 하나가 \(0\) → 그 점에서 접함(중근) + 다른 한 점 → 실근 2개.
  3. 케이스 ③ \(Mm \gt 0\): 극값 부호 같음 (둘 다 양 or 둘 다 음) → 그래프가 \(x\)축을 한 번만 가로지름 → 실근 1개.
  4. 케이스 ④ 극값 없음: 단조함수 → \(x\)축과 정확히 한 점 교차 → 실근 1개.
오개념 ①: "삼차방정식은 항상 인수분해되어야 풀 수 있다." → 1순위 함정. 그래프 추론이 더 강력한 경우 다수.
오개념 ②: "매개변수 분리는 항상 가능하다." → 일부 식은 분리가 어려울 수 있음(예: \(ax^3 + b x = 1\)에서 \(a, b\) 두 매개변수). 분리 가능한 표준 패턴 \(f(x) = a\)를 우선 익힌다.
오개념 ③: "강부등식 \(f \gt 0\) ⇔ \(\min f \gt 0\)" 단순 적용. → 등호 성립 가능성이 있는 경우(예: 어떤 \(x_0\)에서 \(f(x_0) = 0\)) 강부등식은 위배. 등호 검증 필수.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11방정식의 실근 개수
(1) \(x^3 - 6x^2 + 12x + 1 = 0\)   (2) \(3x^4 - 4x^3 - 12x^2 + 5 = 0\)
(1) \(f'(x) = 3(x - 2)^2 \geq 0\) → 부호 변화 없음, 극값 없음, 단조증가. 케이스 ④ → 실근 1개. (2) \(f'(x) = 12x(x - 2)(x + 1)\). 극솟값 \(f(-1) = 0\), 극댓값 \(f(0) = 5\), 극솟값 \(f(2) = -27\). \(f(-1) = 0\)이므로 \(x = -1\)에서 접함(중근 1점) + 극솟값 \(-27 \lt 0\)이므로 그 우측에서 두 점 더 교차 → 서로 다른 실근 3개.
정답: (1) 1개   (2) 3개
판서 포인트: (2)에서 학생들이 가장 자주 놓치는 부분 — "극솟값 = 0"인 점은 접점이지만 서로 다른 실근 카운트에서는 1점으로 센다. 중복도와 서로 다른 실근 수의 차이를 명확히.
L12매개변수 — 세 실근 조건
\(2x^3 - 9x^2 + 12x - 4 + a = 0\)이 세 실근 갖는 \(a\)의 범위.
\(g'(x) = 6(x-1)(x-2)\). 극댓값 \(g(1) = 1 + a\), 극솟값 \(g(2) = a\). 세 실근 ⇔ \((1 + a) \cdot a \lt 0 \Rightarrow -1 \lt a \lt 0\).
정답: \(-1 \lt a \lt 0\)
L13부등식 증명 — 최솟값 = 0 케이스
(1) \(x \gt 0\)일 때 \(2x^3 - 6x^2 + 8 \geq 0\)   (2) 모든 \(x\)에 대해 \(3x^4 - 4x^3 + 1 \geq 0\)
(1) \(f'(x) = 6x(x - 2)\). \(x \gt 0\)에서 \(x = 2\)가 극소. \(f(2) = 16 - 24 + 8 = 0\) → \(\min f = 0\), 등호 \(x = 2\)에서만 성립. (2) \(h'(x) = 12x^2(x - 1)\). \(x = 1\)에서 극소(부호 \(- \to +\)). \(h(1) = 3 - 4 + 1 = 0\). \(\min h = 0\) ✓.
정답: (1) 등호 \(x = 2\)에서 성립   (2) 등호 \(x = 1\)에서 성립. (강부등식 \(\gt\)는 등호 성립점 제외하면 성립.)
L24두 곡선 교점 — 매개변수 분리
\(y = x^3 + x^2 - 5x + 1\), \(y = -2x^2 - 5x + a\)의 교점 개수를 \(a\)에 따라.
교점 ⇔ \(x^3 + 3x^2 + 1 = a\). \(g(x) = x^3 + 3x^2 + 1\), \(g'(x) = 3x(x + 2)\). 극댓값 \(g(-2) = -8 + 12 + 1 = 5\), 극솟값 \(g(0) = 1\). 수평선 \(y = a\)와 \(y = g(x)\)의 교점: 극값 사이면 3개, 일치면 2개, 외부면 1개.
정답: \(a \lt 1\) 또는 \(a \gt 5\) → 1개. \(a = 1\) 또는 \(a = 5\) → 2개. \(1 \lt a \lt 5\) → 3개.
판서 포인트: "두 곡선의 교점" 문제는 항상 차함수 \(h = f_1 - f_2\)로 환원 → 매개변수 분리. 본질적으로 한 가지 전략으로 통합됨을 강조.
L25부등식 매개변수 — 최솟값 \(k\)
모든 실수 \(x\)에 대해 \(-x^4 + 4x^2 - 5 \lt k\) 성립하는 \(k\)의 최솟값.
조건 ⇔ \(k \gt \max_x (-x^4 + 4x^2 - 5)\). \(f(x) = -x^4 + 4x^2 - 5\), \(f'(x) = -4x(x^2 - 2)\). 임계점 \(x = 0, \pm\sqrt{2}\). 부호 \((+,-,+,-)\). 극댓값 \(f(\pm\sqrt{2}) = -4 + 8 - 5 = -1\), 극솟값 \(f(0) = -5\). \(\max f = -1\). \(k \gt -1\) → 최솟값 \(k = -1\) (교과서 표기 관행: 강부등식의 경계 \(-1\)을 "최솟값"으로 답).
정답: \(k = -1\)
L26두 함수 비교 + 매개변수
\(x \gt 0\)에서 \(f(x) = 3x^3 + 8x^2 - 18x + k \geq g(x) = x^3 - x^2 + 6x + 1\) 성립하는 \(k\).
차함수 \(h(x) = f - g = 2x^3 + 9x^2 - 24x + (k - 1)\). 조건 ⇔ \(\min_{x \gt 0} h \geq 0\). \(h'(x) = 6(x + 4)(x - 1)\). \(x \gt 0\)에서 \(x = 1\) 극소. \(h(1) = 2 + 9 - 24 + (k - 1) = k - 14\). \(k - 14 \geq 0 \Rightarrow k \geq 14\).
정답: \(k \geq 14\)
L37🚨 오답률 1위 — 절댓값 + 매개변수 (5개 실근)
\(f(x) = 2x^3 - 3x^2 - 12x + 4\)에 대해 \(|f(x)| = k\)가 서로 다른 5개의 실근을 갖는 \(k\).
STEP 1: \(f'(x) = 6(x-2)(x+1)\). 극댓값 \(f(-1) = -2 - 3 + 12 + 4 = 11\), 극솟값 \(f(2) = 16 - 12 - 24 + 4 = -16\). STEP 2: \(|f(x)|\) 그래프 — \(f \lt 0\) 영역(\(x\)축 사이)이 위로 뒤집힘. 원래 극솟값 \(-16\)이 새 극댓값 \(16\)으로 등장. STEP 3: \(|f|\)의 그래프 봉우리 패턴: 좌측 극대 \(11\), \(x\)축 접점, 큰 봉우리 \(16\), \(x\)축 접점, 우측 단조증가. STEP 4: 수평선 \(y = k\)가 정확히 5점에서 교차 ⇔ \(k = 11\)(좌측 극대값에서 접점 1 + 큰 봉우리 좌우 2 + 우측 단조 영역 2 = 총 5점).
정답: \(k = 11\) (정밀 그래프 분석 결과). 핵심: 절댓값 후의 "새 극댓값"이 원래 극솟값의 절댓값임을 인지.
발표 유도: 학생이 \(|f|\) 그래프를 칠판에 직접 그리게 한다 — \(f \lt 0\) 부분을 빨간색으로 표시한 뒤 그 부분을 \(x\)축 기준 위로 반전. "절댓값 = 양수만"이라는 기계적 사고를 시각적으로 깨뜨리는 결정적 순간.
FC8형성평가 — 두 곡선 세 점 교차
\(y = x^3 + 4x^2\), \(y = -2x^2 - 9x + k\)가 세 점에서 만나는 \(k\).
교점 ⇔ \(x^3 + 6x^2 + 9x = k\). \(g(x) = x^3 + 6x^2 + 9x\), \(g'(x) = 3(x + 1)(x + 3)\). 극댓값 \(g(-3) = -27 + 54 - 27 = 0\), 극솟값 \(g(-1) = -1 + 6 - 9 = -4\). 세 실근 ⇔ \(-4 \lt k \lt 0\). 경계 \(k = -4\) 또는 \(k = 0\)이면 접함(중근) → 실근 2개로 제외.
정답: \(-4 \lt k \lt 0\)
CH1D1⭐ 수능특강 수학Ⅱ 도함수 활용 2 유제 5 — 세제곱합 인수분해 + 강부등식
모든 실수 \(x\)에 대해 \(\dfrac{1}{4}x^4 + (k - 3)^3 x + 12 \gt 0\)이 성립하는 정수 \(k\)의 합. ① 6 ② 7 ③ 8 ④ 9 ⑤ 10
STEP 1: \(f'(x) = x^3 + (k-3)^3\). 세제곱합 \(= (x + k - 3)\{x^2 - (k-3)x + (k-3)^2\}\). 이차인수 \(= \bigl(x - \tfrac{k-3}{2}\bigr)^2 + \tfrac{3}{4}(k-3)^2 \geq 0\) 항상 → \(f' = 0\)은 \(x = 3 - k\) 유일. STEP 2: \(f(3 - k) = \tfrac{1}{4}(3 - k)^4 + (k - 3)^3 (3 - k) + 12 = \tfrac{1}{4}(k-3)^4 - (k-3)^4 + 12 = -\tfrac{3}{4}(k-3)^4 + 12\). STEP 3: \(-\tfrac{3}{4}(k-3)^4 + 12 \gt 0 \Rightarrow (k-3)^4 \lt 16 \Rightarrow (k-3)^2 \lt 4 \Rightarrow 1 \lt k \lt 5\). STEP 4: 정수 \(k = 2, 3, 4\). 합 \(= 9\).
정답: ④ \(9\). 핵심: \(f' = x^3 + c^3\)는 세제곱합 인수분해로 임계점 유일 확인 → 그 지점에서 \(\min f \gt 0\).
CH2D2⭐ 2015학년도 6월 모의평가 나형 17번 [4점] — 근의 부호 분포
\(f(x) = 3x^3 - x^2 - 3x\), \(g(x) = x^3 - 4x^2 + 9x + a\)의 교점 방정식이 두 양의 실근 + 한 음의 실근인 정수 \(a\). ① 6 ② 7 ③ 8 ④ 9 ⑤ 10
STEP 1: \(f = g \Rightarrow 2x^3 + 3x^2 - 12x = a\). \(h(x) = 2x^3 + 3x^2 - 12x\), \(h'(x) = 6(x + 2)(x - 1)\). 극댓값 \(h(-2) = 20\), 극솟값 \(h(1) = -7\), \(h(0) = 0\). STEP 2: 세 실근 ⇔ \(-7 \lt a \lt 20\). 추가 조건: 음근 1 + 양근 2 ⇔ 수평선 \(y = a\)가 \(h(0) = 0\) 아래에 위치(즉 \(a \lt 0\)). STEP 3: 두 조건 결합 → \(-7 \lt a \lt 0\). 정수 \(a \in \{-6, -5, -4, -3, -2, -1\}\) → 6개.
정답: ① \(6\). 핵심: 매개변수 분리 + \(h(0)\) 값을 기준으로 수평선의 부호 분포 통제.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (4케이스 분류) "삼차방정식의 실근 개수가 극값 부호 곱 \(Mm\) 하나로 결정된다는 게 충격적이었다. \(Mm \lt 0\) → 3개, \(= 0\) → 2개(중근), \(\gt 0\) → 1개, 극값 없음 → 1개. 4가지 케이스로 정리하고 나니 차시 16의 개형과 차시 18의 실근 분석이 결국 '극값 부호 → \(x\)축 교차 횟수' 한 원리로 묶였다."
유형 2: 문제해결력 (매개변수 분리) "매개변수 문제에서 처음엔 \(a\)에 숫자를 대입하며 시행착오를 했는데, '\(a\)를 한쪽으로 분리 → 고정 함수 + 움직이는 수평선 교점'이라는 시각화 전략을 익히고 나서 시간이 확 줄었다. 두 곡선의 교점 문제도 차함수 \(h = f - g\)로 환원하면 같은 전략에 흡수된다는 게 보였다."
유형 3: 발전 가능성 (부등식과 최솟값의 동치) "\(f(x) \geq 0\) 항상 성립 ⇔ \(\min f \geq 0\)이라는 동치 변환이 차시 17의 최솟값 사고를 부등식 증명으로 그대로 이어준다. 차시 23(정적분의 활용)에서 운동 거리 = \(\int |v|\,dt\)로 절댓값이 다시 등장할 텐데, Level 3에서 \(|f|\) 그래프를 직접 그려본 경험이 도움이 될 것 같다."
