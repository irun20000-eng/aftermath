C 생각 열기 운영 가이드

발문: 극값과 최댓값·최솟값은 같은 말인가? 닫힌구간 \([-1, 2]\)에서 \(f(x) = 2x^3 - 3x^2 + 2\)의 극댓값은 \(f(0) = 2\)이지만 최댓값은 \(f(2) = 6\)이다. 왜?

예상 반응 ①"극값이 최값이다 — 항상 같다."
피드백: 반례 즉시 제시(\(f(0) = 2 \lt f(2) = 6\)). "극값은 동네 1등, 최값은 전체 1등" 비유로 차이를 각인.
예상 반응 ②"극값이 없는 함수는 최값도 없다."
피드백: 반례 — 단조함수 \(f(x) = x\)는 \([0, 1]\)에서 극값은 없지만 최댓값 \(1\), 최솟값 \(0\) 명확. 구간 끝점이 최값이 될 수 있음을 강조.
예상 반응 ③"열린구간이면 최값이 없다."
피드백: 일반적으로 옳음 — 그러나 임계점이 내부에 하나만 있으면 최값이 존재(상자 부피 예시). "① 패턴 = 실생활 최적화 95%" 메시지 전달.

차시 06 연결: 최대·최소 정리("연속 + 닫힘 → 최값 존재")의 결론이 본 차시의 출발점. "존재성은 차시 06, 계산법은 차시 17"로 역할 분담.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE FORMULA
\( \max = \max\{\text{극값들},\ f(a),\ f(b)\},\quad \min = \min\{\text{극값들},\ f(a),\ f(b)\} \)
📌 판서 시연 4단계 (예제 \([-1, 2]\)에서 \(f = 2x^3 - 3x^2 + 2\)):
  1. \(f'(x) = 6x(x-1)\). 임계점 \(x = 0, 1\) — 두 점 모두 구간 내부 확인.
  2. 증감표 작성. \(f(0) = 2\) 극대, \(f(1) = 1\) 극소.
  3. 후보 4개 모음: \(\{f(-1), f(0), f(1), f(2)\} = \{-3, 2, 1, 6\}\) — 내부 임계점 2개 + 끝점 2개.
  4. 비교. \(\max = 6\) (\(x = 2\), 끝점), \(\min = -3\) (\(x = -1\), 끝점). 극값이 최값이 아님.
오개념 ①: "극댓값 = 최댓값" 자동 동일시. → 1순위 함정. 끝점도 반드시 비교.
오개념 ②: 실생활 문제에서 정의역 누락. → \(x\)가 길이면 \(x \gt 0\), 종이 변이 한정이면 \(0 \lt x \lt 6\) 등 물리적 제약을 항상 명시.
오개념 ③: 임계점이 구간 밖인데 그 점의 함숫값을 후보에 포함. → 임계점이 구간 외부면 후보에서 제외, 구간 끝점만 비교.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11닫힌구간 최댓값·최솟값 직접 계산
(1) \(f(x) = x^3 - 3x^2 - 9x + 5,\ [-2, 4]\)   (2) \(f(x) = x^4 - 8x^3 + 16x^2 + 1,\ [-1, 2]\)
(1) \(f'(x) = 3(x-3)(x+1)\). 내부 임계점 \(x = -1, 3\). 후보 \(f(-2) = 3, f(-1) = 10, f(3) = -22, f(4) = -15\). (2) \(f'(x) = 4x(x-2)(x-4)\). 내부 임계점 \(x = 0\) (\(x = 2\)는 끝점, \(x = 4\)는 외부). 후보 \(f(-1) = 26, f(0) = 1, f(2) = 17\).
정답: (1) \(\max = 10\,(x = -1)\), \(\min = -22\,(x = 3)\)   (2) \(\max = 26\,(x = -1)\), \(\min = 1\,(x = 0)\)
판서 포인트: (2)에서 \(x = 4\)가 구간 \([-1, 2]\) 외부임을 학생이 직접 발견하게 한다. 외부 임계점 함숫값은 후보 제외.
L12매개변수 + 최솟값 조건
\([0, 3]\)에서 \(f(x) = 2x^3 - 3x^2 + a\)의 최솟값이 \(5\)일 때 \(a\).
\(f'(x) = 6x(x-1)\). 내부 임계점 \(x = 1\). 후보 \(f(0) = a, f(1) = a - 1, f(3) = a + 27\). \(\min = a - 1 = 5 \Rightarrow a = 6\).
정답: \(a = 6\)
L13실생활 — 상자 부피 최댓값
한 변 \(18\) cm 정사각형 종이, 귀퉁이 \(x\) cm 잘라 뚜껑 없는 직육면체. 부피 최댓값.
\(V(x) = x(18 - 2x)^2\), \(0 \lt x \lt 9\). \(V'(x) = (18 - 2x)(18 - 6x)\). \(V' = 0 \Rightarrow x = 3\) (정의역 내). 부호 \((+, -)\) → \(x = 3\)에서 극대 = 최대. \(V(3) = 3 \cdot 12^2 = 432\) cm³.
정답: \(432\) cm³ (잘라낸 변 \(x = 3\) cm)
L24도형 넓이 최대화 — 포물선 위 직사각형
\(y = -x^2 + 9\) 위에 두 꼭짓점, \(x\)축 위에 두 꼭짓점인 직사각형 ABCD의 넓이 최댓값. (A는 제1사분면.)
A\(= (x, -x^2 + 9)\), \(0 \lt x \lt 3\). 대칭으로 가로 \(2x\), 세로 \(-x^2 + 9\). \(S(x) = 2x(-x^2 + 9) = -2x^3 + 18x\). \(S'(x) = -6(x^2 - 3)\). \(S' = 0 \Rightarrow x = \sqrt{3}\). 부호 \((+, -)\) → \(x = \sqrt{3}\)에서 극대 = 최대. \(S(\sqrt{3}) = 2\sqrt{3} \cdot 6 = 12\sqrt{3}\).
정답: \(\max S = 12\sqrt{3}\)
판서 포인트: "정의역 \(0 \lt x \lt 3\)"의 \(3\)이 어디서 왔나? — 포물선 \(y = -x^2 + 9\)의 \(x\)축 교점이 \(x = \pm 3\). 물리적 제약은 그래프에서 직접 읽어야 함.
L25원뿔 내접 원기둥 — 부피 최대
밑면 반지름 \(5\) cm, 높이 \(10\) cm 원뿔 내접 원기둥 중 부피 최대인 원기둥의 밑면 반지름.
원기둥 밑면 반지름 \(r\), \(0 \lt r \lt 5\). 닮음꼴 비례 \((5 - r) : h = 5 : 10\) → \(h = 2(5 - r)\). \(V(r) = \pi r^2 \cdot 2(5 - r) = 2\pi(5r^2 - r^3)\). \(V'(r) = 2\pi r(10 - 3r)\). \(V' = 0 \Rightarrow r = \dfrac{10}{3}\) (정의역 내). 부호 \((+, -)\) → \(r = \dfrac{10}{3}\)에서 극대 = 최대.
정답: \(r = \dfrac{10}{3}\) cm, 최대 부피 \(\dfrac{1000\pi}{27}\) cm³
판서 포인트: 닮음꼴 그림 단면도(이등변삼각형 + 내접 직사각형)를 칠판에 그려 비례 \(h = 2(5-r)\)을 학생이 직접 유도하게 한다. 도형 + 미분 결합의 핵심 단계.
L26닫힌구간 최대 - 최소 = 상수 — 매개변수와 무관
\([-2, 1]\)에서 \(f(x) = x^3 + 3x^2 + a\)의 최대·최소 차가 \(4\)인 \(a\).
\(f'(x) = 3x(x + 2)\). 구간 \([-2, 1]\) 내부 임계점은 \(x = 0\) (\(x = -2\)는 끝점). 후보 \(f(-2) = a + 4, f(0) = a, f(1) = a + 4\). \(\max = a + 4\) (\(x = -2\) 또는 \(x = 1\)), \(\min = a\) (\(x = 0\)). 차 \(= 4\) → 모든 \(a\)에 대해 항상 성립.
정답: 모든 실수 \(a\)에 대해 차 = 4 (특정 값 결정 불가). 추가 조건(예: 최댓값 \(=\) 특정 수)이 필요. 학습 포인트: 후보를 모두 모으면 차가 \(a\)와 무관함이 보인다.
L37🚨 오답률 1위 — 단조 vs 극값 존재 분류
\(f(x) = x^3 + 3x^2 + ax + 1\)이 \([-1, 2]\)에서 \(\max = 15, \min = 0\). \(a\) 및 임계점.
STEP 1: \(f'(x) = 3x^2 + 6x + a\). 판별식 \(D/4 = 9 - 3a\). 단조 \(\Leftrightarrow a \geq 3\), 극값 존재 \(\Leftrightarrow a \lt 3\). STEP 2 (단조): \(a \geq 3\)이면 \(f' \geq 0\) → 단조증가. \(\min = f(-1) = 3 - a\), \(\max = f(2) = 21 + 2a\). 조건 \(3 - a = 0 \Rightarrow a = 3\), 검증 \(21 + 6 = 27 \neq 15\) → 모순, 해 없음. STEP 3 (극값 존재): \(a \lt 3\)이면 임계점 두 개. 후보 4개를 \(a\)로 표현 + \(\max = 15, \min = 0\) 조건으로 \(a\) 결정. 일반적으로 정수해 안 나옴 — 학습 목표는 "절차 + 경우 분류" 자체.
정답: 단조 케이스 모순으로 해 없음 → \(a \lt 3\) 케이스에서 후보 비교(절차 학습). 핵심: "극값 = 최값" 자동 동일시 금지 + 단조 vs 극값 분기 + 임계점 구간 내·외 점검.
발표 유도: 학생들이 빠지는 3가지 오답 패턴 — ① 극댓값을 무조건 최댓값으로 ② 단조함수면 "극값 없으니 답 없음"으로 잘못 결론 ③ 외부 임계점을 후보에 포함. 이 셋을 칠판에 적고 ✕로 표시.
FC8형성평가 — 포물선 내접 직사각형 둘레 최대
\(y = -x^2 + 4\) (\(-2 \leq x \leq 2\))와 \(x\)축 사이 영역에 내접하는 직사각형(축 평행) 중 둘레 최대.
점 \((x, -x^2 + 4)\), \(0 \lt x \lt 2\). 대칭으로 가로 \(2x\), 세로 \(-x^2 + 4\). \(L(x) = 4x + 2(-x^2 + 4) = -2x^2 + 4x + 8\). \(L'(x) = -4(x - 1)\). \(L' = 0 \Rightarrow x = 1\). 부호 \((+, -)\) → 극대 = 최대. \(L(1) = -2 + 4 + 8 = 10\).
정답: 최대 둘레 = \(10\) (가로 \(2\), 세로 \(3\)).
CH1D1⭐ 수능특강 수학Ⅱ 도함수 활용 2 유제 1 — 극값 조건 + 닫힌구간 최댓값
\(f(x) = 2x^2 + ax\), \(g(x) = f(x) + f(x - a)\)가 \(x = 1\)에서 극값. \([-2, 2]\)에서 \(g(x)\)의 최댓값.
STEP 1: \(g(x) = (2x^2 + ax) + [2(x-a)^2 + a(x-a)] = 4x^2 - 2ax + a^2\). STEP 2: \(g'(x) = 8x - 2a\). \(g'(1) = 0 \Rightarrow a = 4\). \(g(x) = 4x^2 - 8x + 16 = 4(x-1)^2 + 12\). STEP 3: 축이 \(x = 1\), 아래로 볼록 이차함수. \([-2, 2]\)에서 축으로부터 먼 끝 \(x = -2\)에서 최대. \(g(-2) = 16 + 16 + 16 = 48\).
정답: \(48\). 핵심: 극값 조건으로 \(a\) 결정 → 이차함수로 환원 → 닫힌구간 양 끝 중 축에서 먼 점.
CH2D2⭐ 2011년 4월 학력평가 가형 28번 [4점]
\(12 \times 6\) cm 직사각형 종이 네 모퉁이 \(x\) cm 잘라 직육면체 상자. 부피 최대 \(M\)에 대해 \(\dfrac{\sqrt{3}}{3}M\).
STEP 1: \(V(x) = x(12 - 2x)(6 - 2x) = 4x^3 - 36x^2 + 72x\). 정의역 \(0 \lt x \lt 3\). STEP 2: \(V'(x) = 12(x^2 - 6x + 6)\). 영점 \(x = 3 \pm \sqrt{3}\). 정의역 내는 \(x = 3 - \sqrt{3}\). STEP 3: 부호 \((+, -)\) → 극대 = 최대. \(M = V(3 - \sqrt{3}) = (3 - \sqrt{3})(6 + 2\sqrt{3})(2\sqrt{3})\). STEP 4: \((3 - \sqrt{3})(6 + 2\sqrt{3}) = 18 + 6\sqrt{3} - 6\sqrt{3} - 6 = 12\). \(M = 24\sqrt{3}\). \(\dfrac{\sqrt{3}}{3} \cdot 24\sqrt{3} = \dfrac{24 \cdot 3}{3} = 24\).
정답: \(24\). 핵심: 표준 절차 + 무리수 임계점 \(3 - \sqrt{3}\)에서의 \(\sqrt{3}\) 배수 정리.

피드백: "직사각형 둘레 = 2(가로 + 세로). 가로는 두 \(x\)좌표 사이 거리 \(2x\), 세로는 곡선 위 점의 \(y\)좌표. 정의역은 곡선과 \(x\)축이 만나는 \(x = \pm 2\)에서 정해진다."

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (극값 vs 최값) "극값이 무조건 최값인 줄 알았는데, \(f = 2x^3 - 3x^2 + 2\)를 \([-1, 2]\)에서 보면 극댓값 \(2\)보다 끝점 값 \(6\)이 더 컸다. '극값은 동네 일등, 최값은 전체 일등'으로 외우고, 후보를 '내부 임계점 + 끝점' 4가지로 모두 모으는 절차를 체크리스트로 만들었다. 차시 06 최대·최소 정리가 여기서 그대로 활용된다는 누적이 느껴졌다."
유형 2: 문제해결력 (실생활 모델링 4단계) "상자 부피·직사각형 넓이·원기둥 부피 문제에서 처음엔 식만 세우고 미분했는데, 정의역을 빼먹어 음수 길이가 나온 적이 있었다. '① 변수 → ② 제약식 → ③ 목적 함수 → ④ 미분 + 정의역 명시' 4단계로 정리하고 나니 모델링 실수가 사라졌다. 정의역은 그냥 형식이 아니라 수학과 현실을 잇는 다리라는 게 느껴졌다."
유형 3: 발전 가능성 (최솟값 → 부등식 증명) "오늘 익힌 최솟값 분석이 다음 차시 18에서 '\(f(x) \geq 0\)이 항상 성립 \(\Leftrightarrow \min f \geq 0\)'로 그대로 이어질 것 같다. 즉 부등식 증명도 결국 최솟값 구하는 문제로 환원되는 셈. 미분이 그래프 그리는 도구를 넘어 부등식 증명·매개변수 결정 도구로 진화하는 게 단원의 큰 그림으로 보였다."
