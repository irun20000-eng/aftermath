C 단원 회고 발문 (Warm-up 대체)

발문: "3차시 동안 가장 자주 막혔던 한 가지 도구는 무엇인가? 정적분에서 '상수인지 함수인지' 헷갈렸던 순간은?"

예상 반응 ①"적분상수 C. 붙여야 하는 자리를 자꾸 빠뜨렸다."
피드백: 신호 = 부정적분 결과가 함수일 때 반드시 \(+\,C\). 특정 점을 지날 때 초기조건으로 C를 결정. 정적분에서는 C가 소거되니 구분만 명확히.
예상 반응 ②"\(\int_0^1 tf(t)\,dt\)가 x의 함수인지 상수인지 헷갈렸다."
피드백: 신호 = 양 끝점이 상수이면 정적분 값은 하나의 수. 그래서 k로 치환해도 된다. \(\int_a^x f(t)\,dt\) 형태만 x의 함수.
예상 반응 ③"절댓값 정적분에서 구간 분할을 빠뜨렸다."
피드백: 신호 = 피적분에 \(|g(x)|\) 형태가 보이면 \(g(x) = 0\) 되는 점을 먼저 찾아 구간 분리. 각 구간에서 부호 결정 후 적분.

III-2 예고: 다음 단원 넓이는 \(\int_a^b |f(x) - g(x)|\,dx\). 즉 오늘까지 배운 구간 분할·절댓값 정적분 기술이 넓이 계산의 핵심 도구로 그대로 연결.

D 개념 지도 노트 — 단원 종합 도구 트리

DECISION TREE — 문제 보고 0.5초 안에 도구 선택
끝점 없음? 부정적분 \(\int f\,dx = F(x)+C\)  |  끝점 있음? 기본정리 \(\int_a^b f\,dx = F(b)-F(a)\)  |  \(\int_a^x f(t)\,dt\)? 양변 미분 → \(f(x)\), 이후 \(x=a\) 대입  |  \(\int_0^1 tf(t)\,dt\)? 상수 k 치환 → 자기참조 방정식  |  \(|f(x)|\)? 부호 변경점 찾아 구간 분할
📌 판서 시연 — 단원 합성 5단계 (서술형 대비):
  1. 문제 형태 분류: 부정적분 / 정적분(끝점 상수) / 정적분(끝점 변수) / 적분 항이 식 안에 포함.
  2. 도구 선택: Decision Tree에서 1개 추출. "k 치환" 또는 "양변 미분" 첫 줄에 박스로 표시.
  3. 식 변형: k 치환 시 → 정의식에 대입 → 정적분 계산 → 방정식 k 결정. 양변 미분 시 → \(g' = f\) 도출 → 초기조건으로 상수 결정.
  4. 계산 결과 검증: \(f(a)\) 또는 \(f(0)g(1)\)을 구하는 최종 단계 명시.
  5. 채점 포인트 4개 모두 명시 후 답 박스.
가장 자주 틀리는 3가지: ① 정적분 항 \(\int_0^1 tf(t)\,dt\)를 x의 함수로 착각 ② \(\int_a^x f(t)\,dt\) 미분 시 결과를 F(x)가 아닌 f(x)로 혼동 ③ 절댓값 정적분에서 부호 변경점 미탐색으로 구간 분할 누락.

E 문항별 해설 카드 (14문항 종합)

기초1부정적분 — 항별 적분 + 곱셈 전개
(1) \(\int(x^2-2x+3)\,dx\)   (2) \(\int x(x+1)^2\,dx - \int x(x^2+1)\,dx\)
(1) 항별: \(\dfrac{1}{3}x^3 - x^2 + 3x + C\). (2) 피적분 차: \(x(x+1)^2 - x(x^2+1) = x[(x+1)^2-(x^2+1)] = x \cdot 2x = 2x^2\). 적분 \(\dfrac{2}{3}x^3 + C\).
정답: (1) \(\dfrac{1}{3}x^3-x^2+3x+C\)   (2) \(\dfrac{2}{3}x^3+C\)
기초2정적분 기본 계산 (1)(2)
(1) \(\int_0^2(3x^2+5)\,dx\)   (2) \(\int_{-2}^{1}(x+2)^2\,dx - \int_{-2}^{1}(2x+1)\,dx\)
(1) \([x^3+5x]_0^2 = 8+10 = 18\). (2) 같은 구간 → 피적분 차: \((x+2)^2-(2x+1) = x^2+2x+3\). \([\tfrac{1}{3}x^3+x^2+3x]_{-2}^{1} = \tfrac{13}{3}-(-\tfrac{14}{3}) = 9\).
정답: (1) \(18\)   (2) \(9\)
기초3구간 분할 합치기
\(\int_{-1}^{1}(2x^3-6x)\,dx + \int_1^3(2x^3-6x)\,dx\)
같은 피적분 + 구간 이어짐 → \(\int_{-1}^{3}(2x^3-6x)\,dx\). \([\tfrac{1}{2}x^4-3x^2]_{-1}^{3} = (\tfrac{81}{2}-27)-(\tfrac{1}{2}-3) = \tfrac{27}{2}-(-\tfrac{5}{2}) = \dfrac{32}{2} = 16\).
정답: \(16\)
판서 포인트: 피적분이 같고 끝점이 맞닿으면 한 구간으로 합치기. \(\int_a^b + \int_b^c = \int_a^c\).
기본4도함수 일반항 + 두 조건
\(f'(x) = 1+2x+3x^2+\cdots+nx^{n-1}\), \(f(0)=5\), \(f(1)=11\). 자연수 \(n\) 구하기. (\(n \geq 2\))
항별 적분: \(f(x) = x+x^2+x^3+\cdots+x^n+C\). \(f(0) = C = 5\). \(f(1) = 1+1+\cdots+1\,(n\text{개})+5 = n+5 = 11 \Rightarrow n = 6\).
정답: \(n = 6\)
기본5접선 기울기 + 통과점
점 \((-2, 1)\) 통과, 접선 기울기 \(6x^2+12x+5\). \(f(-1)\) 구하기.
\(f(x) = 2x^3+6x^2+5x+C\). \(f(-2) = -16+24-10+C = 1 \Rightarrow C=3\). \(f(-1) = -2+6-5+3 = 2\).
정답: \(f(-1) = 2\)
기본6도함수 + 극값 조건
\(f'(x) = ax^2-ax\), 극댓값 6·극솟값 5 (\(a \gt 0\)). \(f(x)\) 구하기.
\(f'(x) = ax(x-1)\). \(a \gt 0\) → \(x=0\) 극대, \(x=1\) 극소. \(f(x) = \tfrac{a}{3}x^3-\tfrac{a}{2}x^2+C\). \(f(0)=C=6\). \(f(1)=-\tfrac{a}{6}+6=5 \Rightarrow a=6\).
정답: \(f(x) = 2x^3-3x^2+6\)
기본7부정적분 항등식 + 양변 미분
\(F(x) = xf(x)-3x^4+2x^3\) (\(F\)는 \(f\)의 부정적분), \(f(1) = 6\). \(f(2)\) 구하기.
\(F'(x) = f(x)\). 양변 미분: \(f(x) = f(x)+xf'(x)-12x^3+6x^2\) → \(xf'(x) = 12x^3-6x^2\) → \(f'(x) = 12x^2-6x\). 적분: \(f(x) = 4x^3-3x^2+C\). \(f(1) = 4-3+C = 6 \Rightarrow C=5\). \(f(2) = 32-12+5 = 25\).
정답: \(f(2) = 25\)
판서 포인트: \(F' = f\)를 명시한 뒤 항등식 양변 미분. \(xf'(x)\) 항 도출이 핵심.
기본8정적분 항등식 → \(f(a)\) 결정
\(\int_a^x f(t)\,dt = 6x^2+x-1\) (모든 실수 \(x\), \(a \gt 0\)). \(f(a)\) 구하기.
① 양변 \(x\)로 미분: \(f(x) = 12x+1\). ② \(x = a\) 대입: \(0 = 6a^2+a-1 = (3a-1)(2a+1)\). \(a \gt 0 \Rightarrow a = \tfrac{1}{3}\). \(f(a) = f(\tfrac{1}{3}) = 12 \cdot \tfrac{1}{3}+1 = 5\).
정답: \(f(a) = 5\)
2단계 루틴: ① 양변 미분 → \(f(x)\) 도출. ② \(x=a\) 대입 → 좌변 0 → a 방정식. 두 단계를 서술에 모두 명시.
기본9정적분 이차식 최솟값
\(\int_0^1(6a^2x^2-16ax+5)\,dx\) 최솟값 → \(a\)와 최솟값 구하기.
적분: \([2a^2x^3-8ax^2+5x]_0^1 = 2a^2-8a+5\). 완전제곱: \(2(a-2)^2-3\). \(a=2\)에서 최솟값 \(-3\).
정답: \(a = 2\), 최솟값 \(-3\)
기본10선형결합 연립으로 정적분
\(\int_{-3}^{5}(f-g)\,dx=-3\), \(\int_{-3}^{5}(2f+3g)\,dx=4\). \(\int_{-3}^{5}(f+g)\,dx\) 구하기.
\(p = \int f\), \(q = \int g\). 연립: \(p-q=-3\), \(2p+3q=4\). 풀면 \(p=-1, q=2\). \(p+q = -1+2 = 1\).
정답: \(1\)
기본11구간 분할 합치기 — 3개 조건
\(\int_0^1 f=5\), \(\int_0^4 f=9\), \(\int_{-1}^{1}f=3\). \(\int_{-1}^{4}f\) 구하기.
\(\int_{-1}^{4} = \int_{-1}^{1}+\int_{1}^{4}\). \(\int_1^4 = \int_0^4-\int_0^1 = 9-5 = 4\). 합: \(3+4 = 7\).
정답: \(7\)
판서 포인트: 공통 구간을 겹쳐 빼는 방법. \(\int_0^4 - \int_0^1 = \int_1^4\) 단계를 화살표로 명시.
기본12절댓값 정적분 — 구간 분할
\(\int_{-1}^{2}(x^2-2|x|)\,dx\)
\(x=0\) 부호 변경. \(\int_{-1}^{0}(x^2+2x)\,dx + \int_0^{2}(x^2-2x)\,dx\). \([\tfrac{1}{3}x^3+x^2]_{-1}^{0} + [\tfrac{1}{3}x^3-x^2]_0^{2} = (0-(-\tfrac{1}{3}+1))+((\tfrac{8}{3}-4)-0) = -\tfrac{2}{3}+(-\tfrac{4}{3}) = -2\).
정답: \(-2\)
판서 포인트: \(|x|\)의 부호 변경점 \(x=0\) 먼저 찾기. \(x \lt 0\)이면 \(|x|=-x\), \(x \geq 0\)이면 \(|x|=x\). 구간 분리 후 각각 적분.
도전13⭐ 서술형 — 정적분 항 k 치환
\(f(x) = 4x^2+3x+\int_0^1 tf(t)\,dt\). \(f(1)\) 구하기 (풀이 포함).
① \(\int_0^1 tf(t)\,dt\)는 \(x\)와 무관 → 상수 \(k\)로 치환. \(f(x) = 4x^2+3x+k\). ② 정의식 대입: \(k = \int_0^1 t(4t^2+3t+k)\,dt = \int_0^1(4t^3+3t^2+kt)\,dt\). ③ 계산: \([t^4+t^3+\tfrac{k}{2}t^2]_0^1 = 1+1+\tfrac{k}{2} = 2+\tfrac{k}{2}\). ④ 방정식: \(k = 2+\tfrac{k}{2} \Rightarrow \tfrac{k}{2} = 2 \Rightarrow k = 4\). \(f(x) = 4x^2+3x+4\). ⑤ \(f(1) = 4+3+4 = 11\).
정답: \(f(1) = 11\). 서술 채점: ① k 치환 ② 정의식에 f(x) 대입 ③ 정적분 계산 ④ k 결정 ⑤ f(1) 계산. 5단계 모두 명시 시 만점.
도전14⭐ 서술형 — 양변 미분으로 \(g' = f\) + 둘째 식 인수분해
일차함수 \(f(x)\), 다항함수 \(g(x)\)가 \(xf(x)=g(x)+\int_0^x tf'(t)\,dt\)와 \(f(x)g(x)=2x^3+9x^2+9x\)를 만족. \(f(0)g(1)\) 구하기.
① 첫 식 양변 \(x\)로 미분: \(f(x)+xf'(x) = g'(x)+xf'(x)\) → \(g'(x) = f(x)\). ② 일차함수 \(f(x) = ax+b\) → \(g(x) = \int f\,dx = \tfrac{a}{2}x^2+bx+C\). ③ \(x=0\) 대입: \(0 = g(0)+0 \Rightarrow g(0) = C = 0\). \(g(x) = \tfrac{a}{2}x^2+bx\). ④ \(f(x)g(x) = (ax+b)\cdot x(\tfrac{a}{2}x+b) = x(ax+b)(\tfrac{a}{2}x+b) = 2x^3+9x^2+9x = x(2x+3)(x+3)\). 대응: \((ax+b) \leftrightarrow (2x+3)\), \((\tfrac{a}{2}x+b) \leftrightarrow (x+3)\) → \(a=2, b=3\). 검증: \(f \cdot g = (2x+3)(x^2+3x) = 2x^3+9x^2+9x\) ✓. ⑤ \(f(0) = 3\), \(g(1) = 1+3 = 4\). \(f(0)g(1) = 12\).
정답: \(f(0)g(1) = 12\). 서술 채점: ① \(g' = f\) 도출 ② 일차 가정 + 상수결정 ③ \(g(0)=0\) ④ 인수분해 a·b 결정 ⑤ 최종 계산. 5단계.

G 중단원 성찰 — 3유형 우수 답안 예

유형 1: 단원 종합 통찰 "3차시 동안 부정적분·정적분 정의·기본정리를 별개로 배웠는데, 마무리에서 보니 '부정적분 \(\int f\,dx = F(x)+C\)를 이용해 \(F\)를 구한 뒤 끝점에 대입하는 것이 기본정리 \(F(b)-F(a)\)'라는 한 흐름이었다. 그 흐름을 알고 나니 절댓값·구간 분할도 같은 기본정리를 구간마다 쓰는 것일 뿐이었다."
유형 2: 자주 틀린 유형 분석 "\(\int_0^1 tf(t)\,dt = k\)에서 k가 상수라는 걸 순간적으로 잊고 x에 의존한다고 생각해 미분하려 했다. 이제는 '정적분의 양 끝점이 모두 상수이면 결과는 하나의 수 → k로 치환 가능' 확인 카드를 첫 단계로 둔다. 반면 \(\int_a^x\) 형태는 x의 함수이므로 바로 미분."
유형 3: 다음 단원으로의 확장 "정적분이 부호 있는 넓이라는 정의를 살리면, 다음 단원 도형 넓이는 \(\int_a^b |f(x)-g(x)|\,dx\)로 자연스럽게 확장될 것 같다. 절댓값 분할 기술이 그대로 연결되는 게 보인다. 또 속도를 적분하면 이동거리가 나오는데, 이것도 부호 있는 면적의 절댓값 합이라는 구조가 동일하다."
