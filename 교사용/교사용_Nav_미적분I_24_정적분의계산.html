24 III-1 부정적분과 정적분 정적분의 계산 — 미적분의 기본정리와 절댓값 정적분
한 줄 핵심: \(\displaystyle\int_a^b f(x)\,dx = F(b) - F(a)\) — 부정적분의 차로 정적분을 계산하는 미적분의 기본정리가 오늘 단원의 정점.
성취기준
[12미적-03-03] 미적분의 기본정리를 이해하고 정적분을 계산한다.
선수학습
22차시 부정적분 공식, 23차시 정의·약속·적분미분 관계. 절댓값의 부호 분기(단원 I).
학습 목표
① 기본정리 \([F(x)]_a^b\) 표기법 ② 선형성·구간 분할 ③ 절댓값 정적분 4단계 ④ 연립으로 정적분 결정.

B 50분 수업 타임라인

도입 5'
교사: "23차시 정의로 \(\int_0^1 x^2\,dx\)를 어떻게 구하겠는가?" — 도형으로는 불가능함을 확인 후 부정적분 \(\dfrac{1}{3}x^3\)으로 연결. \(F(1) - F(0) = \dfrac{1}{3}\) 판서.
학생: 학습지 00 Thinking Space에 \(\dfrac{1}{3}x^3 + C\)에서 \(C\)가 소거됨을 직접 확인.
개념 15'
교사: 기본정리 → \([F(x)]_a^b\) 표기법 → 선형성(부정적분과 동일) → 구간 분할 → 절댓값 포함 정적분 순서. 영점 찾기 → 구간 나누기 → 부호 살리기 → 합산의 4단계를 색깔로 강조.
학생: 개념 구성 4블록에서 각 reveal-box 직접 클릭. 블록 4의 예시 \(\int_{-2}^{1}|x+1|\,dx\)를 단계별로 따라 계산.
L1·L2 15'
교사: L1(1~3번) 짝활동 5분 — 1번 각 4항 순서대로 \([F]_a^b\) 표기 연습. 2번은 전개 후 합치기, 3번은 구간 분할 직관. L2(4~6번) 10분 — 4번 연립, 5번 \(a\)에 관한 이차함수 완전제곱, 6번 연속 조건 활용.
학생: 1번 계산 후 검산(미분으로 역방향). 6번은 \(f(1^-) = f(1)\) 연속 조건 명시.
L3·형성 10'
교사: 7번(절댓값 정적분 함정) — "통째로 적분하려다 막힌다"는 패턴을 먼저 말하게 한 뒤 함정 경고 클릭. \(x=0\) 분할을 색깔로 판서. 8번은 구간 합치기 연습.
학생: 7번 \(-\tfrac{2}{3} + (-\tfrac{4}{3}) = -2\) 검산. 8번 \(\int_{-1}^4 = \int_{-1}^1 + \int_1^4\) 분할 경로 명시.
정리 5'
교사: "적분상수 \(C\)가 왜 정적분에서 사라지는가?" 1명 발표. 도전 1·2 과제 및 25차시(중단원 마무리) 예고.
학생: 성찰지 3유형 중 하나 이상. 도전 1·2 과제.

C 생각 열기 운영 가이드

발문: "22차시 부정적분 \(F(x) = \dfrac{1}{3}x^3\)와 23차시 적분미분 관계 \(S'(x) = f(x)\)를 합치면 어떤 한 줄 공식이 나올 것 같은가?"

예상 반응 ①"\(\int_0^1 x^2\,dx = \dfrac{1}{3}\)일 것 같다."
피드백: 정확! \(F(1) - F(0) = \dfrac{1}{3} - 0 = \dfrac{1}{3}\). 오늘 배울 기본정리가 이것. 곡선 아래 면적을 도형 없이 단 한 줄로 계산.
예상 반응 ②"\(+C\)가 결과에 남을 것 같다."
피드백: 실제로는 \(F(b)+C - (F(a)+C)\)에서 \(C\)가 소거됨. 오늘 개념 블록 1에서 직접 확인하자.
예상 반응 ③"아직 잘 모르겠다."
피드백: 22차시 부정적분 공식 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)에 \(x=1, x=0\)을 대입하여 차를 구하면 된다. 오늘 그 과정을 체계화한다.

23차시 연결: \(S(x) = \int_a^x f(t)\,dt\)가 \(f\)의 부정적분이므로 \(S(b) = F(b) - F(a)\). 기본정리는 23차시 적분미분 관계의 직접적 귀결.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

FUNDAMENTAL THEOREM OF CALCULUS — 미적분의 기본정리
\( \displaystyle \int_a^b f(x)\,dx = \big[F(x)\big]_a^b = F(b) - F(a) \quad (F'(x) = f(x)) \)
📌 판서 시연 5단계:
  1. \(\big[F(x)\big]_a^b = F(b) - F(a)\) 표기법과 "위끝 먼저, 아래끝 나중"을 확실히 각인.
  2. \(\int_1^3 x\,dx = \big[\tfrac{1}{2}x^2\big]_1^3 = \tfrac{9}{2} - \tfrac{1}{2} = 4\). 적분상수 \(C\)를 쓰지 않아도 되는 이유 설명.
  3. 선형성: \(\int_a^b kf = k\int_a^b f\), \(\int_a^b(f+g) = \int_a^b f + \int_a^b g\). 부정적분과 동일.
  4. 구간 분할: \(\int_a^c + \int_c^b = \int_a^b\) — \(F(c)-F(a) + F(b)-F(c) = F(b)-F(a)\) 대수적 증명.
  5. 절댓값: \(\int_{-2}^{1}|x+1|\,dx\) → 영점 \(x=-1\) → \([-2,-1]\)과 \([-1,1]\) 분할 → 부호별 적분 → 합산.
오개념 ①: "\(\int_a^b f\,dx = F(a) - F(b)\)" — 위끝 \(b\)가 먼저, 아래끝 \(a\)가 나중. 순서 반전 실수 잦음.
오개념 ②: "\(\int_{-1}^{2}|x|\,dx = \int_{-1}^{2}x\,dx\)" — 절댓값 \(\neq\) 그냥 \(x\). \(x \lt 0\) 구간에서 \(|x| = -x\).
오개념 ③: "구간 분할 시 \(c\)가 \([a,b]\) 안에 있어야 한다" — \(c\)가 바깥에 있어도 자동 성립.

E 문항별 해설 카드 (L1 1~3 · L2 4~6 · L3 7 · 형성 8 · 도전 1·2)

L11기본정리로 다항함수 정적분
(1) \(\int_{-2}^{2}3x^2\,dx\) (2) \(\int_0^4 x^3\,dx\) (3) \(\int_1^3 4x\,dx\) (4) \(\int_{-2}^{1}5x^4\,dx\)
(1) \([x^3]_{-2}^{2} = 8 - (-8) = 16\). (2) \([\tfrac{1}{4}x^4]_0^4 = 64 - 0 = 64\). (3) \([2x^2]_1^3 = 18 - 2 = 16\). (4) \([x^5]_{-2}^{1} = 1 - (-32) = 33\).
정답: (1) \(16\) (2) \(64\) (3) \(16\) (4) \(33\)
L12선형성과 구간 분할
(1) \(\int_0^2(8x^3-x+5)\,dx\)   (2) \(\int_{-1}^3(x+1)^3\,dx - \int_{-1}^3(x-1)^3\,dx\)
(1) \([2x^4 - \tfrac{1}{2}x^2 + 5x]_0^2 = 32 - 2 + 10 = 40\). (2) 피적분 차: \((x+1)^3 - (x-1)^3 = 6x^2 + 2\). \([2x^3 + 2x]_{-1}^{3} = 60 - (-4) = 64\).
정답: (1) \(40\)   (2) \(64\)
L13구간 분할로 합치기
(1) \(\int_0^5(4x^3+x)\,dx + \int_5^2(4x^3+x)\,dx\)   (2) \(\int_{-1}^4(x+2)^2\,dx - \int_2^4(x+2)^2\,dx\)
(1) \(\int_0^5 + \int_5^2 = \int_0^2\). \([x^4+\tfrac{1}{2}x^2]_0^2 = 16 + 2 = 18\). (2) \(\int_{-1}^4 - \int_2^4 = \int_{-1}^2\). \([\tfrac{1}{3}(x+2)^3]_{-1}^{2} = \tfrac{64}{3} - \tfrac{1}{3} = 21\).
정답: (1) \(18\)   (2) \(21\)
L24선형결합 연립으로 정적분 결정
\(\int_{-3}^{5}\{f-g\}\,dx = -3\), \(\int_{-3}^{5}\{2f+3g\}\,dx = 4\). \(\int_{-3}^{5}\{f+g\}\,dx\) 구하기.
\(p = \int_{-3}^{5}f\), \(q = \int_{-3}^{5}g\)로 치환. ① \(p - q = -3\), ② \(2p + 3q = 4\). 연립: \(①\times2: 2p - 2q = -6\). ②-①×2: \(5q = 10 \Rightarrow q = 2\), \(p = -1\). 답: \(p + q = 1\).
정답: \(1\)
L25정적분의 최솟값 (\(a\)에 관한 이차함수)
\(\int_0^1(6a^2x^2-16ax+5)\,dx\) 최솟값과 그때의 \(a\).
적분: \([2a^2x^3-8ax^2+5x]_0^1 = 2a^2-8a+5\). \(I(a) = 2(a-2)^2 - 3\). 최솟값: \(a = 2\), \(I = -3\).
정답: \(a = 2\)일 때 최솟값 \(-3\)
L26구간별 도함수 → 연속 조건으로 \(f(-1)\)
\(f'(x) = x - 3\,\,(x \geq 1)\), \(-2x^2\,\,(x \lt 1)\). \(f(2) = -2\)일 때 \(f(-1)\).
\(x \geq 1\): \(f(x) = \tfrac{1}{2}x^2 - 3x + C_1\). \(f(2) = 2-6+C_1=-2 \Rightarrow C_1=2\). \(f(1) = \tfrac{1}{2}-3+2 = -\tfrac{3}{2}\). \(x \lt 1\): \(f(x) = -\tfrac{2}{3}x^3 + C_2\). 연속: \(-\tfrac{2}{3}+C_2 = -\tfrac{3}{2} \Rightarrow C_2 = -\tfrac{5}{6}\). \(f(-1) = \tfrac{2}{3} - \tfrac{5}{6} = -\tfrac{1}{6}\).
정답: \(f(-1) = -\dfrac{1}{6}\)
L37🚨 절댓값 정적분 — 구간 분할 함정
\(\int_{-1}^{2}(x^2-2|x|)\,dx\) 구하기.
STEP 1: \(x = 0\)에서 부호 변경. \(\int_{-1}^0 + \int_0^2\)로 분할. STEP 2: \(x \in [-1,0]\): \(|x| = -x\) → 피적분 \(= x^2 + 2x\). \(x \in [0,2]\): \(|x| = x\) → 피적분 \(= x^2 - 2x\). STEP 3: \([\tfrac{1}{3}x^3+x^2]_{-1}^0 = 0 - (-\tfrac{1}{3}+1) = -\tfrac{2}{3}\). STEP 4: \([\tfrac{1}{3}x^3-x^2]_0^2 = \tfrac{8}{3}-4 = -\tfrac{4}{3}\). 합: \(-\tfrac{2}{3} - \tfrac{4}{3} = -2\).
정답: \(-2\). 핵심: 절댓값 → 부호 바뀌는 점 찾기 → 구간 분할 → 부호별 적분 → 합산.
발표 포인트: "왜 \(x = 0\)에서 나누는가?"를 학생에게 묻는다. "\(|x|\)가 \(x \lt 0\)이면 \(-x\), \(x \geq 0\)이면 \(x\)이므로 부호 경계에서 나눠야 한다."
FC8형성평가 — 구간 합치기로 정적분 계산
\(\int_0^1 f = 5\), \(\int_0^4 f = 9\), \(\int_{-1}^{1}f = 3\). \(\int_{-1}^{4}f\,dx\) 구하기.
① \(\int_1^4 = \int_0^4 - \int_0^1 = 9 - 5 = 4\). ② \(\int_{-1}^4 = \int_{-1}^1 + \int_1^4 = 3 + 4 = 7\).
정답: \(7\). 핵심: 구간을 자유롭게 끊어 잇는 발상이 핵심.
CH1D1⭐ 수능특강 수학Ⅱ 부정적분과 정적분 유제 5
\(\int_0^3|x^2-2x|\,dx + \int_0^2(x^2-2x)\,dx\)의 값은? ① \(\dfrac{1}{3}\) ② \(\dfrac{2}{3}\) ③ \(1\) ④ \(\dfrac{4}{3}\) ⑤ \(\dfrac{5}{3}\)
STEP 1: 영점 \(x(x-2)=0 \Rightarrow x=0,2\). \([0,2]\)에서 \(x^2-2x \leq 0\), \([2,3]\)에서 \(\geq 0\). STEP 2: \(\int_0^3|x^2-2x|\,dx = -\int_0^2(x^2-2x)\,dx + \int_2^3(x^2-2x)\,dx\). STEP 3: 원식 \(= -\int_0^2(x^2-2x)\,dx + \int_2^3(x^2-2x)\,dx + \int_0^2(x^2-2x)\,dx = \int_2^3(x^2-2x)\,dx\). STEP 4: \([\tfrac{x^3}{3}-x^2]_2^3 = (9-9) - (\tfrac{8}{3}-4) = \dfrac{4}{3}\).
정답: ④ \(\dfrac{4}{3}\). 핵심: \([0,2]\) 구간의 두 정적분이 부호 반대로 상쇄 → \(\int_2^3\)만 남음.
CH2D2⭐ 2024학년도 수능 공통 8번 [3점]
삼차함수 \(f(x)\)가 \(xf(x) - f(x) = 3x^4 - 3x\). \(\int_{-2}^{2}f(x)\,dx\)는? ① \(12\) ② \(16\) ③ \(20\) ④ \(24\) ⑤ \(28\)
STEP 1: \((x-1)f(x) = 3x(x-1)(x^2+x+1)\) → \(f(x) = 3x(x^2+x+1) = 3x^3+3x^2+3x\). STEP 2: \(3x^3, 3x\)는 기함수 → \(\int_{-2}^{2} = 0\). \(3x^2\)은 우함수. STEP 3: \(\int_{-2}^{2}3x^2\,dx = 2\int_0^2 3x^2\,dx = 2[x^3]_0^2 = 2 \cdot 8 = 16\).
정답: ② 16. 핵심: 공통 인수 \((x-1)\) 약분 → 기함수·우함수 분리 → 대칭 구간 정적분.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 "\(S(x) = \int_a^x f(t)\,dt\)가 \(f\)의 부정적분이므로 \(S(a) = 0\)에서 \(C = -F(a)\). 따라서 \(S(b) = F(b) - F(a)\). 기본정리가 23차시의 귀결이라는 게 보였다. \(+C\)가 차로 인해 자동 소거되는 것도 이제 당연하게 느껴진다."
유형 2: 결정적 실수와 극복 "\(\int_0^3|x^2-1|\,dx\)를 통째로 넣으려다 막혔다. '① 영점 \(x=\pm1\) → ② 구간 분할 \([0,1], [1,3]\) → ③ 음수 구간에 \(-\) 붙이기 → ④ 합산' 4단계 카드를 만들었다. 절댓값이 나오면 반드시 1단계부터 시작."
유형 3: 발전 가능성 "기본정리로 정적분이 \(F(b)-F(a)\) 한 줄로 정리됐다. 다음 단원에서 곡선 아래 넓이·두 곡선 사이·속도-거리에 이게 쓰일 텐데, 미분 단원의 속도가 적분에서 거리로 연결되는 큰 그림이 미리 보인다."
