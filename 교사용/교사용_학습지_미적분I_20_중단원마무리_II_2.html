01. 실전 훈련 (Textbook Mastery)

💡 비상 미적분 I 교과서 중단원 마무리 14문항을 그대로 학습지화. 기초 1~3 · 기본 4~12 · 도전 13·14 (서술형 킬러)

[기초-접선의 방정식]
1. 곡선 \(y = x^3 - 3x^2 + 4\) 위의 점 \((1, 2)\)에서의 접선의 방정식을 구하시오. (교과서 P.96 중단원 기초 01번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f(x) = x^3 - 3x^2 + 4\)에서 \(f'(x) = 3x^2 - 6x\). 점 \((1, 2)\)에서의 기울기 \(f'(1) = 3 - 6 = -3\). 접선의 방정식 \(y - 2 = -3(x - 1)\) → \(y = -3x + 5\).
[기초-속도와 가속도]
2. 수직선 위를 움직이는 점 \(P\)의 시각 \(t\)에서 위치 \(x\)가 \( x = 2t^3 - 5t \)일 때, \(t = 1\)에서 점 \(P\)의 속도와 가속도를 구하시오. (교과서 P.96 중단원 기초 02번)
🎯 풀이 전략 보기
💡 전략 힌트: 속도 \(v = \dfrac{dx}{dt} = 6t^2 - 5\). 가속도 \(a = \dfrac{dv}{dt} = 12t\). \(t = 1\) 대입 → 속도 \(= 6 - 5 = 1\), 가속도 \(= 12\).
[기초-접선과 도형의 넓이]
3. 점 \((0, 2)\)에서 곡선 \(y = -x^3 + x + 4\)에 그은 접선과 \(x\)축, \(y\)축으로 둘러싸인 부분의 넓이를 구하시오. (교과서 P.97 중단원 기본 03번)
🎯 풀이 전략 보기
💡 전략 힌트: 접점을 \((t, -t^3+t+4)\)라 하자. \(f'(x) = -3x^2 + 1\). 접선이 \((0, 2)\)를 지나므로 \(-t^3 + t + 4 = (-3t^2 + 1)(0 - t) + 2\) → \(2t^3 + 2 = 0\) → \(t = -1\). 접점 \((-1, 2)\), 기울기 \(-2\) → 접선 \(y = -2x + 2\). \(x\)절편 \(1\), \(y\)절편 \(2\) → 넓이 \(\dfrac{1}{2} \cdot 1 \cdot 2 = 1\).
[기본-공통접선 (두 곡선)]
4. 두 곡선 \( f(x) = x^3 + ax + b \), \( g(x) = cx^2 + 1 \) 이 점 \((1, 2)\)에서 만나고 이 점에서 공통인 접선을 가질 때, 실수 \(a, b, c\)의 값을 구하시오. (교과서 P.97 중단원 기본 04번)
🎯 풀이 전략 보기
💡 전략 힌트: 조건 ①: 두 곡선이 \((1, 2)\)에서 만남 → \(f(1) = 1 + a + b = 2\), \(g(1) = c + 1 = 2\). 조건 ②: 공통접선 → \(f'(1) = g'(1)\). \(f'(x) = 3x^2 + a\), \(g'(x) = 2cx\) → \(3 + a = 2c\). \(c = 1\) → \(3 + a = 2\) → \(a = -1\). \(1 + (-1) + b = 2\) → \(b = 2\). 따라서 \(a = -1, b = 2, c = 1\).
[기본-평균값 정리 적용]
5. 함수 \(f(x) = x^3 + x + 1\)에 대하여 닫힌구간 \([-1, a]\)에서 평균값 정리를 만족시키는 실수 \(c\)의 값이 \(1\)일 때, 실수 \(a\)의 값을 구하시오. (단, \(a \gt 1\)) (교과서 P.97 중단원 기본 05번)
🎯 풀이 전략 보기
💡 전략 힌트: 평균값 정리: \(\dfrac{f(a) - f(-1)}{a - (-1)} = f'(c)\). \(f(-1) = -1\), \(f(a) = a^3 + a + 1\), \(f'(x) = 3x^2 + 1\), \(f'(1) = 4\). \(\dfrac{a^3 + a + 1 - (-1)}{a + 1} = \dfrac{a^3 + a + 2}{a + 1} = 4\). 분자 인수분해: \(a^3 + a + 2 = (a + 1)(a^2 - a + 2)\) → \(a^2 - a + 2 = 4\) → \(a^2 - a - 2 = 0\) → \((a-2)(a+1) = 0\) → \(a \gt 1\)이므로 \(a = 2\).
[기본-구간 전체에서 증가 조건]
6. 삼차함수 \(f(x) = x^3 + ax^2 + 3x + 1\)이 구간 \((-\infty, \infty)\)에서 증가하도록 하는 실수 \(a\)의 값의 범위를 구하시오. (교과서 P.97 중단원 기본 06번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(x) = 3x^2 + 2ax + 3 \geq 0\) (모든 실수 \(x\)에 대해) ⇔ 판별식 \(D/4 = a^2 - 9 \leq 0\) ⇔ \(-3 \leq a \leq 3\).
[기본-도함수 그래프 해석 → 극솟값]
7. 다음 그림은 최고차항의 계수가 \(-1\)인 삼차함수 \(f(x)\)의 도함수 \(y = f'(x)\)의 그래프이다. 함수 \(f(x)\)의 극댓값이 \(7\)일 때, 함수 \(f(x)\)의 극솟값을 구하시오. (교과서 P.97 중단원 기본 07번)
🎯 풀이 전략 보기
💡 전략 힌트: 그래프에서 \(f'(0) = 0\), \(f'(2) = 0\), 위로 볼록한 포물선 → \(f'(x) = -3x(x - 2) = -3x^2 + 6x\). 따라서 \(f(x) = -x^3 + 3x^2 + C\). \(f'\)의 부호: \(x \lt 0\)에서 음, \(0 \lt x \lt 2\)에서 양, \(x \gt 2\)에서 음 → \(x = 0\) 극소, \(x = 2\) 극대. 극댓값 \(f(2) = -8 + 12 + C = 4 + C = 7\) → \(C = 3\). 극솟값 \(f(0) = C = 3\).
[기본-닫힌구간 최솟값]
8. 닫힌구간 \([0, 3]\)에서 함수 \( f(x) = 2x^3 - 3x^2 + a \) 의 최솟값이 \(5\)일 때, 실수 \(a\)의 값을 구하시오. (교과서 P.97 중단원 기본 08번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(x) = 6x^2 - 6x = 6x(x - 1) = 0\) → \(x = 0, 1\). 구간 내 후보: \(f(0) = a\), \(f(1) = 2 - 3 + a = a - 1\), \(f(3) = 54 - 27 + a = a + 27\). 최솟값 \(= a - 1 = 5\) → \(a = 6\).
[기본-두 곡선이 세 점에서 만남]
9. 두 곡선 \( y = x^3 + 4x^2 \), \( y = -2x^2 - 9x + k \) 가 서로 다른 세 점에서 만나도록 하는 실수 \(k\)의 값의 범위를 구하시오. (교과서 P.98 중단원 도전 09번)
🎯 풀이 전략 보기
💡 전략 힌트: 두 곡선의 교점 ⇔ \(x^3 + 4x^2 = -2x^2 - 9x + k\) ⇔ \(x^3 + 6x^2 + 9x = k\). \(g(x) = x^3 + 6x^2 + 9x\)의 극값 분석: \(g'(x) = 3x^2 + 12x + 9 = 3(x+1)(x+3) = 0\) → \(x = -3, -1\). 극댓값 \(g(-3) = -27 + 54 - 27 = 0\), 극솟값 \(g(-1) = -1 + 6 - 9 = -4\). 서로 다른 세 점에서 만나려면 극솟값 \(\lt k \lt\) 극댓값 → \(-4 \lt k \lt 0\).
[기본-부등식 항상 성립]
10. \(x \gt 0\)일 때, 두 함수 \( f(x) = 3x^3 + 8x^2 - 18x + k \), \( g(x) = x^3 - x^2 + 6x + 1 \) 에 대하여 부등식 \( f(x) \gt g(x) \)가 성립하도록 하는 실수 \(k\)의 값의 범위를 구하시오. (교과서 P.98 중단원 도전 10번)
🎯 풀이 전략 보기
💡 전략 힌트: \(h(x) = f(x) - g(x) = 2x^3 + 9x^2 - 24x + (k - 1)\). \(x \gt 0\)에서 \(h(x) \gt 0\) ⇔ \(x \gt 0\)에서 \(h\)의 최솟값 \(\gt 0\). \(h'(x) = 6x^2 + 18x - 24 = 6(x + 4)(x - 1)\). \(x \gt 0\)에서 \(x = 1\)에서 극소(유일). \(h(1) = 2 + 9 - 24 + k - 1 = k - 14 \gt 0\) → \(k \gt 14\).
[기본-두 점의 속도가 같아지는 거리]
11. 수직선 위를 움직이는 두 점 \(P, Q\)의 시각 \(t\)에서 위치가 각각 \( x_P(t) = \dfrac{2}{3}t^3 + 18t - 20 \), \( x_Q(t) = 6t^2 - 7 \) 일 때, 두 점 \(P, Q\)의 속도가 같아지는 순간의 두 점 사이의 거리를 구하시오. (교과서 P.98 중단원 도전 11번)
🎯 풀이 전략 보기
💡 전략 힌트: 속도 \(v_P(t) = 2t^2 + 18\), \(v_Q(t) = 12t\). 같아지는 시각: \(2t^2 + 18 = 12t\) → \(t^2 - 6t + 9 = 0\) → \((t-3)^2 = 0\) → \(t = 3\). 위치: \(x_P(3) = 18 + 54 - 20 = 52\), \(x_Q(3) = 54 - 7 = 47\). 거리 \(= |52 - 47| = 5\).
[기본-사차함수의 극댓값 조건]
12. 함수 \( f(x) = x^4 - 4x^3 - ax^2 \) 이 극댓값을 갖기 위한 실수 \(a\)의 값의 범위를 구하시오. (교과서 P.98 중단원 도전 12번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(x) = 4x^3 - 12x^2 - 2ax = 2x(2x^2 - 6x - a)\). 사차함수(최고차 양수)가 극댓값을 가지려면 \(f'(x) = 0\)이 서로 다른 세 실근을 가져야 한다 (그래야 부호 변화가 \(+ \to - \to + \to -\) 아닌 \(- \to + \to - \to +\) 등 극대 출현). \(2x = 0\)과 \(2x^2 - 6x - a = 0\)이 서로 다른 세 근을 가지려면 ① \(2x^2 - 6x - a = 0\)이 \(0\)이 아닌 서로 다른 두 실근 → \(a \neq 0\)이고 판별식 \(36 + 8a \gt 0 \Rightarrow a \gt -\dfrac{9}{2}\). 종합: \( a \gt -\dfrac{9}{2}\)이고 \(a \neq 0\).
[도전-방정식 |f(x)|=k의 5실근 (서술형)]
13. 함수 \( f(x) = 2x^3 - 3x^2 - 12x + 4 \) 에 대하여 방정식 \( |f(x)| - k = 0 \)이 서로 다른 \(5\)개의 실근을 갖도록 하는 실수 \(k\)의 값을 풀이 과정과 함께 구하시오. (교과서 P.98 중단원 도전 13번)
🎯 서술형 풀이 전략 보기
💡 전략: ① \(f'(x) = 6x^2 - 6x - 12 = 6(x-2)(x+1) = 0\) → \(x = -1, 2\). \(f(-1) = -2 - 3 + 12 + 4 = 11\) (극대), \(f(2) = 16 - 12 - 24 + 4 = -16\) (극소). ② \(|f(x)| = k\)의 실근 = 곡선 \(y = |f(x)|\)와 수평선 \(y = k\)의 교점 개수. \(y = |f(x)|\)의 극값은 \(f\)의 극값의 절댓값과 \(f(x) = 0\) 근에서의 0 (꺾임점). \(y = |f(x)|\)의 모양: \(x \to -\infty\)에서 \(|f| \to \infty\), \(x = -1\)에서 극대 11, \(f\)가 0이 되는 점에서 0 (꺾임), \(x = 2\)에서 \(|f| = 16\)이 극대 (원래 극소의 절댓값), \(x \to \infty\)에서 \(|f| \to \infty\). ③ 수평선 \(y = k\)와 5개의 교점을 가지려면 \(k\)가 작은 극대(11)과 큰 극대(16) 사이여야 한다 → \(11 \lt k \lt 16\).
서술 채점 포인트: \(f\)의 극값 도출, \(|f|\) 그래프 개형 설명 (특히 두 극대값 \(11, 16\)의 존재), \(k\) 범위 결정 근거.
[도전-평균값 정리 + 구간 단속 (서술형)]
14. 구간 단속제란 특정 지점이 아닌 구간의 시작 지점과 종료 지점에 단속 카메라를 설치하여 자동차의 구간 평균속도를 측정한 후 제한 속도 준수 여부를 판정하는 방식이다. 제한 속도가 \(80\,\text{km/h}\)인 도로에서 \(15\,\text{km}\) 구간에 대해 구간 단속을 실시하고 있다. 어느 자동차가 이 구간을 통과하는 데 \(10\)분 걸렸을 때, 이 자동차의 속도가 제한 속도를 초과한 순간이 있었는지 판단하고 그 이유를 평균값 정리를 사용하여 설명하시오. (교과서 P.98 중단원 도전 14번)
🎯 서술형 풀이 전략 보기
💡 전략: ① 시각 \(t\)(시간)에서 위치를 \(x(t)\)라 하자. \(x(t)\)는 \([0, \tfrac{1}{6}]\)에서 연속이고 \((0, \tfrac{1}{6})\)에서 미분가능하다고 가정한다. ② 구간 평균속도 \(= \dfrac{x(1/6) - x(0)}{1/6 - 0} = \dfrac{15}{1/6} = 90\,\text{km/h}\). ③ 평균값 정리: 어떤 시각 \(c \in (0, \tfrac{1}{6})\)가 존재하여 \(x'(c) = 90\,\text{km/h}\). ④ 즉, 이 자동차의 순간속도가 \(90\,\text{km/h}\)인 순간이 반드시 존재한다. \(90 \gt 80\)이므로 제한 속도를 초과한 순간이 반드시 있었다.
서술 채점 포인트: 평균값 정리의 가정 (연속·미분가능), 평균속도 90 도출 (단위 환산 명확), 정리 적용 결론.
