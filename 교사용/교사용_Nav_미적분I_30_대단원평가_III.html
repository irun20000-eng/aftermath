C 단원 회고 발문 (대단원 평가 도입)

발문: "단원 III에서 부정적분·정적분·넓이·속도를 배웠다. 지금 이 순간 가장 자신 없는 도구 하나는 무엇이고, 그 도구가 필요한 문제를 만났을 때 보내는 신호는?"

예상 반응 ①"정적분 안에 함수가 있는 자기참조 문제 — k로 치환하는 게 언제인지 모르겠다."
피드백: 신호 = "적분 안에 \(\int f(t)\,dt\) 형태 + 상수처럼 보이는 부분". 그 즉시 \(k = \int_{a}^{b} f(t)\,dt\)로 놓고 치환.
예상 반응 ②"움직인 거리에서 적분 범위 분할을 빠뜨린다."
피드백: 신호 = "\(\int |v|\,dt\)" 또는 "움직인 거리". 자동으로 \(v(t) = 0\) 먼저 찾아서 부호 변환점 확인.
예상 반응 ③"두 곡선 넓이에서 위·아래가 헷갈린다."
피드백: 교점 구한 뒤 구간 내 임의 점 대입 → 크기 비교. 절대로 감으로 위·아래 결정하지 말 것.

학기말 예고: 다음 차시 학기말 종합은 단원 I·II·III 통합. 오늘 총정리한 도구가 다른 단원과 어떻게 융합되는지를 의식하며 대단원 평가를 마무리한다.

D 개념 지도 노트 — 단원 III 도구 결정 트리 + 오개념

DECISION TREE — 도구별 묶음 분류 (0.5초 선택)
부정적분 기본? 역미분  |  \(\int_a^b f\) 계산? 원함수 대입·사칙  |  자기참조(\(\int f(t)dt\) 상수)? k 치환  |  절댓값 포함? 부호 변환점 구간 분할  |  두 곡선 넓이? 교점→위아래→\(\int(\text{위}-\text{아래})\)  |  속도→위치? \(\int v\,dt\)  |  움직인 거리? \(\int |v|\,dt\) 구간 분할
📌 판서 시연 — 서술형 대비 4단계:
  1. 도구 박스: "이 문제는 [ ] 도구를 쓴다" 첫 줄에 명시.
  2. 핵심 전제 도출: 자기참조→\(k\) 치환, 미정계수→분자→0, 속도→인수분해+부호표.
  3. 식 변형 1회 + 부정형 해소 또는 구간 분할 + 적분 계산.
  4. 최종 답 박스 + 검산 (역미분으로 원래 식 복원 또는 직관 일치 확인).
오개념 ①: "\(\int f(x)\,dx = F(x)\)에서 \(+ C\) 생략 가능." → 부정적분에서 \(+C\) 누락은 감점. 정적분에서만 \(C\)가 소거된다.
오개념 ②: "움직인 거리 = 위치 변화량." → \(\int_a^b v\,dt\)는 위치 변화, \(\int_a^b |v|\,dt\)가 거리. 서술형에서 반드시 구분.
오개념 ③: "극댓값·극솟값이 최댓값·최솟값이다." → 닫힌구간에서는 끝점 후보도 함께 비교.

E 문항별 해설 카드 (14문항 전체 — Q1~Q9 표준 + Q10~Q14 서술형)

표준1부정적분 정의 + \(f(0)\)
\( \int \{x^2 - 2f(x)\}\,dx = 3x^2 + 4x + C \)일 때 \(f(0)\).
양변 미분: \(x^2 - 2f(x) = 6x + 4\). \(f(x) = \dfrac{x^2 - 6x - 4}{2}\). \(f(0) = \dfrac{-4}{2} = -2\).
정답: \(-2\)
표준2두 부정적분의 차 + \(f(3)\)
\(f(x) = \int\!\left(x+\dfrac{1}{x}\right)^2 dx - \int\!\left(x-\dfrac{1}{x}\right)^2 dx\), \(f(1)=-3\).
합산: 피적분함수 차 \(= \left(x+\tfrac{1}{x}\right)^2 - \left(x-\tfrac{1}{x}\right)^2 = 4\). \(f(x) = \int 4\,dx = 4x + C\). \(f(1) = 4+C = -3 \Rightarrow C = -7\). \(f(3) = 12 - 7 = 5\).
정답: \(5\)
표준3정적분 조건 + \(a\) 결정
\(\displaystyle\int_0^2 (-4x^3+3x^2+ax)\,dx = 0\). \(a\) 구하기.
\(\int_0^2 -4x^3\,dx = -16,\quad \int_0^2 3x^2\,dx = 8,\quad \int_0^2 ax\,dx = 2a\). \(-16 + 8 + 2a = 0 \Rightarrow a = 4\).
정답: \(a = 4\)
표준4유리 절댓값 정적분
\(\displaystyle\int_0^2 \dfrac{|x^2-1|}{x+1}\,dx\).
\(\dfrac{|x^2-1|}{x+1} = |x-1|\) (단 \(x \geq 0\)에서 \(x+1 \gt 0\)). \(\int_0^1(1-x)\,dx + \int_1^2(x-1)\,dx = \dfrac{1}{2} + \dfrac{1}{2} = 1\).
정답: \(1\)
표준5적분으로 정의된 함수의 극댓값
\(f(x) = \displaystyle\int_0^x (t^2-3t+2)\,dt\) (\(x \geq 0\)). 극댓값.
\(f'(x) = x^2-3x+2 = (x-1)(x-2)\). 부호: \(0\leq x\lt 1\) 양, \(1\lt x\lt 2\) 음, \(x\gt 2\) 양. \(x=1\)에서 극대. \(f(1) = \left[\dfrac{t^3}{3}-\dfrac{3t^2}{2}+2t\right]_0^1 = \dfrac{1}{3}-\dfrac{3}{2}+2 = \dfrac{5}{6}\).
정답: 극댓값 \(\dfrac{5}{6}\)
판서 포인트: \(x=2\)에서는 극소 (\(f(2)=0\) 계산 불필요). "극댓값" 물음이므로 \(x=1\) 결과만.
표준6넓이 이등분 + \(a\) 결정
\(y=x^2\) (\(x\geq 0\)), \(y\)축, \(y=1\)로 둘러싼 넓이를 \(y=ax^2\) (\(a\gt 1\))이 이등분. \(a\).
전체 넓이 \(S = \int_0^1 \sqrt{y}\,dy = \dfrac{2}{3}\). 이등분 목표 \(= \dfrac{1}{3}\). 두 곡선 사이 \(y\)방향 넓이: \(\int_0^1\!\left(\sqrt{y}-\sqrt{y/a}\right)dy = \dfrac{2}{3}\!\left(1-\dfrac{1}{\sqrt{a}}\right)\). \(\dfrac{2}{3}\!\left(1-\dfrac{1}{\sqrt{a}}\right) = \dfrac{1}{3} \Rightarrow \sqrt{a}=2 \Rightarrow a=4\).
정답: \(a = 4\)
표준7곡선과 직선 넓이 \(\to m\)
\(y=x^2-x\)와 \(y=m(x-1)\) (\(m\gt 1\)) 넓이 \(=\dfrac{9}{2}\). \(m\).
교점: \(x^2-(m+1)x+m=0\) → \((x-1)(x-m)=0\) → \(x=1, m\). 공식 \(\dfrac{(\beta-\alpha)^3}{6}\): 넓이 \(=\dfrac{(m-1)^3}{6}=\dfrac{9}{2}\) → \((m-1)^3=27\) → \(m=4\).
정답: \(m = 4\)
표준8속도 그래프 (꺾은선) + \(k\)
꺾은선 \(v(t)\), \(t=5\)에서 위치 \(-\dfrac{5}{2}\). \(k\) 구하기.
구간별 부호있는 넓이: \([0,2]\) 삼각형 \(+2\), \([2,3]\) 삼각형 \(-1\), \([3,4]\) 사다리꼴 \(-\dfrac{3}{2}\), \([4,5]\) 사다리꼴 \(\dfrac{k-1}{2}\). 합 \(= 2-1-\dfrac{3}{2}+\dfrac{k-1}{2} = -\dfrac{5}{2}\) → \(\dfrac{k}{2}=-\dfrac{5}{2}\) → \(k=-5\).
정답: \(k = -5\)
판서 포인트: 꺾은선 그래프 넓이 = 삼각형·사다리꼴. 부호(v의 부호)를 넓이에 곱하는 것을 학생이 잊는 경우 많음.
표준9속도와 거리 (정지까지)
\(v(t)=20-4t\). 제동 후 정지까지 거리.
정지: \(v=0\) → \(t=5\). 거리 \(=\int_0^5(20-4t)\,dt = [20t-2t^2]_0^5 = 100-50 = 50\,\text{m}\).
정답: \(50\,\text{m}\)
서술형10함수방정식 + \(f'(0)\) → \(f(x)\)
\(f(x+y)=f(x)+f(y)+3xy\), \(f'(0)=5\). \(f(x)\) 구하기.
① \(x=y=0\): \(f(0)=2f(0)\) → \(f(0)=0\). ② 미분계수 정의: \(f'(x) = \lim_{h\to 0}\dfrac{f(x)+f(h)+3xh-f(x)}{h} = \lim_{h\to 0}\dfrac{f(h)}{h}+3x = f'(0)+3x = 5+3x\). ③ 부정적분: \(f(x)=\int(3x+5)\,dx = \dfrac{3}{2}x^2+5x+C\). \(f(0)=0\) → \(C=0\).
정답: \(f(x) = \dfrac{3}{2}x^2 + 5x\)
서술 채점 포인트: ① \(f(0)=0\) 도출 ② 미분계수 정의에서 \(f'(0)\) 활용 ③ 부정적분+상수 결정 — 3단계 모두 명시.
서술형11합·곱의 도함수 → \(f(x), g(x)\) 결정
\(\dfrac{d}{dx}\{f+g\}=4x-2\), \(\dfrac{d}{dx}\{fg\}=6x^2-14x+10\), \(f(0)=4\), \(g(0)=-2\).
① 합 부정적분: \(f+g = 2x^2-2x+C_1\). \(x=0\): \(4+(-2)=C_1=2\). 즉 \(f+g=2x^2-2x+2\). ② 곱 부정적분: \(fg = 2x^3-7x^2+10x+C_2\). \(x=0\): \(4\cdot(-2)=C_2=-8\). 즉 \(fg=2x^3-7x^2+10x-8\). ③ 인수 탐색: \(x=2\)에서 \(fg=0\) → 인수 \((x-2)\). 조립제법: \(fg=(x-2)(2x^2-3x+4)\). ④ 합 확인: \((x-2)+(2x^2-3x+4)=2x^2-2x+2\) ✓. 상수항: \(f(0)=4\to f=2x^2-3x+4\), \(g=x-2\).
정답: \(f(x)=2x^2-3x+4\), \(g(x)=x-2\)
서술 채점 포인트: 합·곱 각각 부정적분+상수 결정, 인수정리(\(x-2\)) 적용, 상수항으로 \(f,g\) 식별 — 4단계.
서술형12자기참조 정적분 (\(k\) 치환) — 두 물음
\(f(x)=-4x^3+3x^2+\displaystyle\int_{-1}^{1}f(t)\,dt\). (1) \(f(0)\) (2) \(\displaystyle\int_{-2}^{2}f(x)\,dx\).
① \(k=\int_{-1}^{1}f(t)\,dt\) 놓으면 \(f(x)=-4x^3+3x^2+k\). ② \(k=\int_{-1}^{1}(-4t^3+3t^2+k)\,dt\): 홀함수 \(-4t^3\to 0\), \(\int_{-1}^{1}3t^2\,dt=2\), \(\int_{-1}^{1}k\,dt=2k\). \(k=2+2k\) → \(k=-2\). ③ \(f(x)=-4x^3+3x^2-2\). (1) \(f(0)=-2\). ④ (2) \(\int_{-2}^{2}f(x)\,dx\): 홀함수 \(-4x^3\to 0\). \(2\int_0^2(3x^2-2)\,dx=2[x^3-2x]_0^2=2(8-4)=8\).
정답: (1) \(f(0)=-2\)   (2) \(8\)
서술 채점 포인트: \(k\) 치환 명시, 홀·짝함수 대칭 활용, 두 단계 답 각각 명시.
서술형13정적분 안 함수 미분 → 넓이
\(x^2 f(x)=\dfrac{1}{5}x^5-x^4+\displaystyle\int_0^x 2t\,f(t)\,dt\), \(f(0)=0\). (1) \(f(x)\) (2) \(y=f(x)\)와 \(x\)축 넓이.
① 양변 미분: \(2xf(x)+x^2f'(x)=x^4-4x^3+2xf(x)\) → \(x^2f'(x)=x^4-4x^3\) → \(f'(x)=x^2-4x\). ② \(f(x)=\dfrac{x^3}{3}-2x^2+C\). \(f(0)=0\) → \(C=0\). \(f(x)=\dfrac{x^2(x-6)}{3}\). ③ 교점: \(f(x)=0\) → \(x=0\)(이중근), \(x=6\). \([0,6]\)에서 \(f\leq 0\). 넓이 \(=-\int_0^6\!\left(\dfrac{x^3}{3}-2x^2\right)dx = -\left[\dfrac{x^4}{12}-\dfrac{2x^3}{3}\right]_0^6 = -(108-144)=36\).
정답: (1) \(f(x)=\dfrac{x^3}{3}-2x^2\)   (2) \(36\)
서술 채점 포인트: 양변 미분+정적분 부분 미분 명시, 적분상수 결정, 교점+부호 분석, 넓이 계산 — 4단계.
서술형14속도 — 속도 최대 위치 + 방향 전환까지 거리
\(v(t)=-3t^2+12t-9\). 원점 출발. (1) 속도 최대 시 위치 (2) 두 번째 방향 전환까지 움직인 거리.
① 인수분해: \(v(t)=-3(t-1)(t-3)\). 부호: \(t\lt 1\) 음, \(1\lt t\lt 3\) 양, \(t\gt 3\) 음. 방향 전환: \(t=1,3\). (1) \(v(t)=-3(t-2)^2+3\) → 최대 \(t=2\). 위치 \(x(2)=\int_0^2(-3t^2+12t-9)\,dt=[-t^3+6t^2-9t]_0^2=-8+24-18=-2\). (2) \([0,1]\)에서 \(v\lt 0\): \(\int_0^1|v|\,dt=[t^3-6t^2+9t]_0^1=1-6+9=4\). \([1,3]\)에서 \(v\gt 0\): \(\int_1^3 v\,dt=[-t^3+6t^2-9t]_1^3=(0)-(-4)=4\). 합 \(4+4=8\).
정답: (1) 위치 \(-2\)   (2) 거리 \(8\)
서술 채점 포인트: \(v\) 인수분해+부호표, 최대 속도 시점·위치 적분, 방향 전환 시각 식별, 움직인 거리 구간 분할 — 4단계.

G 대단원 성찰 — 3유형 우수 답안 예

유형 1: 단원 종합 통찰 (부정적분 ↔ 정적분 ↔ 넓이·거리) "단원 III를 정리하면서 부정적분이 미분의 역연산이라는 출발점에서 시작해, 정적분이 '함수값의 누적 변화'를 측정한다는 본질로 확장된다는 게 보였다. 미적분 기본정리 \(\int_a^b f'(x)\,dx=f(b)-f(a)\)가 '도함수의 누적 = 원함수의 변화'라는 한 줄로 단원 II와 III을 잇는 다리였다. 넓이·속도·\(\int f(t)\,dt=g(x)\)형 방정식이 모두 이 다리의 응용이라는 게 한 그림으로 정리됐다."
유형 2: 실수 분석 (단원 III 자주 틀린 유형) "단원 III에서 가장 자주 틀린 건 ① 부정적분에서 \(+C\) 누락, ② 절댓값 정적분에서 구간 분할 누락, ③ 움직인 거리에서 \(\int v\,dt\)와 \(\int|v|\,dt\) 혼동 세 가지였다. '① 부정적분이면 \(+C\) 표시 → ② \(|f|\)면 부호 변환점에서 구간 분할 → ③ 위치 변화 vs 움직인 거리 구분' 3단계 체크리스트를 시험지 여백에 적어두고 풀고 있다."
유형 3: 학기 전체 통합 (학기말 종합으로의 다리) "단원 I의 극한 정의가 단원 II의 미분계수 \(\lim_{h\to 0}\dfrac{f(x+h)-f(x)}{h}\)와 단원 III의 구분구적법 \(\int_a^b f\,dx=\lim_{n\to\infty}\sum f(x_k)\Delta x\)를 동시에 받친다는 게 신기했다. 학기말 종합 평가에서 이 세 축의 연결을 의식하며 문제를 풀려고 한다."
