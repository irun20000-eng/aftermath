C 생각 열기 운영 가이드

발문: 도함수 \(y = f'(x)\)가 \(x = -1, 1\)에서 \(x\)축을 가로지른다는 사실만 알아도 함수 \(y = f(x)\)의 그래프 모양은 하나로 정해질까? 정해지지 않는다면 무엇을 더 알아야 할까?

예상 반응 ①"증감만 알아도 그래프는 하나로 정해진다."
피드백: 반례 — \(f(x) = x^3 - 3x\)와 \(f(x) + 100\)은 증감이 같지만 그래프 위치가 다르다. "평행이동의 자유도"를 제거하는 정보가 필요함을 유도.
예상 반응 ②"극값의 \(y\)좌표를 알면 결정된다."
피드백: 정답에 매우 근접 — 극값의 \(y\)좌표 + \(y\)절편이면 충분. 5단계 STEP 3·4가 정확히 이 두 정보를 잡는 단계임을 예고.
예상 반응 ③"\(x\)절편을 알면 된다."
피드백: 부분 정답. \(x\)절편은 위치 결정에 결정적이지만 정확한 값이 무리수일 수도 — 개형에는 "몇 개인지"가 더 중요. 차시 18 실근 개수와 직결됨을 예고.

1학기 연결: 1학기 수학 II에서 익힌 "다항함수 그래프 그리기" 직관(\(x \to \pm\infty\) 거동, \(y\)절편)을 미분 도구(증감·극값)로 정량화·체계화하는 차시.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE ALGORITHM — 5 STEPS
\( f'(x) \to f'(x) = 0 \text{의 해} \to \text{증감표} + \text{극값} \to \text{좌표축 교점} \to \text{개형} \)
📌 판서 시연 5단계 (예제 \(y = 2x^3 - 6x + 1\)):
  1. \(f'(x) = 6x^2 - 6 = 6(x+1)(x-1)\) → 인수분해까지 완성.
  2. \(f'(x) = 0 \Rightarrow x = -1, 1\) (두 임계점).
  3. 증감표: 부호 \((+, -, +)\) → 극댓값 \(f(-1) = 5\), 극솟값 \(f(1) = -3\).
  4. \(y\)절편 \(f(0) = 1\). \(x\)절편은 정확한 값 불요(개형은 "위치"만 알면 충분).
  5. 좌하 \(\to (-1, 5)\) 극대 \(\to (0, 1)\) \(\to (1, -3)\) 극소 \(\to\) 우상. 양 끝 방향은 최고차계수 부호로 결정.
오개념 ①: "사차함수면 항상 \(W\) 모양 (극값 3개)." → 1순위 함정. \(f'(x) = 0\)의 실근 개수에 따라 극값이 1개 또는 3개. 2개나 0개는 불가능.
오개념 ②: "\(f'(x) = 0\)이면 극값이다." → 부호 변화 없으면 정체점(예: \(y = -x^3 + 6x^2 - 12x + 4\)의 \(x = 2\)). 차시 15에서 강조한 규칙 재확인.
오개념 ③: "극솟값이 \(0\)이면 \(x\)축과 한 번 만난다." → 정반대. 극값이 \(0\)이면 그 점에서 접함(중근) → 그 외 한 점 더 = 총 두 점에서 교차. (단원 18의 핵심 규칙 미리보기.)

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11삼차함수 개형 — 5단계 완주
(1) \(y = x^3 + 3x^2 - 2\)   (2) \(y = -x^3 + 6x^2 - 12x + 4\)
(1) \(f'(x) = 3x(x+2)\). 부호 \((+,-,+)\). 극댓값 \(f(-2) = -8+12-2 = 2\), 극솟값 \(f(0) = -2\). 좌하 \(\to (-2, 2)\) 극대 \(\to (0, -2)\) 극소 \(\to\) 우상. (2) \(f'(x) = -3(x-2)^2 \leq 0\) 항상 → 부호 변화 없음 → 극값 없이 단조감소. \(x = 2\)는 정체점. \(y\)절편 \(4\). 좌상 \(\to\) 단조감소 \(\to\) 우하.
정답: (1) 극대 \(2\), 극소 \(-2\) (2) 극값 없음(단조감소), \(x = 2\) 정체점
판서 포인트: (2)에서 "\(f'(2) = 0\)인데 왜 극값이 아닌가?"를 학생이 직접 답하게 한다 — "부호가 안 바뀌니까". 차시 15 규칙 재확인.
L12사차함수 개형 — 극값 3개 vs 1개
(1) \(y = x^4 - 2x^2 - 2\)   (2) \(y = -\dfrac{3}{4}x^4 + 2x^3 + 1\)
(1) \(f'(x) = 4x(x-1)(x+1)\). 부호 \((-,+,-,+)\). 극솟값 \(f(\pm 1) = 1 - 2 - 2 = -3\), 극댓값 \(f(0) = -2\). \(W\) 모양, \(y\)축 대칭. (2) \(f'(x) = -3x^2(x-2)\). \(f' = 0\)의 해 \(x = 0, 2\). \(x = 0\)은 좌우 부호 같음(\(+, +\)) → 정체점. \(x = 2\)만 부호 \(+ \to -\) → 극댓값 \(f(2) = -12 + 16 + 1 = 5\). 단조증가 \(\to (2, 5)\) 극대 \(\to\) 우하.
정답: (1) 극솟값 \(-3\) (2회), 극댓값 \(-2\) (2) 극댓값 \(5\)만 존재, \(x = 0\) 정체점
L13좌표축 교점 — 극솟값 = 0이면 접함(중근)
\(y = x^3 - 3x^2 + 4\)와 \(x\)축의 교점을 모두 구하고 개형을 그리시오.
\(f'(x) = 3x(x - 2)\). 부호 \((+,-,+)\). 극댓값 \(f(0) = 4\), 극솟값 \(f(2) = 8 - 12 + 4 = 0\). \(x\)축 교점: \(x^3 - 3x^2 + 4 = 0\). 인수분해 \((x+1)(x-2)^2 = 0\) → \(x = -1\) 또는 \(x = 2\)(중근). 교점은 \((-1, 0), (2, 0)\) 두 곳. \(x = 2\)에서는 \(x\)축에 접한다(극솟값이 정확히 \(0\)이므로).
정답: 교점 \((-1, 0), (2, 0)\). \((2, 0)\)에서 접함. 좌하 \(\to (-1, 0)\) 가로지름 \(\to (0, 4)\) 극대 \(\to (2, 0)\) 접점·극소 \(\to\) 우상.
차시 18 미리보기: "극솟값 \(= 0 \Leftrightarrow x\)축과 접함(중근) \(\Leftrightarrow\) 방정식이 중근 보유"를 학생 노트에 색깔로 기록시킨다. 차시 18 형성평가의 핵심 규칙.
L24도함수 그래프 \(\to\) 원함수 개형
도함수 \(y = f'(x)\) 그래프(아래로 볼록 포물선, \(x = \pm 1\)에서 \(x\)축)가 주어지고 \(f(-1) = 2, f(1) = -2\). \(y = f(x)\) 개형을 그리시오.
도함수 부호: \(x \lt -1\)에서 \(+\), \(-1 \lt x \lt 1\)에서 \(-\), \(x \gt 1\)에서 \(+\) → \((+,-,+)\). \(x = -1\)에서 극댓값 \(f(-1) = 2\), \(x = 1\)에서 극솟값 \(f(1) = -2\). 두 함숫값까지 주어졌으므로 평행이동 자유도 제거 → 개형 유일 결정. 좌하 \(\to (-1, 2)\) 극대 \(\to (1, -2)\) 극소 \(\to\) 우상.
정답: 극댓값 \(2\) (\(x = -1\)), 극솟값 \(-2\) (\(x = 1\)). 극값 부호가 반대 → \(x\)축과 세 점에서 교차(차시 18 미리보기).
판서 포인트: 도함수 그래프에서 "\(f'\)이 0인 \(x\) 좌표 + 부호" 두 가지만 읽으면 원함수의 임계점·증감이 결정. 학생이 흔히 도함수 그래프의 "\(y\)값"을 원함수의 \(y\)값으로 혼동 — 반드시 분리시킬 것.
L25매개변수 결정 + 개형 (근과 계수의 관계)
\(f(x) = x^3 + ax^2 + bx + c\)가 \((0, 4)\) 지나고 \(x = 1\) 극대, \(x = 3\) 극소. \(a, b, c\) 및 극솟값 + 개형.
\(f(0) = c = 4\). \(f'(x) = 3x^2 + 2ax + b\). 극값 조건 → \(f'(1) = f'(3) = 0\), 두 근의 합·곱: \(1 + 3 = -\dfrac{2a}{3} \Rightarrow a = -6\), \(1 \cdot 3 = \dfrac{b}{3} \Rightarrow b = 9\). \(f(x) = x^3 - 6x^2 + 9x + 4\). 극댓값 \(f(1) = 1 - 6 + 9 + 4 = 8\), 극솟값 \(f(3) = 27 - 54 + 27 + 4 = 4\). 두 극값 모두 양수 → 그래프는 \(x\)축과 한 점에서만 만남.
정답: \(a = -6, b = 9, c = 4\), 극솟값 \(4\). \(x\)절편 1개(차시 18 케이스 ③ "곱 \(\gt 0\)").
L26개형 + 실근 개수 (차시 18 미리보기)
\(f(x) = x^3 - 3x + 2\)의 개형을 그리고 \(x^3 - 3x + 2 = 0\)의 서로 다른 실근의 개수.
\(f'(x) = 3(x-1)(x+1)\). 부호 \((+,-,+)\). 극댓값 \(f(-1) = -1 + 3 + 2 = 4\), 극솟값 \(f(1) = 1 - 3 + 2 = 0\). 극솟값이 정확히 \(0\) → \(x = 1\)에서 \(x\)축에 접함(중근). 인수분해 \((x-1)^2(x+2) = 0\) → \(x = 1\)(중근) 또는 \(x = -2\). 서로 다른 실근은 2개(\(x = -2, 1\)). 단, \(x = 1\)은 중근이므로 중복도 포함 시 3개.
정답: 서로 다른 실근 2개. 핵심 규칙: "극값 중 하나가 \(0\) \(\Leftrightarrow\) 중근" — 차시 18 표준 규칙.
L37🚨 오답률 1위 — 사차함수 극값 1개 vs 3개
\(f(x) = x^4 + ax^2 + b\)에서 (1) 극값이 1개 (2) 극값이 3개인 \(a\)의 부호.
\(f'(x) = 4x^3 + 2ax = 2x(2x^2 + a)\). \(f' = 0 \Rightarrow x = 0\) 또는 \(2x^2 + a = 0\), 즉 \(x^2 = -\dfrac{a}{2}\). ① \(a \geq 0\): \(x^2 = -\dfrac{a}{2} \leq 0\) → 해 없음 또는 중근. \(f' = 0\)인 실수는 \(x = 0\) 단 하나. \(x \lt 0\)에서 \(f' \lt 0\), \(x \gt 0\)에서 \(f' \gt 0\) → 부호 \((-,+)\) → 극소만 (극값 1개). ② \(a \lt 0\): \(x = \pm\sqrt{-\dfrac{a}{2}}\). 세 임계점. 부호 \((-,+,-,+)\) → 극소·극대·극소 3개 (\(W\) 모양).
정답: (1) 극값 1개 ⇔ \(a \geq 0\) (2) 극값 3개 ⇔ \(a \lt 0\). (\(b\)는 \(y\)절편이므로 개형 무영향.)
발표 유도: "사차함수에서 극값이 2개나 0개일 수 있는가?"를 학생이 답하게 한다 — 답: 불가능(\(f'\)이 3차이므로 실근 1개 또는 3개). 학생들이 가장 흔히 빠지는 함정: "사차함수면 \(W\) 모양". 도함수 영점 개수가 결정이라는 원리로 환원.
FC8형성평가 — \(x^3 - 3x^2 + a = 0\)이 세 실근
\(f(x) = x^3 - 3x^2 + a\)의 그래프가 \(x\)축과 서로 다른 세 점에서 만나는 \(a\)의 범위.
\(f'(x) = 3x(x - 2)\). 부호 \((+,-,+)\). 극댓값 \(f(0) = a\), 극솟값 \(f(2) = 8 - 12 + a = a - 4\). 세 실근 조건: 극대 \(\gt 0 \gt\) 극소, 즉 (극댓값)(극솟값) \(\lt 0\). \(a(a - 4) \lt 0 \Rightarrow 0 \lt a \lt 4\). 경계 검증: \(a = 0\)이면 극댓값 \(0\), 접함(중근) → 실근 2개. \(a = 4\)이면 극솟값 \(0\), 접함 → 실근 2개. 두 경계 모두 제외.
정답: \(0 \lt a \lt 4\). 차시 18에서 이 사고를 매개변수 분리(\(g(x) = -x^3 + 3x^2 = a\))로 일반화.
CH1D1⭐ 수능특강 수학Ⅱ 도함수의 활용 2 유제 2 — 절댓값 + 닫힌구간 최댓값·최솟값
\([-2, 2]\)에서 \(f(x) = x^3 - 3|x + 1| + 3\)의 \(M, m\)일 때 \(M + m\). ① \(-6\) ② \(-4\) ③ \(-2\) ④ \(0\) ⑤ \(2\)
STEP 1: \(x \lt -1\): \(|x+1| = -(x+1)\) → \(f(x) = x^3 + 3x + 6\), \(f'(x) = 3x^2 + 3 \gt 0\) → 단조증가. STEP 2: \(x \geq -1\): \(|x+1| = x+1\) → \(f(x) = x^3 - 3x\), \(f'(x) = 3(x-1)(x+1)\) → \(x = 1\)에서 극소(부호 \(- \to +\)). STEP 3: 후보값 \(f(-2) = -8\), \(f(-1) = 2\) (좌·우 일치), \(f(1) = -2\), \(f(2) = 2\). STEP 4: \(M = 2\) (at \(x = -1\) 또는 \(x = 2\)), \(m = -8\) (at \(x = -2\)). \(M + m = -6\).
정답: ① \(-6\). 핵심: 절댓값 분기점에서 구간을 쪼개 부분함수 명시 + 분기점 연속성 확인 + 양 끝값·극값 모두 비교.
CH2D2⭐ 2004년 10월 학력평가 가형 6번 [3점]
삼차함수 \(y = f(x)\)의 도함수 \(y = f'(x)\) 그래프(위로 볼록 포물선, \(x = \pm 2\)에서 \(x\)축, 정점 \(y = 3\)). \(f(0) = 0\)일 때 \(f(x) = kx\)가 세 실근을 가질 \(k\)의 범위. ① \(k \gt 2\) ② \(k \gt 3\) ③ \(k \lt 3\) ④ \(-4 \lt k \lt 4\) ⑤ \(k \lt -2\) 또는 \(k \gt 2\)
STEP 1: \(f'(x) = -\dfrac{3}{4}(x+2)(x-2) = -\dfrac{3}{4}x^2 + 3\) (정점 \(f'(0) = 3\) 일치). STEP 2: \(f(x) = -\dfrac{x^3}{4} + 3x + C\). \(f(0) = 0 \Rightarrow C = 0\). 따라서 \(f(x) = -\dfrac{x^3}{4} + 3x\). STEP 3: \(f(x) = kx \Leftrightarrow x\!\left(-\dfrac{x^2}{4} + 3 - k\right) = 0\). \(x = 0\)은 항상 근. 나머지 두 근은 \(x^2 = 4(3 - k)\). STEP 4: 두 0이 아닌 실근 \(\Leftrightarrow 4(3 - k) \gt 0 \Leftrightarrow k \lt 3\). (\(k = 3\)이면 \(x = 0\) 중근으로 실근 1개.)
정답: ③ \(k \lt 3\). 핵심: 도함수 그래프 \(\to\) 원함수 식 복원(적분상수는 초기조건) \(\to\) 매개변수 방정식 분석.
