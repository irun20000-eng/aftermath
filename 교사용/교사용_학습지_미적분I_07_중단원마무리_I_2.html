01. 실전 훈련 (Textbook Mastery)

💡 비상 미적분 I 교과서 중단원 마무리(P.40~P.42) 14문항을 그대로 학습지화. 기초 1~3 · 기본 4~12 · 도전 13·14 (서술형 킬러)

[기초-한 점 연속·불연속 판정]
1. 다음 함수가 \(x = 3\)에서 연속인지 불연속인지 판정하시오. (교과서 P.40 중단원 기초 01번)
(1) \( f(x) = \dfrac{2}{x+1} \)    (2) \( f(x) = \begin{cases} x + 1 & (x \neq 3) \\ -2 & (x = 3) \end{cases} \)
🎯 풀이 전략 보기
💡 전략 힌트: (1) \(f(3) = \dfrac{2}{4} = \dfrac{1}{2}\), \(\lim_{x \to 3} f(x) = \dfrac{1}{2}\) 일치 → 연속. (2) \(\lim_{x \to 3}(x+1) = 4\)이지만 \(f(3) = -2\) → 두 값이 다름 → 불연속 (제거 가능형).
[기초-연속인 구간 구하기]
2. 다음 함수가 연속인 구간을 구하시오. (교과서 P.40 중단원 기초 02번)
(1) \( f(x) = x^2 + x + 3 \)    (2) \( f(x) = \sqrt{2x - 4} \)
🎯 풀이 전략 보기
💡 전략 힌트: (1) 다항함수 → 모든 실수에서 연속, 즉 \((-\infty, \infty)\). (2) 근호 안 \(2x - 4 \geq 0 \iff x \geq 2\). 끝점 \(x = 2\)에서 우극한 = \(f(2) = 0\)이므로 \([2, \infty)\)에서 연속.
[기초-유리함수의 연속성 조사]
3. 두 함수 \( f(x) = x^2 + 1 \), \( g(x) = x^2 - 2x - 3 \) 에 대하여 다음 함수의 연속성을 조사하시오. (교과서 P.40 중단원 기초 03번)
(1) \( \dfrac{f(x)}{g(x)} \)    (2) \( \dfrac{g(x)}{f(x)} \)
🎯 풀이 전략 보기
💡 전략 힌트: 두 다항함수의 비. 분모 영점만 점검. (1) \(g(x) = (x-3)(x+1) = 0\) → \(x = 3, -1\)에서 불연속, 그 외 연속. (2) \(f(x) = x^2 + 1 \geq 1 \gt 0\) → 분모 0 안 됨 → 모든 실수에서 연속.
[기본-닫힌구간 최댓값·최솟값]
4. 다음 구간에서 함수의 최댓값과 최솟값을 구하시오. (교과서 P.40 중단원 기초 04번)
(1) \( f(x) = x^2 + 2x - 5 \),  \([-4, 1]\)    (2) \( f(x) = \sqrt{x - 2} \),  \([3, 6]\)
🎯 풀이 전략 보기
💡 전략 힌트: (1) 꼭짓점 \(x = -1\)에서 최솟값 \(-6\), 양끝 \(f(-4) = 3, f(1) = -2\) → 최댓값 \(3\). (2) 증가함수, 최솟값 \(f(3) = 1\), 최댓값 \(f(6) = 2\).
[기본-그래프로 보기 판정]
5. 함수 \(y = f(x)\)의 그래프가 다음 그림과 같다. <보기> 중 옳은 것을 모두 고르시오. (교과서 P.41 중단원 기본 05번)
<보기> ㄱ. \( \displaystyle \lim_{x \to 1} f(x) = 4 \)
ㄴ. \( \displaystyle \lim_{x \to 0} f(x) = f(0) \)
ㄷ. 열린구간 \((0, 5)\)에서 \(f(x)\)가 불연속인 \(x\)의 값은 2개이다.
ㄹ. 열린구간 \((0, 5)\)에서 \(f(x)\)의 극한값이 존재하지 않는 \(x\)의 값은 2개이다.
🎯 풀이 전략 보기
💡 전략 힌트: 그림에서 의심 후보점은 \(x = 1, 3, 4\) (각각 빈 원·검은 점이 표시됨). 각 점에서 좌극한·우극한·함숫값을 표로 정리. 극한 존재 ⇔ 좌·우극한 일치. 연속 ⇔ (극한 존재) + (극한값 = 함숫값). 두 개념이 별개임에 유의 — "불연속 개수"와 "극한 존재하지 않는 개수"는 다를 수 있음.
[기본-구간 분기 + 한 점 연속 미정계수]
6. 함수 \( f(x) = \begin{cases} \dfrac{x^2 + ax + b}{x - 2} & (x \neq 2) \\ 3 & (x = 2) \end{cases} \) 가 모든 실수 \(x\)에서 연속일 때, 실수 \(a, b\)의 값을 구하시오. (교과서 P.41 중단원 기본 06번)
🎯 풀이 전략 보기
💡 전략 힌트: \(x = 2\)에서 연속 → 극한값 \(= 3\). 분모 \(\to 0\)이고 극한이 실수 → 분자도 \(\to 0\) → \(4 + 2a + b = 0\). 분자 인수분해해 \(x = 2\)를 근으로 가지면 \( = (x-2)(x - r) \). \(\lim = 2 - r = 3 \Rightarrow r = -1\). 따라서 \(x^2 + ax + b = (x-2)(x+1) = x^2 - x - 2\) → \(a = -1, b = -2\).
[기본-구간 분기 함수의 연속·미정계수 곱]
7. 모든 실수 \(x\)에서 함수 \( f(x) = \begin{cases} 3x^2 - 1 & (x \geq a) \\ 2 - 5x & (x \lt a) \end{cases} \) 가 연속일 때, 모든 실수 \(a\)의 값의 곱을 구하시오. (교과서 P.41 중단원 기본 07번)
🎯 풀이 전략 보기
💡 전략 힌트: 경계 \(x = a\)에서 좌극한 = 우극한·함숫값. \(2 - 5a = 3a^2 - 1 \Rightarrow 3a^2 + 5a - 3 = 0\). 비에타에 의해 두 근의 곱 \(= \dfrac{-3}{3} = -1\).
[기본-곱 함수의 연속 조건]
8. \(x \neq 2\)인 모든 실수 \(x\)에서 연속인 함수 \(f(x)\)가 \( (x - 3)\,f(x) = \sqrt{x + a} - 1 \) 을 만족시킬 때, \(f(3)\)의 값을 구하시오. (단, \(a\)는 실수) (교과서 P.41 중단원 기본 08번)
🎯 풀이 전략 보기
💡 전략 힌트: \(x = 3\)에서 연속이려면 \(\lim_{x \to 3}(x-3)f(x) = 0 = \lim_{x \to 3}(\sqrt{x+a} - 1) = \sqrt{3+a} - 1\) → \(a = -2\). 그러면 \(f(x) = \dfrac{\sqrt{x-2} - 1}{x - 3}\) (\(x \neq 3\)). 유리화하여 \(\dfrac{1}{\sqrt{x-2} + 1}\). \(x \to 3\) → \(\dfrac{1}{2}\). 따라서 \(f(3) = \dfrac{1}{2}\).
[기본-극한 조건 → 함숫값]
9. \(x \neq 0\)인 모든 실수 \(x\)에서 연속인 함수 \(f(x)\)가 \( \displaystyle \lim_{x \to 1}\dfrac{(\sqrt{x} - 1)\,f(x)}{x^3 - 1} = 5 \) 를 만족시킬 때, \(f(1)\)의 값을 구하시오. (교과서 P.41 중단원 기본 09번)
🎯 풀이 전략 보기
💡 전략 힌트: 분자·분모 유리화 및 인수분해. \(\sqrt{x} - 1\)에 \(\sqrt{x} + 1\) 곱해 \( \dfrac{x - 1}{\sqrt{x} + 1}\). \(x^3 - 1 = (x-1)(x^2 + x + 1)\). 약분 후 \(\dfrac{f(x)}{(\sqrt{x}+1)(x^2+x+1)}\). \(x \to 1\) → \(\dfrac{f(1)}{2 \cdot 3} = 5\) → \(f(1) = 30\).
[기본-연속함수의 사칙 보기 판정]
10. 두 함수 \(f(x), g(x)\)가 각각 \(x = a\)에서 연속일 때, 다음 보기 중 \(x = a\)에서 반드시 연속인 함수를 모두 고르시오. (교과서 P.42 중단원 기본 10번)
<보기> ㄱ. \( 2f(x) - g(x) \)    ㄴ. \( \{f(x)\}^2 \)    ㄷ. \( \dfrac{f(x)}{g(x)} \)    ㄹ. \( \dfrac{2}{\{f(x)\}^2 + 1} \)
🎯 풀이 전략 보기
💡 전략 힌트: 사칙 정리에서 분수만 분모 \(\neq 0\) 조건이 필요. ㄱ, ㄴ — 항상 연속. ㄷ — \(g(a)=0\)일 수 있어 보장 X. ㄹ — 분모 \(\geq 1 \gt 0\) → 항상 연속. 정답: ㄱ, ㄴ, ㄹ.
[기본-닫힌구간 위 분수함수 최댓값]
11. 닫힌구간 \([1, 3]\)에서 함수 \( f(x) = \dfrac{a}{|x - 4|} \)의 최댓값이 \(8\)일 때, 양수 \(a\)의 값을 구하시오. (교과서 P.42 중단원 기본 11번)
🎯 풀이 전략 보기
💡 전략 힌트: \([1, 3]\)에서 \(x \lt 4\)이므로 \(|x-4| = 4 - x\). \(f(x) = \dfrac{a}{4 - x}\)는 \(x\)가 증가할수록 분모 감소 → \(f\) 증가. 최댓값은 \(x = 3\)에서 \(\dfrac{a}{1} = a = 8\).
[기본-사잇값 정리로 실근 구간 식별]
12. 방정식 \( 2x^3 - 3x^2 - x + 1 = 0 \)은 서로 다른 세 실근을 갖는다. 다음 보기 중 이 실근이 존재하는 구간을 모두 고르시오. (교과서 P.42 중단원 기본 12번)
<보기>  ㄱ. \((-1, 0)\)    ㄴ. \((0, 1)\)    ㄷ. \((1, 2)\)    ㄹ. \((2, 3)\)
🎯 풀이 전략 보기
💡 전략 힌트: \(f(x) = 2x^3 - 3x^2 - x + 1\). 정수점에서의 부호: \(f(-1) = -3, f(0) = 1, f(1) = -1, f(2) = 3, f(3) = 25\). 부호 변화: \(-1 \to 0, 0 \to 1, 1 \to 2\). 정답: ㄱ, ㄴ, ㄷ.
[도전-구간 분기 곱 함수의 연속 (서술형)]
13. 두 함수 \( f(x) = x^2 + ax + b \), \( g(x) = \begin{cases} x^2 & (|x| \leq 1) \\ 2 & (|x| \gt 1) \end{cases} \) 에 대하여 함수 \(f(x) \cdot g(x)\)가 모든 실수 \(x\)에서 연속일 때, 실수 \(a, b\)의 값을 풀이 과정과 함께 구하시오. (교과서 P.42 중단원 도전 13번)
🎯 서술형 풀이 전략 보기
💡 전략: ① \(g\)는 \(x = \pm 1\)에서 불연속, \(f\)는 모든 실수에서 연속. ② \((fg)\)가 \(x = 1\)에서 연속 — 좌극한 \(= f(1) \cdot 1\), 우극한 \(= f(1) \cdot 2\). 일치 조건 → \(f(1) = 0 \Rightarrow 1 + a + b = 0\). ③ \(x = -1\)에서 좌극한 \(= f(-1) \cdot 2\), 우극한 \(= f(-1) \cdot 1\) → \(f(-1) = 0 \Rightarrow 1 - a + b = 0\). ④ 두 식 연립 → \(a = 0, b = -1\).
서술 채점 포인트: 두 경계점 각각의 좌·우극한 식 명시, \(f(\pm 1) = 0\) 도출 근거, 연립 과정 1행 이상.
[도전-사잇값 정리로 두 그래프 교점 증명 (서술형)]
14. 모든 실수 \(x\)에서 연속인 함수 \(f(x)\)에 대하여 \( f(-3) f(-1) \lt 0,\ f(1) f(3) \lt 0 \) 이고 \(f(1) = -1\)이다. 방정식 \(f(x) = 0\)이 열린구간 \((-3, 3)\)에서 오직 두 개의 실근을 가질 때, 두 함수 \(y = f(x)\), \(y = -2x^3\)의 그래프는 \(-1 \lt x \lt 1\)에서 적어도 하나의 교점을 가짐을 풀이 과정과 함께 보이시오. (교과서 P.42 중단원 도전 14번)
🎯 서술형 풀이 전략 보기
💡 전략: ① 조건에서 \(f(-3)f(-1) \lt 0\) → 사잇값 정리로 \((-3, -1)\)에 \(f = 0\)의 실근 1개 존재. \(f(1) = -1 \lt 0\)이고 \(f(1)f(3) \lt 0\) → \(f(3) \gt 0\), \((1, 3)\)에 실근 1개. 따라서 \((-3, 3)\)에서 2개의 실근은 이 두 구간에서 각각 발생. 즉 \(-1 \lt x \lt 1\)에서는 \(f(x) = 0\)이 실근을 갖지 않음. ② 또한 \(f(-1), f(1)\)은 같은 부호여야 하며 (만약 다르면 \((-1, 1)\)에 추가 실근 발생) \(f(1) = -1 \lt 0\) → \(f(-1) \lt 0\). ③ 보조함수 \(h(x) = f(x) - (-2x^3) = f(x) + 2x^3\)을 \([-1, 1]\)에서 정의. \(h\)는 연속. \(h(-1) = f(-1) + 2(-1)^3 = f(-1) - 2 \lt -2 \lt 0\). \(h(1) = f(1) + 2 = -1 + 2 = 1 \gt 0\). ④ 사잇값 정리에 의해 \(h(c) = 0\)인 \(c \in (-1, 1)\)이 적어도 하나 존재. 즉 \(f(c) = -2c^3\) → 두 그래프의 교점.
서술 채점 포인트: 두 실근 구간 식별 + \(f(-1)\) 부호 결정 + 보조함수 \(h\) 정의 + 양 끝 부호 + 사잇값 정리 인용.
