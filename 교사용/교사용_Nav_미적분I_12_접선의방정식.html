12 II-2a 도함수의 활용 1 접선의 방정식 — 점-기울기 공식의 미분 확장
한 줄 핵심: "공통수학 2의 \(y - y_1 = m(x - x_1)\)에 \((x_1, y_1) = (a, f(a))\), \(m = f'(a)\)를 그대로 꽂으면 끝" — 접선식 \(y - f(a) = f'(a)(x - a)\)는 새 공식이 아니라 직선식의 미분 버전이다.
성취기준
[12미적-02-04] 접선의 방정식을 구하고, 곡선 위의 점에서/곡선 밖의 점에서의 접선을 분기 풀이한다.
선수학습
II-1 미분계수 \(f'(a)\)의 기하적 의미(접선의 기울기), 공통수학 2 점-기울기 직선식, 인수분해.
학습 목표

C 생각 열기 운영 가이드

발문: 곡선 \(y = -x^2 + 3\) 위의 \((1, 2)\)에서의 접선을 한 줄로 쓰려면 어떤 정보가 필요할까? 미분계수 \(f'(1) = -2\)를 어떻게 활용할까?

예상 반응 ①"기울기는 \(-2\)인데 직선식이 뭐였더라…"
피드백: 공통수학 2의 "점-기울기 공식 \(y - y_1 = m(x - x_1)\)"을 직접 떠올리게 유도. 외운 공식이 아니라 직선의 두 정보(점·기울기)임을 강조.
예상 반응 ②"\(y = -2x + b\)에 점을 대입하면 \(b = 4\)니까 \(y = -2x + 4\)다."
피드백: 정답. 다만 점-기울기 공식 \(y - 2 = -2(x - 1) \Rightarrow y = -2x + 4\)가 한 줄에 끝남을 보이고, 미적분에서는 이 표준형을 쓰자고 약속.
예상 반응 ③"\(f'(1) = -2\)는 그냥 미분한 값인데 그게 왜 기울기야?"
피드백: II-1 미분계수의 기하적 의미("접선의 기울기")를 5초 복습. 단원 II-1 \(\to\) II-2의 연결 고리임을 명시.

1학기 연결 & II-1 연결: 1학기에서 \(\dfrac{f(x) - f(a)}{x - a}\)의 극한이 접선의 기울기였고, II-1 차시 9·10에서 그 극한을 \(f'(a)\)로 정의했다. 본 차시는 \(f'(a)\)를 "값"이 아니라 "직선식의 기울기"로 적극 활용하는 첫 차시.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE FORMULA
\( y - f(a) = f'(a)(x - a) \)    (접점 \((a, f(a))\), 기울기 \(f'(a)\))
📌 판서 시연 5단계 (시나리오 ③ 곡선 밖 점):
  1. 주어진 점 \((p, q)\)를 곡선식에 대입 → \(q = f(p)\)인지 검증. 같으면 시나리오 ①, 다르면 시나리오 ③.
  2. 곡선 밖이면 접점을 \((t, f(t))\)로 매개변수화. \(f'(t)\)가 기울기.
  3. 접선식 \(y - f(t) = f'(t)(x - t)\) 작성.
  4. 주어진 점 \((p, q)\)를 접선식에 대입 → \(t\)에 대한 방정식 풀이.
  5. 각 \(t\) 값별로 접선식 작성 (이차함수 → 최대 2개, 삼차함수 → 최대 3개).
오개념 ①: "주어진 점이 곡선 위인지 무조건 검증 없이 \(f'(p)\)를 기울기로 쓴다." → L3 7번의 함정. \(f(p) \neq q\)이면 곡선 밖. 매개변수 \(t\) 필수.
오개념 ②: "이차함수에 곡선 밖 점은 접선이 1개." → 일반적으로 2개 (특수 경우만 1개). 삼차함수는 최대 3개.
오개념 ③: "공통 접선은 점만 같으면 된다." → 점 \(\cdot\) 기울기 두 조건이 동시에 만족되어야 한다.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11접점 주어진 접선 — 직접 적용
(1) \(y = -2x^2 + 3x - 1\), 점 \((2, -3)\)    (2) \(y = x^3 + 2x^2 - 1\), 점 \((-1, 0)\)
(1) \(f'(x) = -4x + 3\), \(f'(2) = -5\). 접선: \(y + 3 = -5(x - 2) \Rightarrow y = -5x + 7\). (2) \(f'(x) = 3x^2 + 4x\), \(f'(-1) = 3 - 4 = -1\). 접선: \(y - 0 = -1(x + 1) \Rightarrow y = -x - 1\).
정답: (1) \(y = -5x + 7\)   (2) \(y = -x - 1\)
L12기울기 주어진 접선 — 매개변수 \(t\)
기울기 \(1\)인 접선. (1) \(y = 2x^2 - 7x + 3\)   (2) \(y = x^3 - 2x + 1\)
(1) \(f'(x) = 4x - 7\), \(4t - 7 = 1 \Rightarrow t = 2\). 접점 \((2, -3)\), 접선 \(y = x - 5\). (2) \(f'(x) = 3x^2 - 2\), \(3t^2 - 2 = 1 \Rightarrow t = \pm 1\). \(t=1\): 접점 \((1, 0)\), 접선 \(y = x - 1\). \(t=-1\): 접점 \((-1, 2)\), 접선 \(y = x + 3\).
정답: (1) \(y = x - 5\)   (2) \(y = x - 1\) 또는 \(y = x + 3\)
L13곡선 위 점에서의 접선 단순 계산
\(y = x^3 - 3x^2 + 4\), 점 \((1, 2)\)에서 접선.
검증: \(f(1) = 1 - 3 + 4 = 2\) ✓ (곡선 위). \(f'(x) = 3x^2 - 6x\), \(f'(1) = -3\). 접선: \(y - 2 = -3(x - 1) \Rightarrow y = -3x + 5\).
정답: \(y = -3x + 5\)
L24곡선 밖의 점에서의 접선 — 매개변수 \(t\) 정착
(1) \(y = x^2 - 4x\), 점 \((-2, 3)\)    (2) \(y = -x^3 + 2x + 7\), 점 \((0, 5)\)
(1) 검증: \(f(-2) = 4 + 8 = 12 \neq 3\) → 곡선 밖. \(f'(x) = 2x - 4\), 접점 \((t, t^2 - 4t)\). 접선 \(y - (t^2 - 4t) = (2t - 4)(x - t)\)에 \((-2, 3)\) 대입: \(3 - t^2 + 4t = (2t - 4)(-2 - t) = -4t - 2t^2 + 8 + 4t = -2t^2 + 8\). \(\Rightarrow t^2 + 4t - 5 = 0 \Rightarrow (t+5)(t-1) = 0 \Rightarrow t = 1, -5\). \(t=1\): 기울기 \(-2\), 접점 \((1, -3)\), 접선 \(y = -2x - 1\). \(t=-5\): 기울기 \(-14\), 접점 \((-5, 45)\), 접선 \(y = -14x - 25\). (2) 검증: \(f(0) = 7 \neq 5\) → 곡선 밖. \(f'(x) = -3x^2 + 2\), 접점 \((t, -t^3 + 2t + 7)\). 접선에 \((0, 5)\) 대입: \(5 - (-t^3 + 2t + 7) = (-3t^2 + 2)(-t) = 3t^3 - 2t\). \(\Rightarrow t^3 - 2t - 2 = 3t^3 - 2t \Rightarrow -2 = 2t^3 \Rightarrow t = -1\). 기울기 \(-3 + 2 = -1\), 접점 \((-1, 1 - 2 + 7) = (-1, 6)\), 접선 \(y = -x + 5\).
정답: (1) \(y = -2x - 1\), \(y = -14x - 25\) (2개)   (2) \(y = -x + 5\) (1개)
판서 포인트: (1)에서 접선이 2개 나오는 것이 일반적. (2)에서 \(t = -1\) 한 개만 나오는 이유는 \(t^3 = -1\)이 실근 1개만 가지기 때문 — 삼차 곡선이지만 외부점 위치에 따라 접선 개수가 달라짐을 보임.
L25두 곡선의 공통 접선
\(f(x) = x^3 + ax + b\), \(g(x) = cx^2 + 1\)이 \((1, 2)\)에서 공통 접선. \(a, b, c\) 구하라.
조건 ① 점 공통: \(f(1) = 1 + a + b = 2\), \(g(1) = c + 1 = 2 \Rightarrow c = 1\). 조건 ② 기울기 일치: \(f'(1) = 3 + a\), \(g'(1) = 2c = 2\). \(3 + a = 2 \Rightarrow a = -1\). ①에서 \(b = 2 - 1 + 1 = 2\).
정답: \(a = -1\), \(b = 2\), \(c = 1\).
L26섬·육지 다리 최소 길이 (수리 언어 소양)
섬 곡선 \(y = x^2 - 4x + 3\), 육지 직선 \(y = 2x - 11\). 다리 최소 길이?
최소 거리는 곡선의 접선이 직선과 평행할 때. \(f'(x) = 2x - 4 = 2 \Rightarrow x = 3\), 접점 \((3, 0)\). 직선 \(2x - y - 11 = 0\)에서 점 \((3, 0)\)까지의 거리: \(\dfrac{|2 \cdot 3 - 0 - 11|}{\sqrt{4 + 1}} = \dfrac{5}{\sqrt{5}} = \sqrt{5}\).
정답: \(\sqrt{5}\).
L37🚨 오답률 1위 — 곡선 위/밖 검증 필수
점 \((0, 2)\)에서 \(y = -x^3 + x + 4\)에 그은 접선과 \(x\)축, \(y\)축으로 둘러싸인 부분의 넓이.
STEP 1 검증: \(f(0) = 4 \neq 2\) → 곡선 밖. 시나리오 ③. STEP 2: 접점 \((t, -t^3 + t + 4)\), \(f'(t) = -3t^2 + 1\). 접선식 \(y - (-t^3 + t + 4) = (-3t^2 + 1)(x - t)\). STEP 3: \((0, 2)\) 대입. \(2 - (-t^3 + t + 4) = (-3t^2 + 1)(-t) = 3t^3 - t\). \(\Rightarrow t^3 - t - 2 = 3t^3 - t \Rightarrow -2 = 2t^3 \Rightarrow t^3 = -1 \Rightarrow t = -1\). STEP 4: 접점 \((-1, 1 - 1 + 4) = (-1, 4)\), 기울기 \(-3 + 1 = -2\). 접선 \(y - 4 = -2(x + 1) \Rightarrow y = -2x + 2\). \((0, 2)\) 검증 ✓. STEP 5: 접선과 축의 교점 \((1, 0)\), \((0, 2)\). 직각삼각형 넓이 \(= \tfrac{1}{2} \cdot 1 \cdot 2 = 1\).
정답: \(1\).
발표 유도: "왜 \(f'(0)\)을 바로 못 쓰지?" 학생에게 직접 발표. "곡선 위가 아니라서, \(f'(0)\)은 \((0, 4)\)에서의 기울기일 뿐 \((0, 2)\)에서의 정보가 아니다"가 정답.
FC8형성평가 — 접선이 곡선과 만나는 다른 한 점
\(y = x^3 - 3x^2 + 2x + 1\) 위의 \((0, 1)\)에서의 접선이 곡선과 만나는 다른 한 점.
\(f'(x) = 3x^2 - 6x + 2\), \(f'(0) = 2\). 접선 \(y = 2x + 1\). 교점: \(x^3 - 3x^2 + 2x + 1 = 2x + 1 \Rightarrow x^3 - 3x^2 = 0 \Rightarrow x^2(x - 3) = 0\). \(x = 0\)은 접점(중근), \(x = 3\)이 다른 한 점. \(y = 2 \cdot 3 + 1 = 7\).
정답: \((3, 7)\). 핵심: 접점은 인수분해에서 중근으로 나타남 (\(x^2\) 인수).
CH1D1⭐ 수능특강 도함수의 활용 1 유제 1 [26009-0075]
\(y = -2x^4 + 3x^2 + a\) 위의 \((1, a+1)\)에서의 접선이 \((0, -3)\)을 지날 때 \(a\)는? ① \(-6\) ② \(-7\) ③ \(-8\) ④ \(-9\) ⑤ \(-10\).
STEP 1: \(f'(x) = -8x^3 + 6x\), \(f'(1) = -2\). (검증: \(f(1) = -2 + 3 + a = a + 1\) ✓ — 곡선 위.) STEP 2: 접선 \(y - (a+1) = -2(x - 1) \Rightarrow y = -2x + a + 3\). STEP 3: \((0, -3)\) 대입. \(-3 = a + 3 \Rightarrow a = -6\).
정답: ① \(-6\). 핵심: 곡선 위의 점에서의 접선이 외부의 한 점을 지나는 표준 유형. 사차함수도 다항함수이므로 미분 공식 그대로 적용.
CH2D2⭐ 1998학년도 수능 나형 9번 [3점]
포물선 \(y = -\tfrac{1}{4}x^2\)의 \((2, -1)\)에서의 접선과 \(x\)축, \(y\)축으로 둘러싸인 삼각형의 넓이는? ① \(\tfrac{1}{2}\) ② \(\tfrac{1}{4}\) ③ \(\tfrac{3}{4}\) ④ \(\tfrac{3}{2}\) ⑤ \(\tfrac{5}{4}\).
STEP 1: \(f'(x) = -\tfrac{1}{2}x\), \(f'(2) = -1\). STEP 2: 접선 \(y - (-1) = -1(x - 2) \Rightarrow y = -x + 1\). STEP 3: \(x\)절편 \((1, 0)\), \(y\)절편 \((0, 1)\). 둘 다 양 → 제1사분면 직각삼각형. STEP 4: 넓이 \(= \tfrac{1}{2} \cdot 1 \cdot 1 = \tfrac{1}{2}\).
정답: ① \(\tfrac{1}{2}\). 핵심: "접선이 좌표축으로 둘러싸는 넓이"는 평가원·수능 단골. 4단 분해(도함수→접선→두 절편→넓이) 정확히.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (점-기울기 공식의 일반화) "공통수학 2의 \(y - y_1 = m(x - x_1)\)에 \((x_1, y_1) = (a, f(a))\), \(m = f'(a)\)를 그대로 꽂으면 접선식이라는 게 보였다. 새 공식이 아니라 직선의 두 정보를 곡선의 미분 정보로 바꿔 끼우는 것뿐. 외울 게 줄었다."
유형 2: 문제해결력 (시나리오 분기 점검) "7번에서 \(x = 0\)이 곡선 위인 줄 알고 \(f'(0)\)을 바로 썼다가 틀렸다. \(f(0) = 4 \neq 2\)이니 곡선 밖이라 매개변수 \(t\)로 분기해야 했다. 이제는 푸는 시작 전에 '점이 곡선 위인지 검증' 한 줄 먼저 적는다."
유형 3: 발전 가능성 (접선 → 공통접선 → 평균값 정리) "공통 접선 조건이 '점 공통 + 기울기 일치' 두 개라는 게 정리되니, 다음 차시 평균값 정리에서 '직선 AB에 평행한 접선이 존재'한다는 명제도 같은 발상 같다. 도함수 활용 단원이 '접선의 기울기' 한 가지로 이어진다."
