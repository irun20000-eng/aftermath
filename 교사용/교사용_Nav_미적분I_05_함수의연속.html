발문: 식물의 키(시간 따라 부드럽게)와 매월 받는 용돈(1일에 불쑥)은 그래프로 그렸을 때 어떻게 다른가?

예상 반응 ①"용돈은 계단처럼 생겼고 식물은 곡선이다."
피드백: 좋은 시각화. 계단의 "꺾이는 순간"이 곧 좌·우극한이 다른 지점임을 4차시 \(\frac{|x|}{x}\) 그래프로 연결.
예상 반응 ②"식물은 끊어지지 않고 용돈은 갑자기 뛴다."
피드백: 정답에 가깝다 — "끊어지지 않음"을 수학적으로 어떻게 표현할지 묻고 \(\lim_{x \to a} f(x) = f(a)\)로 유도.
예상 반응 ③"용돈도 누적하면 부드러워 보일 수 있다."
피드백: 측정 단위에 따라 다르게 보이는 통찰. 그래도 1일 0시 0분 0초의 "그 순간"은 점프임을 강조 — 수학적 정의는 모든 점에서 일관되어야 함.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE DEFINITION
\( f(x) \)가 \( x = a \)에서 연속 \(\Longleftrightarrow\) ① \( f(a) \) 존재  &  ② \( \displaystyle \lim_{x \to a} f(x) \) 존재  &  ③ \( \displaystyle \lim_{x \to a} f(x) = f(a) \)
📌 판서 시연 5단계:
  1. 다항함수 \(f(x) = x^2 - 1\)을 그리고 \(x = 0\)에서 \(f(0) = -1 = \lim_{x \to 0} f(x)\) → 연속의 모범 예.
  2. 제거 가능 불연속: \(g(x) = \frac{x^2 - 1}{x - 1}\) (단, \(g(1) = 5\))를 그려 ②는 성립하지만 ③이 깨짐을 색깔로 표시.
  3. 점프 불연속: \(\frac{|x|}{x}\)에서 좌극한 \(-1\), 우극한 \(1\) → ② 깨짐.
  4. 무한 불연속: \(\frac{1}{x-2}\)에서 \(x \to 2^\pm\)로 \(\pm\infty\) 발산 → 점근선.
  5. 구간 분기 함수: \(f(x) = 3x^2-1\,(x\geq a),\,2-5x\,(x\lt a)\)를 그려 "경계점에서만 점검" 원리 시연.
오개념 ①: "함숫값만 있으면 연속이다." → 극한 존재(②)와 일치(③)가 별도로 필요.
오개념 ②: "극한값과 함숫값이 같으면 그 점에서만 연속이라고 한다." → 극한 자체가 존재(③ ⊃ ②)해야 비교 가능. 좌·우극한 불일치면 ③은 가짓수조차 없음.
오개념 ③: "분수함수는 항상 불연속이다." → 분모가 \(0\)이 되는 점만 불연속. 그 외에서는 연속.
오개념 ④: "닫힌구간 끝점에서 양쪽 극한이 필요하다." → 끝점 바깥은 정의되지 않으므로 한쪽 극한만 일치하면 충분.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L113조건 직접 점검
(1) \(f(x) = x^2 - 1\) — \(x=0\) 연속?  (2) \(g(x) = |x|\,(x\neq 0)\), \(g(0)=2\) — \(x=0\) 연속?
(1) 다항함수 → 모든 실수 연속. \(f(0) = -1\), \(\lim_{x\to 0} f(x) = -1\) → 일치 → 연속. (2) 좌극한 = 우극한 = \(\lim_{x\to 0}|x| = 0\). 그러나 \(g(0) = 2 \neq 0\) → ③ 깨짐 → 제거 가능 불연속.
정답: (1) 연속 (2) 불연속(제거 가능형)
L12연속인 구간 (정의역 자체가 연속구간)
(1) \(x^2 - 4x + 2\)  (2) \(\frac{3}{x+2}\)  (3) \(\sqrt{1-x}\)  (4) \(6\)
(1) 다항 → \((-\infty, \infty)\). (4) 상수 → \((-\infty, \infty)\). (2) 유리 → 분모 \(\neq 0\) → \(x \neq -2\) → \((-\infty, -2) \cup (-2, \infty)\). (3) 근호 안 \(\geq 0\) → \(x \leq 1\). 끝점 \(x=1\) 우극한 없음, 좌극한 = \(f(1) = 0\) → \((-\infty, 1]\).
정답: (1) \((-\infty, \infty)\)  (2) \((-\infty,-2)\cup(-2,\infty)\)  (3) \((-\infty, 1]\)  (4) \((-\infty, \infty)\)
L23한 점 연속 미정계수 \(a, b\)
\(f(x) = \frac{2x+a}{x-2}\,(x\neq 2)\), \(f(2)=b\). 모든 실수 연속.
모든 실수 연속 ⇒ \(x=2\)에서 연속 ⇒ \(\lim_{x\to 2}\frac{2x+a}{x-2} = b\). 분모 \(\to 0\), 극한 실수 ⇒ 분자 \(\to 0\) ⇒ \(4 + a = 0\) ⇒ \(a = -4\). \(\frac{2x-4}{x-2} = \frac{2(x-2)}{x-2} = 2\). \(\lim = 2 = b\).
정답: \(a = -4,\ b = 2\)
L24구간 분기 연속 — 모든 \(a\)의 곱
\(f(x) = 3x^2 - 1\,(x \geq a)\), \(2 - 5x\,(x \lt a)\). 모든 실수 연속.
두 다항식은 각 구간 연속 → 경계 \(x=a\)에서만 점검. 좌극한 \(= 2 - 5a\), 우극한·함숫값 \(= 3a^2 - 1\). 일치 조건 \(3a^2 + 5a - 3 = 0\). 비에타: 두 근의 곱 \(= \frac{-3}{3} = -1\).
정답: \(-1\). (실제 근을 구할 필요 없이 계수만으로 곱 도출.)
L35🚨 오답률 1위 — 극한 존재 \(\neq\) 연속
\(f(x) = \frac{x^2 - x}{|x-1|}\,(x \neq 1)\), \(f(1) = 1\). \(f(1)\), 좌·우극한, 극한 존재 여부, 연속/불연속.
(1) \(f(1) = 1\) (정의). (2) \(x \lt 1\): \(|x-1| = -(x-1)\) → \(\frac{x(x-1)}{-(x-1)} = -x\) → 좌극한 \(= -1\). (3) \(x \gt 1\): \(|x-1| = x-1\) → \(\frac{x(x-1)}{x-1} = x\) → 우극한 \(= 1\). (4) 좌(\(-1\)) \(\neq\) 우(\(1\)) → \(\lim_{x\to 1} f(x)\) 존재하지 않음. ② 깨짐 → 불연속(점프형). 함정 풀이: 함숫값 \(f(1) = 1\)이 우극한과 우연히 같다고 해서 연속이 아님! ②가 깨지면 ③은 비교조차 불가.
정답: \(f(1) = 1\), 좌 \(=-1\), 우 \(=1\), 극한 존재 X, 불연속.
발표 유도: "왜 \(f(1) = 1\)과 우극한이 같은데도 연속이 아닌가?"를 학생에게 직접 한 문장으로 말하게 한다 — "양쪽이 모두 같지 않으면 극한 자체가 없기 때문." 02차시 \(\frac{|x|}{x}\)의 확장판임을 환기.
FC6형성평가 — 유리식 연속성 조사
\(f(x) = x^2+1\), \(g(x) = x^2 - 2x - 3\). (1) \(\frac{f}{g}\)  (2) \(\frac{g}{f}\) — 연속/불연속 판정 (불연속이면 \(x\) 모두).
\(f, g\) 모두 다항함수 → 분자·분모 모두 모든 실수에서 연속. 유리식의 연속성은 분모의 영점만 점검. (1) \(g(x) = (x-3)(x+1) = 0\) ⇒ \(x = 3, -1\)에서 불연속, 그 외 연속. (2) \(f(x) = x^2 + 1 \geq 1 \gt 0\) → 분모 \(= 0\)인 실수 없음 → 모든 실수 연속.
정답: (1) \(x = 3, -1\)에서 불연속   (2) 모든 실수에서 연속.
CH1D1⭐ 수능특강 수학Ⅱ 함수의연속 유제 2 — 두 점 동시 연속
\(f(x) = 3x + 4a\,(1 \lt x \lt b)\), \(x^2 + a\,(x \leq 1\ \text{또는}\ x \geq b)\). \(x=1, x=b\)에서 연속, \(b \gt 1\). \(a+b\)는? ① \(\tfrac13\) ② \(\tfrac23\) ③ \(1\) ④ \(\tfrac43\) ⑤ \(\tfrac53\)
STEP 1 \(x=1\) 연속: 좌극한 \(= 1+a\) (\(x\leq 1\) 식), 우극한 \(= 3+4a\) (\(1\lt x\lt b\) 식), \(f(1) = 1+a\). 일치: \(1+a = 3+4a\) ⇒ \(a = -\tfrac{2}{3}\). STEP 2 \(x=b\) 연속: 좌극한 \(= 3b + 4a = 3b - \tfrac{8}{3}\), 우극한·함숫값 \(= b^2 + a = b^2 - \tfrac{2}{3}\). 일치: \(b^2 - 3b + 2 = 0\) ⇒ \((b-1)(b-2) = 0\). STEP 3 \(b \gt 1\) ⇒ \(b = 2\). STEP 4 \(a+b = -\tfrac{2}{3} + 2 = \tfrac{4}{3}\).
정답: ④ \(\frac{4}{3}\). 핵심: 두 경계에서 각각 좌·우극한 식을 어느 구간 식으로 잡는지가 결정적.
CH2D2⭐ 2021년 10월 학평 5번 [3점] — 그래프 × 곱 연속
그래프 \(y=f(x)\) (images/calc_05_chal2_kichul.png)에서 \((x^2+ax+b)f(x)\)가 \(x=1\)에서 연속. \(a+b\)는?
STEP 1 그래프 읽기: \(\lim_{x\to 1^-} f(x) = 1\) (수평선 \(y=1\), \(-1\leq x \lt 1\)). \(\lim_{x\to 1^+} f(x) = 3\) (우하향 직선 ○에서 시작). \(f(1) = 1\) (●). \(f\)는 \(x=1\)에서 점프 불연속. STEP 2 \(g(x) = x^2 + ax + b\)는 다항 → 모든 점 연속. 곱 \((g \cdot f)\)가 \(x=1\)에서 연속이려면 좌극한 = 우극한: \(g(1) \cdot 1 = g(1) \cdot 3\). STEP 3 \(g(1)(1 - 3) = 0\) ⇒ \(g(1) = 0\). 즉 \(f\)의 좌·우극한이 다른 점에서 곱이 연속이려면 다항 인수가 그 점에서 영점이어야 함. STEP 4 \(g(1) = 1 + a + b = 0\) ⇒ \(a + b = -1\).
정답: ② \(-1\). 핵심: ●(함숫값)과 ○(극한 위치) 구분 + "불연속 흡수" 영점 전략.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 "연속이 \(\lim = f(a)\) 한 줄이지만 사실 ① 정의 ② 극한 존재 ③ 두 값 일치 세 조건을 한꺼번에 요구한다는 걸 알게 됐다. 5번 문제에서 좌극한 \(-1\), 우극한 \(1\), \(f(1)=1\)이 우극한과 같음에도 연속이 아닌 이유는 ②가 깨지면 ③ 비교가 불가능하기 때문이라는 점이 강렬했다."
유형 2: 결정적 실수와 극복 "3번 \(\frac{2x+a}{x-2} = b\)에서 \(a, b\) 둘을 한꺼번에 풀려다 실패. '분모 \(\to 0\) + 극한 실수 → 분자 \(\to 0\)' 순서로 \(a\)를 먼저 확정한 뒤 \(b\)를 구하는 루틴을 노트에 적어뒀다. 이 순서가 4차시 \(\frac{0}{0}\) 부정형 처리와 동일하다는 게 보였다."
유형 3: 발전 가능성 "두 연속함수의 합성 \(f(g(x))\)도 연속일까 궁금했다. 다항 \(g\) + 근호 \(f\)의 합성을 \(\sqrt{x^2-3}\)으로 보니 \(g\) 정의역의 일부에서만 \(f\)가 정의됨이 보였다. 다음 차시 사잇값 정리에서 합성·합성의 합으로 어디까지 끌고 갈 수 있을지 기대된다."

H 수업 후 메모 (반 별 기록)
