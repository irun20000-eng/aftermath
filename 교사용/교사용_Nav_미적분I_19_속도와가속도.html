19 II-2b 도함수 활용 2 속도와 가속도 — 도함수의 물리적 해석
한 줄 핵심: "위치 \(\to\) (미분) \(\to\) 속도 \(\to\) (미분) \(\to\) 가속도" 사슬로 미분이 물리량을 잇는 언어가 됨을 깨닫는다. 핵심 함정: 변위 \(\neq\) 운동 거리.
성취기준
[12미적I-02-11] 속도와 가속도에 대한 문제를 도함수를 이용하여 해결할 수 있다.
선수학습
차시 09(미분계수), 차시 13(평균값 정리), 차시 14·15(증감·극값), 일차·이차함수 미분.
학습 목표

C 생각 열기 운영 가이드

발문: 운동 방향이 "양 \(\to\) 음"으로 바뀌는 순간 속도는 어떤 값이어야 할까? 가속도가 \(0\)이라는 것의 물리적 의미는?

예상 반응 ①"방향 전환 시 속도는 \(0\)이다."
피드백: 정답에 매우 가까움. 단 \(v = 0\)만으로는 부족 — 부호 변화까지 필요(예: \(v(t) = t^2 - 2t + 1 = (t-1)^2\)은 \(t = 1\)에서 \(v = 0\)이지만 부호 안 바뀌어 방향 전환 아님). 차시 15 극값 규칙과 동일 구조.
예상 반응 ②"가속도 \(0\)이면 멈춘다."
피드백: 흔한 오답. 정정 — 가속도 \(0\) = 속도가 일정한 순간(등속). 멈춤은 \(v = 0\). 가속도와 속도를 혼동하지 말 것을 강조.
예상 반응 ③"가속도 \(0\)이면 속도가 최대/최소이다."
피드백: 정확한 직관 — \(a(t) = v'(t) = 0\)이고 부호 변화면 \(v\)의 극값. Level 2 문제 6에서 직접 확인할 수 있음을 예고.

차시 14·15 연결: "도함수 부호 변화 = 함수 증감"이 본 차시에서 "속도 부호 변화 = 운동 방향 전환"으로 자연 확장. 동일한 수학 구조의 물리적 재해석.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE CHAIN
\( x = f(t) \xrightarrow{\,\dfrac{d}{dt}\,} v(t) = f'(t) \xrightarrow{\,\dfrac{d}{dt}\,} a(t) = v'(t) = f''(t) \)
📌 판서 시연 — 예제 \(x = -t^3 + 6t^2\) (\(t \geq 0\)):
  1. 속도 \(v(t) = -3t^2 + 12t = -3t(t - 4)\). 가속도 \(a(t) = -6t + 12\).
  2. \(t = 1\)에서 \(v(1) = 9 \gt 0\) (양 방향 운동), \(a(1) = 6 \gt 0\) (가속 중).
  3. 방향 전환 시각: \(v(t) = 0 \Rightarrow t = 0, 4\). \(t \gt 0\)이므로 \(t = 4\) 유일. 부호 \((+, -)\) 변화 확인 → 방향 전환.
  4. \(x(4) = -64 + 96 = 32\) → 운동의 최고 도달 지점(위치 함수의 극댓값과 일치).
  5. \(a(t) = 0 \Rightarrow t = 2\) → 이 순간 속도 \(v(2) = -12 + 24 = 12\)가 속도 함수의 극댓값(최대 속도 순간).
오개념 ①: "위치 변화량 = 운동 거리" → 1순위 함정 (Level 3 핵심). 방향 전환이 있으면 둘이 다름. 변위 = \(x(\text{끝}) - x(\text{시작})\), 운동 거리 = 각 구간의 변위 절댓값 합.
오개념 ②: "\(v(t) = 0\) ⇔ 방향 전환" → 부호 변화 필수. 예: \(v = (t - 1)^2\)은 \(t = 1\)에서 \(v = 0\)이지만 부호 동일 → 방향 그대로.
오개념 ③: "가속도 = 속도의 크기" → 가속도는 속도의 변화율. 등속운동(가속도 \(0\))에서도 속도는 \(0\)이 아닐 수 있다.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11속도·가속도 + 방향 전환
\(x = t^3 - t^2 - 8t\)에서 (1) \(t = 3\)일 때 속도·가속도 (2) 운동 방향 전환 시각.
\(v(t) = 3t^2 - 2t - 8 = (3t + 4)(t - 2)\), \(a(t) = 6t - 2\). (1) \(v(3) = 27 - 6 - 8 = 13\), \(a(3) = 18 - 2 = 16\). (2) \(v(t) = 0 \Rightarrow t = -4/3\) (제외) 또는 \(t = 2\). 부호 검증: \(v(1) = -7 \lt 0\), \(v(3) = 13 \gt 0\) → 부호 \(- \to +\) 변화 → \(t = 2\)에서 방향 전환.
정답: (1) 속도 \(13\), 가속도 \(16\)   (2) \(t = 2\) (음 → 양)
L12자유낙하 — 특정 높이의 속도
\(h = -5t^2 + 3000\). 높이 \(2500\) m에서의 속도.
\(-5t^2 + 3000 = 2500 \Rightarrow t^2 = 100 \Rightarrow t = 10\) (양수). \(v(t) = -10t\). \(v(10) = -100\) m/s.
정답: \(-100\) m/s (속력 \(100\) m/s, 음 부호는 아래 방향)
L13수직선 운동 — 속도·가속도 직접
\(x = 2t^3 - 5t\)에서 \(t = 1\) 속도·가속도.
\(v(t) = 6t^2 - 5\), \(a(t) = 12t\). \(v(1) = 1\), \(a(1) = 12\).
정답: 속도 \(1\), 가속도 \(12\)
L24두 점의 속도 일치 — 중근의 의미
\(x_P = \dfrac{2}{3}t^3 + 18t - 20\), \(x_Q = 6t^2 - 7\). 속도 일치 시각의 두 점 거리.
\(v_P = 2t^2 + 18\), \(v_Q = 12t\). \(2t^2 + 18 = 12t \Rightarrow t^2 - 6t + 9 = (t - 3)^2 = 0 \Rightarrow t = 3\) (중근). \(x_P(3) = 18 + 54 - 20 = 52\), \(x_Q(3) = 54 - 7 = 47\). 거리 \(|52 - 47| = 5\).
정답: 거리 \(= 5\)
판서 포인트: 중근 \((t - 3)^2 = 0\)의 물리적 의미 — 두 점의 속도가 일치하는 순간이 딱 한 번만 발생(접함). 두 속도 그래프가 \(t = 3\)에서 접한다고 시각화.
L25위치 그래프 해석 — 학생 대화
위치 함수가 이차함수일 때 학생 A("꼭짓점에서 \(v = 0\)"), B("평균속도 = 두 끝 연결 직선 기울기"), C("가속도는 항상 같다") 중 옳은 학생.
이차함수 \(x(t) = at^2 + bt + c\)이면 \(v(t) = 2at + b\) (일차), \(a(t) = 2a\) (상수). A: 꼭짓점은 위치의 극값 → \(v = 0\) ✓. B: 평균변화율 = 시컨트 기울기 ✓. C: \(a(t) = 2a\) 상수 ✓.
정답: A, B, C 모두 옳음.
발표 유도: 각 학생의 주장을 다른 학생이 검증하게 한다. 특히 C에 대해 "이차함수의 2계 도함수는 상수"라는 일반 원리로 설명하게 함.
L26가속도 0 시각의 물리적 의미
\(x = t^3 - 6t^2 + 9t + 1\)에서 (1) 모든 방향 전환 시각 (2) 가속도 \(0\)인 시각 + 그때 속도.
\(v(t) = 3(t - 1)(t - 3)\), \(a(t) = 6(t - 2)\). (1) \(v = 0 \Rightarrow t = 1, 3\). 부호 \((+, -, +)\) → 두 시각 모두 방향 전환. (2) \(a(t) = 0 \Rightarrow t = 2\). \(v(2) = 12 - 24 + 9 = -3\). 속도가 음수 중 절댓값 최대 = "감속 → 가속" 전환점(속도 함수의 극솟값).
정답: (1) \(t = 1, 3\) 두 번   (2) \(t = 2\), 속도 \(-3\). 핵심: \(a = 0\)은 속도의 극값 시각.
L37🚨 오답률 1위 — 변위 vs 운동 거리
\(x(t) = t^3 - 6t^2 + 9t\) (\(0 \leq t \leq 4\)). (1) 변위 (2) 운동 거리(왕복 포함). 두 값이 다른 이유.
STEP 1: \(v(t) = 3(t-1)(t-3)\). 방향 전환 시각 \(t = 1, 3\). 부호 \(+, -, +\). STEP 2: \(x(0) = 0, x(1) = 4, x(3) = 0, x(4) = 4\). STEP 3 — (1) 변위: \(x(4) - x(0) = 4\). STEP 4 — (2) 운동 거리: \(|x(1) - x(0)| + |x(3) - x(1)| + |x(4) - x(3)| = 4 + 4 + 4 = 12\). STEP 5: 변위는 단순 끝-시작 차이, 운동 거리는 왕복을 포함한 누적. \(t = 1\)에서 후진, \(t = 3\)에서 다시 전진 → 운동 거리가 더 큼.
정답: 변위 \(= 4\), 운동 거리 \(= 12\). 핵심: 변위 \(\neq\) 이동 거리(물리·수학 공통 원리).
발표 유도: 학생이 위치 그래프를 칠판에 직접 그리게 한다 — \(t = 0 \to 1 \to 3 \to 4\)에서 \(x\)값 \(0 \to 4 \to 0 \to 4\)의 등산-하산-재등산 패턴을 시각화. 차시 23 적분 공식(\(\int |v|\,dt\)) 예고.
FC8형성평가 — 자유낙하 종합
\(h = -5t^2 + 40t\). (1) 최고 높이·시각 (2) 지면 도착 속도 (3) 최고점~지면 운동 거리.
\(v(t) = -10t + 40\). (1) \(v = 0 \Rightarrow t = 4\), \(h(4) = -80 + 160 = 80\) m. (2) \(h(t) = 0 \Rightarrow -5t(t - 8) = 0 \Rightarrow t = 8\). \(v(8) = -80 + 40 = -40\) m/s. (3) \(t = 4 \to 8\)에서 \(v \lt 0\) 일관(단조 하강, 방향 전환 없음). 운동 거리 = \(|h(8) - h(4)| = |0 - 80| = 80\) m.
정답: (1) 최고 \(80\) m, \(t = 4\)초   (2) \(-40\) m/s   (3) \(80\) m
학생 함정: (3)에서 "쏘아 올린 직후부터 지면까지" 운동 거리를 묻는다면 \(0 \to 4\) 80 m 상승 + \(4 \to 8\) 80 m 하강 = 160 m. 본 문제는 "최고점부터"이므로 80 m. 시작 시각을 정확히 읽을 것.
CH1D1⭐ 수능특강 수학Ⅱ 도함수 활용 2 유제 7 — 두 번째 만남 시각
\(x_1 = t^3 - 4t^2\), \(x_2 = t^2 - 4t\). 두 번째 만남에서 속도 \(p, q\)의 차 \(p - q\).
STEP 1: \(x_1 = x_2 \Rightarrow t^3 - 5t^2 + 4t = t(t-1)(t-4) = 0\). 만남 시각 \(t = 0, 1, 4\). STEP 2: 출발이 \(t = 0\)이므로 첫 만남 \(t = 1\), 두 번째 만남 \(t = 4\). STEP 3: \(v_1(t) = 3t^2 - 8t\), \(v_2(t) = 2t - 4\). \(v_1(4) = 48 - 32 = 16\), \(v_2(4) = 4\). STEP 4: \(p - q = 16 - 4 = 12\).
정답: \(12\). 핵심: 동시 출발 \(\Rightarrow t = 0\) 자동 일치 → 두 번째 만남은 그 이후 두 번째 근.
CH2D2⭐ 2025학년도 6월 모평 공통 11번 [4점] — 보기 ㄱㄴㄷ 종합
\(x = t^3 - t^2 - t + 1\). ㄱ. \(t = 1\)에서 위치 \(= 1\)   ㄴ. \(t = 1\)에서 속도 \(= 0\)   ㄷ. 방향 전환 시각의 가속도 \(= 4\). ① ㄱ ② ㄴ ③ ㄷ ④ ㄱ,ㄷ ⑤ ㄴ,ㄷ
\(v(t) = 3t^2 - 2t - 1 = (3t + 1)(t - 1)\), \(a(t) = 6t - 2\). ㄱ: \(x(1) = 1 - 1 - 1 + 1 = 0 \neq 1\) → 거짓. ㄴ: \(v(1) = 4 \cdot 0 = 0\) → 참. ㄷ: \(t \geq 0\) 범위의 \(v\) 영점은 \(t = 1\) (\(t = -1/3\) 제외). 부호 \(- \to +\)로 변화 → 방향 전환 시각 \(t = 1\). \(a(1) = 6 - 2 = 4\) → 참.
정답: ⑤ ㄴ, ㄷ. 핵심: ㄱ의 "위치"와 ㄴ의 "속도"를 혼동 금지. ㄷ에서 "\(v = 0\) AND 부호 변화"라는 두 조건 동시 확인.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (도함수 사슬) "위치 → 속도 → 가속도가 미분의 2단계 사슬이라는 게 핵심이었다. 차시 09에서 배운 미분계수가 결국 '순간속도'와 같은 식이라는 게 연결되니, 수학 공식이 물리 현실을 그대로 묘사한다는 게 실감 났다. \(v = f'(t), a = f''(t)\)라는 두 식이 단원 전체를 압축한다."
유형 2: 문제해결력 (방향 전환과 변위·운동 거리) "방향 전환 시각을 '\(v = 0\)'만으로 잡았다가 부호 변화 없는 케이스에서 틀린 적이 있다. '\(v = 0\) AND 부호 변화'라는 두 조건 체크리스트로 정리하니 실수가 사라졌다. 또 Level 3에서 변위 \(\neq\) 운동 거리라는 함정을 직접 체험하니 '시작-끝 차이'가 아닌 '왕복 포함 누적'을 자동으로 떠올리게 됐다."
유형 3: 발전 가능성 (적분의 활용 예고) "운동 거리를 손으로 구간 나눠 합하는 게 번거로웠는데, 선생님 말씀대로 차시 23에서 \(\int |v(t)|\,dt\)라는 적분 공식으로 자동화된다니 미분과 적분이 서로 짝을 이룬다는 단원의 큰 그림이 보였다. 평균값 정리(차시 13)가 구간 단속의 수학적 근거라는 것도 일상과 수학이 연결되는 인상적인 사례였다."
