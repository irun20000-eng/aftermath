C 생각 열기 운영 가이드

발문: 차시 14의 \(f(x) = x^3 - 3x^2 + 1\) 증감표를 다시 보자. \(x = 0, 2\) 중 봉우리·골짜기는 어느 쪽? 도함수 부호와 어떻게 연결?

예상 반응 ①"\(x = 0\)에서 \(+\to-\)이니 봉우리, \(x = 2\)에서 \(-\to+\)이니 골짜기다."
피드백: 완벽한 직관. 이를 "극대·극소 판정 정리"로 정확히 명명하고, \(f(0) = 1\), \(f(2) = -3\) 함숫값을 극댓값·극솟값으로 매김.
예상 반응 ②"극댓값이 극솟값보다 항상 크지 않나?"
피드백: 직관에는 맞지만 일반적으로는 거짓. \(y = x + 1/x\)에서 극댓값 \(-2 \lt\) 극솟값 \(2\) 예시로 깨뜨림. "극은 지역(local) 개념"임을 강조.
예상 반응 ③"\(f'(x) = 0\)이면 다 극값 아닌가?"
피드백: 핵심 함정. \(f(x) = x^3\)에서 \(f'(0) = 0\)이지만 극값 아님 — 차시 14 L3에서 만난 \(x^3\) 반례가 본 차시의 출발점.

차시 14 → 15 연결: 차시 14에서 증감표 작성을 정착시켰다. 본 차시는 그 증감표의 임계점에서 "부호 변화 패턴"만 추가로 본다 — \(+\to-\) 극대, \(-\to+\) 극소, 같은 부호 유지면 극값 아님.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE CRITERIA
필요: \(f\)가 \(x = a\)에서 극값, 미분가능 \(\Rightarrow f'(a) = 0\) (페르마)     충분: \(f'(a) = 0\) AND 좌우 부호 변화 \(\Rightarrow\) 극값
📌 판서 시연 5단계 — 매개변수 결정 (\(f = x^3 + ax^2 + bx + 4\), \(x = 1\)에서 극댓값 \(8\)):
  1. 필요조건: \(f'(1) = 0 \Rightarrow 3 + 2a + b = 0\) …①
  2. 함숫값: \(f(1) = 1 + a + b + 4 = 8 \Rightarrow a + b = 3\) …②
  3. 연립: \(a = -6, b = 9\).
  4. 검증(필수): \(f'(x) = 3x^2 - 12x + 9 = 3(x - 1)(x - 3)\). \(x = 1\) 좌우 \(+\to-\) (극대 ✓), \(x = 3\) 좌우 \(-\to+\) (극소).
  5. 극솟값 \(f(3) = 27 - 54 + 27 + 4 = 4\).
오개념 ①: "\(f'(a) = 0 \Rightarrow\) 극값" — 거짓 (\(x^3\) 반례). 부호 변화 검증 필수. L3 7번 학생 A의 주장.
오개념 ②: "극값 \(\Rightarrow\) 미분가능" — 거짓 (\(|x|\) 반례). \(x = 0\)에서 극솟값이지만 미분 불가. L3 7번 학생 B의 주장.
오개념 ③: "삼차함수 극값 조건 \(D \geq 0\)" — 거짓. \(D = 0\)이면 도함수 중근 → 부호 변화 없음 → 극값 없음. 정확히는 \(D \gt 0\) (등호 제외). 도전 2 함정.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11극값 직접 구하기 (삼차·사차)
(1) \(f = -2x^3 + 3x^2 + 1\)    (2) \(f = x^4 - 8x^2 + 6\)
(1) \(f'(x) = -6x^2 + 6x = -6x(x - 1)\). \(f' = 0 \Rightarrow x = 0, 1\). 부호 \((-,+,-)\) → \(x = 0\) 극소 \(f(0) = 1\), \(x = 1\) 극대 \(f(1) = -2 + 3 + 1 = 2\). (2) \(f'(x) = 4x^3 - 16x = 4x(x - 2)(x + 2)\). \(f' = 0 \Rightarrow x = -2, 0, 2\). 부호 \((-,+,-,+)\) → \(x = \pm 2\) 극소 \(f(\pm 2) = 16 - 32 + 6 = -10\), \(x = 0\) 극대 \(f(0) = 6\).
정답: (1) 극솟값 \(1\), 극댓값 \(2\)   (2) 극솟값 \(-10\) (2번), 극댓값 \(6\). 판서 포인트: (2)의 사차함수는 극솟값이 두 \(x\)값에서 같음 — 좌·우 대칭 구조 자연스럽다.
L12매개변수 결정 — \(a, b\) + 극댓값
\(f(x) = -2x^3 + 3x^2 + ax + b\)가 \(x = -1\)에서 극솟값 \(-2\). \(a, b\)와 극댓값.
필요조건: \(f'(x) = -6x^2 + 6x + a\), \(f'(-1) = -6 - 6 + a = 0 \Rightarrow a = 12\). 함숫값: \(f(-1) = 2 + 3 - 12 + b = -2 \Rightarrow b = 5\). 검증: \(f'(x) = -6x^2 + 6x + 12 = -6(x - 2)(x + 1)\). \(x = -1\) 좌우 \(-\to+\) (극소 ✓), \(x = 2\) 좌우 \(+\to-\) (극대). 극댓값: \(f(2) = -16 + 12 + 24 + 5 = 25\).
정답: \(a = 12, b = 5\), 극댓값 \(25\). 핵심: 필요조건 + 함숫값 두 식으로 \(a, b\) 결정 후 반드시 부호 변화 검증 (STEP 4가 안 통과하면 답 무효).
L13극댓값 - 극솟값
\(f(x) = 2x^3 - 9x^2 + 12x - 3\)의 극댓값 - 극솟값.
\(f'(x) = 6x^2 - 18x + 12 = 6(x - 1)(x - 2)\). \(f' = 0 \Rightarrow x = 1, 2\). 부호 \((+,-,+)\). 극댓값 \(f(1) = 2 - 9 + 12 - 3 = 2\), 극솟값 \(f(2) = 16 - 36 + 24 - 3 = 1\). 차 \(= 1\).
정답: \(1\).
L24극값 갖지 않을 조건 (\(D \leq 0\), 등호 포함)
\(f(x) = x^3 + 3x^2 + ax + 1\)이 극값 없을 \(a\) 범위.
\(f'(x) = 3x^2 + 6x + a\). 극값 없음 \(\Leftrightarrow f'\) 부호 변화 없음 \(\Leftrightarrow D \leq 0\). \(D/4 = 9 - 3a \leq 0 \Rightarrow a \geq 3\).
정답: \(a \geq 3\). 등호 포함: \(a = 3\)일 때 \(f'(x) = 3(x + 1)^2 \geq 0\), 중근에서 부호 변화 없음 → 극값 없음 ✓.
L25극값 가질 조건 (\(D \gt 0\), 등호 제외)
\(f(x) = x^3 - 3ax^2 + 3x + 2\)가 극댓값·극솟값 모두 가질 \(a\) 범위.
\(f'(x) = 3x^2 - 6ax + 3\). 극값 모두 \(\Leftrightarrow f' = 0\) 서로 다른 두 실근 \(\Leftrightarrow D \gt 0\). \(D/4 = 9a^2 - 9 \gt 0 \Rightarrow a^2 \gt 1 \Rightarrow a \lt -1\) 또는 \(a \gt 1\).
정답: \(a \lt -1\) 또는 \(a \gt 1\). 4번과 비교: 4번 \(\leq 0\) (등호 포함) vs 5번 \(\gt 0\) (등호 제외) — 등호 포함 여부가 결정적.
L26도함수 그래프 3개 — 극값 판정 (수리 언어 소양)
그림에서 \(y = f'(x)\) 세 가지: (1) 아래로 볼록 포물선 \(x = a, b\)에서 \(x\)축 통과 (2) 아래로 볼록 포물선 \(x = c\)에서 \(x\)축 접 (3) 아래로 볼록 포물선 \(x\)축과 안 만남.
(1) \(f'\)의 부호 \((+,-,+)\) → \(x = a\) 극대, \(x = b\) 극소. 극값 2개. (2) \(f'(x) \geq 0\) 항상, 등호 \(x = c\)에서만. 부호 변화 없음 → \(x = c\)는 극값 아님. 극값 없음. (\(x^3\) 유형 정체점) (3) \(f'(x) \gt 0\) 항상. \(f\) 실수 전체 단조증가. 극값 없음.
정답: (1) 2개 / (2) 0개 / (3) 0개. 핵심: 도함수 그래프가 \(x\)축을 가로지르는가(부호 변화·극값) vs 접하는가(부호 유지·극값 X)를 구별.
L37🚨 오답률 1위 — 학생 A·B 두 주장
📢 A: "\(f'(a) = 0 \Rightarrow\) 극값."   📢 B: "극값 \(\Rightarrow\) 미분가능." 둘 다 참/거짓?
A 거짓: \(f(x) = x^3\), \(f'(0) = 0\)이지만 \(x = 0\) 좌우 부호 변화 없음 → 극값 아님. \(f\)는 전체에서 증가. B 거짓: \(f(x) = |x|\), \(x = 0\)에서 극솟값 \(0\). 그러나 좌미분 \(-1\), 우미분 \(+1\) → 미분 불가. 정확한 명제 4가지: ① 미분가능 + 극값 \(\Rightarrow f'(a) = 0\) (참) ② \(f'(a) = 0 \Rightarrow\) 극값 (거짓) ③ 극값 \(\Rightarrow\) 미분가능 (거짓) ④ \(f'(a) = 0\) AND 부호 변화 \(\Leftrightarrow\) 극값 (참).
정답: A·B 둘 다 거짓. 반례 \(x^3, |x|\). 핵심: 필요조건·충분조건 구분 — 충분조건이 되려면 부호 변화 추가. 차시 16 변곡점으로 직접 연결.
발표 유도: A·B 각각 손들기 후, \(x^3\)·\(|x|\) 두 그래프를 칠판에 직접 그리고 학생에게 "이 점은 극값인가?"를 정의 그대로 검증하게 한다.
FC8형성평가 — 극값의 합 = 0 (근과 계수의 관계)
\(f(x) = x^3 + 3x^2 + ax - 4\)의 극댓값 + 극솟값 = 0인 \(a\). (단 \(a \lt 3\))
STEP 1: \(f'(x) = 3x^2 + 6x + a\). 극값 존재 \(\Leftrightarrow D/4 = 9 - 3a \gt 0 \Rightarrow a \lt 3\) ✓. STEP 2: \(f' = 0\)의 두 실근 \(\alpha, \beta\) (\(\alpha \lt \beta\)). 근과 계수 \(\Rightarrow \alpha + \beta = -2\), \(\alpha\beta = a/3\). STEP 3: \(f(\alpha) + f(\beta) = (\alpha^3 + \beta^3) + 3(\alpha^2 + \beta^2) + a(\alpha + \beta) + 2 \cdot (-4)\). \(\alpha^2 + \beta^2 = 4 - 2a/3\), \(\alpha^3 + \beta^3 = (\alpha + \beta)^3 - 3\alpha\beta(\alpha + \beta) = -8 + 2a\). STEP 4: \((-8 + 2a) + 3(4 - 2a/3) + a(-2) - 8 = -8 + 2a + 12 - 2a - 2a - 8 = -4 - 2a = 0 \Rightarrow a = -2\). 검증: \(a = -2 \lt 3\) ✓. \(f'(x) = 3x^2 + 6x - 2\) 두 실근 ✓.
정답: \(a = -2\). 핵심: 극값의 합·곱은 \(f' = 0\)의 두 근 \(\alpha, \beta\)에 대한 대칭식 → 근과 계수의 관계로 직접 계산. 두 극값을 따로 구하지 않아도 됨.
CH1D1⭐ 수능특강 도함수의 활용 1 유제 6 [26009-0080]
\(g(x) = (x^2 - x)f(x)\), (가) \(f\) 일차함수 (나) \(g\)는 \(x = 3\)에서 극값 \(12\). \(g(-3)\)? ① 142 ② 144 ③ 146 ④ 148 ⑤ 150.
STEP 1: \(g(3) = 12\) \(\Rightarrow (9 - 3)f(3) = 6f(3) = 12 \Rightarrow f(3) = 2\). STEP 2: \(g'(3) = 0\) (필요조건). 곱의 미분: \(g'(x) = (2x - 1)f(x) + (x^2 - x)f'(x)\). \(g'(3) = 5 \cdot 2 + 6 f'(3) = 10 + 6f'(3) = 0 \Rightarrow f'(3) = -5/3\). STEP 3: \(f(x) = ax + b\), \(a = f'(3) = -5/3\). \(f(3) = -5 + b = 2 \Rightarrow b = 7\). \(f(x) = -\tfrac{5}{3}x + 7\). STEP 4: \(g(-3) = (9 + 3)f(-3) = 12 \cdot (5 + 7) = 12 \cdot 12 = 144\).
정답: ② 144. 핵심: 극값 조건은 "\(g'(a) = 0\) AND \(g(a) = (\text{극값})\)" 이중 정보. 한쪽으로 \(f'(3)\), 다른 쪽으로 \(f(3)\) 결정 → 일차함수 \(f\)의 두 계수 묶음.
CH2D2⭐ 2014학년도 4월 학평 가형 7번 [3점]
\(f(x) = x^3 + ax^2 + (a^2 - 4a)x + 3\)이 극값 갖도록 하는 정수 \(a\) 개수. ① 5 ② 6 ③ 7 ④ 8 ⑤ 9.
STEP 1: 극값 있음 \(\Leftrightarrow f' = 0\) 서로 다른 두 실근 \(\Leftrightarrow D \gt 0\). STEP 2: \(f'(x) = 3x^2 + 2ax + (a^2 - 4a)\). STEP 3: \(D/4 = a^2 - 3(a^2 - 4a) = -2a^2 + 12a = -2a(a - 6) \gt 0 \Rightarrow a(a - 6) \lt 0 \Rightarrow 0 \lt a \lt 6\). STEP 4: 정수 \(a \in \{1, 2, 3, 4, 5\}\) → 5개. STEP 5 검증: \(a = 0\) 또는 \(6\)에서 \(D = 0\), 도함수 중근 → 부호 변화 없음 → 극값 없음. 경계 제외 ✓.
정답: ① 5. 함정: 학생들이 \(D \geq 0\)로 풀어 \(a = 0, 6\)을 포함시키는 오류 빈발. "극값 있음" \(\Leftrightarrow D \gt 0\) (등호 제외) — Level 2 5번과 정확히 같은 패턴의 평가원 출제.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (필요조건 vs 충분조건) "'\(f'(a) = 0 \Rightarrow\) 극값'을 그냥 적었다가 \(x^3\) 반례에서 깨졌다. 필요조건('미분가능 + 극값 \(\Rightarrow f' = 0\)')과 충분조건('\(f' = 0\) + 부호 변화 \(\Rightarrow\) 극값')이 다르다는 걸 분명히 구분했다. 차시 14의 단조함수 함정과 같은 '필요·충분 비대칭' 문제임을 묶어 정리했다."
유형 2: 문제해결력 (극값 갖다 vs 안 갖다 등호 처리) "삼차 극값 조건을 처음엔 '\(D \geq 0\)'으로 적었다가 틀렸다. \(D = 0\)이면 도함수 중근 → 부호 변화 없음 → 극값 없음. 그래서 '극값 존재 \(\Leftrightarrow D \gt 0\)' (등호 제외)이 맞다. 등호 하나로 답이 갈리니, 부등호·등호는 마지막에 꼭 다시 점검."
유형 3: 발전 가능성 (극값 → 그래프 → 방정식) "극댓·극솟값이 단순한 봉우리·골짜기가 아니라 그래프 개형과 \(f(x) = 0\) 실근 개수를 결정하는 핵심 정보라는 걸 알게 됐다. 삼차 '극댓값 \(\gt 0\)이고 극솟값 \(\lt 0\)이면 서로 다른 세 실근' 규칙을 미리 예측해뒀더니 차시 16~18에서 그대로 쓰이는 게 보였다."
