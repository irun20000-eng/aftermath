23 III-1 부정적분과 정적분 정적분의 정의 — 부호 있는 면적과 함수 결정형
한 줄 핵심: \(\displaystyle\int_a^b f(x)\,dx\)는 "부호 있는 면적" — \(x\)축 위는 \(+\), 아래는 \(-\). 도형 넓이와 정적분은 다르다.
성취기준
[12미적-03-02] 정적분의 뜻을 알고 기하적 의미(부호 있는 면적)와 적분·미분 관계를 이해한다.
선수학습
22차시 부정적분 정의·공식. 삼각형·사다리꼴 넓이(중학). 합성함수 미분(단원 II).
학습 목표

C 생각 열기 운영 가이드

발문: "\(y = x\)와 \(x\)축, \(x = 2\)로 둘러싸인 삼각형 넓이 \(S_1\)과 \(y = -x\)로 둘러싸인 \(S_2\)의 값은? \(\int_0^2(-x)\,dx\)는 \(S_2\)와 같은가?"

예상 반응 ①"\(S_1 = S_2 = 2\), 정적분도 \(2\)."
피드백: 넓이는 맞지만 \(\int_0^2(-x)\,dx = -2\). 정적분은 "\(x\)축 아래에서 음수"라는 약속이 있기 때문. 도형 넓이 \(\neq\) 정적분이 오늘의 핵심.
예상 반응 ②"\(-x\)가 음수이니 정적분도 음수일 것."
피드백: 정확한 직관. 구체적으로 \(-\dfrac{1}{2} \cdot 2 \cdot 2 = -2\). 도형 넓이를 구하려면 \(\int |f|\,dx\)가 필요하다는 차이점으로 연결.
예상 반응 ③"삼각형 넓이 공식이 먼저 보인다."
피드백: 좋다. 오늘 L1은 바로 이 도형 계산으로 정적분을 구한다. 다음 차시(미적분 기본정리)에서는 도형을 그리지 않아도 된다.

22차시 연결: 부정적분은 함수족 \(F(x) + C\)를 반환. 오늘의 정적분 \(\int_a^b f\,dx\)는 한 개의 수를 반환. 두 개념이 다음 차시 기본정리에서 \(F(b) - F(a)\)로 합쳐진다.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE DEFINITION — 정적분과 적분·미분 관계
\( \displaystyle \int_a^b f(x)\,dx = S_+ - S_- \qquad \dfrac{d}{dx}\int_a^x f(t)\,dt = f(x) \)
📌 판서 시연 5단계:
  1. \(y = f(x)\)가 \([a, b]\)에서 양인 경우 그래프 → 도형 넓이 \(S\) → \(\int_a^b f\,dx = S\).
  2. \(y = f(x)\)가 음인 경우: 같은 도형이 \(x\)축 아래 → \(\int_a^b f\,dx = -S\). 부호만 다름.
  3. 혼합 경우: \(x\)축 위쪽 \(S_+\), 아래쪽 \(S_-\) → 정적분 \(= S_+ - S_-\). 삼각형 두 개로 시각화.
  4. \(\int_a^a f = 0\), \(\int_a^b = -\int_b^a\) 약속 — "폭이 0이면 넓이 0", "끝점을 바꾸면 부호 반전".
  5. 함수 결정형: \(\int_a^x f(t)\,dt = g(x)\) → ① 양변 미분 \(f(x) = g'(x)\) → ② \(x = a\) 대입 \(g(a) = 0\).
오개념 ①: "정적분 = 도형 넓이" — 음수 구간이 있으면 다르다. 넓이는 \(\int |f|\,dx\).
오개념 ②: "\(\dfrac{d}{dx}\int_a^x f(t)\,dt = f(t)\)" — 결과는 \(f(x)\). \(t\)를 \(x\)로 바꾼다.
오개념 ③: "함수 결정형에서 \(g(a) = 0\)이 아니라 상수" — 좌변 \(\int_a^a f(t)\,dt = 0\)이므로 반드시 \(g(a) = 0\).

E 문항별 해설 카드 (L1 1~3 · L2 4~6 · L3 7 · 형성 8 · 도전 1·2)

L11기하 도형으로 정적분 계산
(1) \(\int_{-2}^{2}(x+4)\,dx\)   (2) \(\int_{-2}^{4}\!\left(\tfrac{1}{2}x+1\right)\!dx\)
(1) \([-2,2]\)에서 \(x+4 \geq 0\). \(x=-2\)에서 \(2\), \(x=2\)에서 \(6\). 사다리꼴 \(\dfrac{(2+6)\cdot4}{2} = 16\). (2) \([-2,4]\)에서 \(\tfrac{1}{2}x+1 \geq 0\). \(x=-2\)에서 \(0\), \(x=4\)에서 \(3\). 삼각형 \(\dfrac{1}{2}\cdot6\cdot3 = 9\).
정답: (1) \(16\)   (2) \(9\)
판서 포인트: 그래프의 \(x\)절편 위치 확인 필수. \([-2,4]\)에서 \(\tfrac{1}{2}x+1=0 \Rightarrow x=-2\)이 구간 왼쪽 끝이므로 전체가 양수.
L12적분과 미분의 관계
\(x\)에 대하여 미분. (1) \(\int_2^x(3t^2-2t+1)\,dt\)   (2) \(\int_{-3}^{x}(t-1)(2t^3+t-1)\,dt\)
공식: \(\dfrac{d}{dx}\int_a^x f(t)\,dt = f(x)\). \(t\)를 \(x\)로 바꾸기만. (1) \(3x^2 - 2x + 1\). (2) \((x-1)(2x^3+x-1)\). — 전개 불필요.
정답: (1) \(3x^2-2x+1\)   (2) \((x-1)(2x^3+x-1)\)
L13함수 결정형 2단계
모든 실수 \(x\)에 대해 \(\int_1^x f(t)\,dt = 2x^3 - x + a\). \(f(x)\)와 \(a\)의 값 구하기.
STEP 1 — 양변 미분: \(f(x) = 6x^2 - 1\). STEP 2 — \(x = 1\) 대입: 좌변 \(= 0\). 우변 \(= 2 - 1 + a = 1 + a = 0 \Rightarrow a = -1\).
정답: \(f(x) = 6x^2 - 1\), \(a = -1\)
L24아래끝이 미지수인 함수 결정형
모든 실수 \(x\)에 대해 \(\int_{-1}^{x}f(t)\,dt = x^4 - x^3 + ax^2 - 5\). \(f(x)\)와 \(a\) 구하기.
STEP 1: \(f(x) = 4x^3 - 3x^2 + 2ax\). STEP 2: \(x = -1\) 대입 → 좌변 \(= 0\), 우변 \(= 1 + 1 + a - 5 = a - 3 = 0 \Rightarrow a = 3\). 따라서 \(f(x) = 4x^3 - 3x^2 + 6x\).
정답: \(f(x) = 4x^3 - 3x^2 + 6x\), \(a = 3\)
L25그래프 + 정적분 접선의 기울기
\(F(x) = \int_0^x f(t)\,dt\). 그래프에서 \(f(3) = 0\)일 때, \(y = F(x)\) 위 점 \((3, F(3))\)의 접선의 기울기.
접선의 기울기 \(= F'(3)\). 적분·미분 관계: \(F'(x) = f(x) \Rightarrow F'(3) = f(3)\). 그래프에서 \(f(3) = 0\) (영점).
정답: 접선의 기울기 \(= 0\)
판서 포인트: \(F(x)\)의 접선 기울기는 \(F'(x)\), 이것이 그래프 \(f(x)\)의 값. 그래프를 읽어내는 능력이 핵심.
L26위끝이 \(x\)인 정적분의 미분값
\(G(x) = \int_0^x(t^3+2t)\,dt\). \(G'(2)\) 구하기.
\(G'(x) = x^3 + 2x\). \(G'(2) = 8 + 4 = 12\). — 적분 없이 피적분함수에 대입.
정답: \(G'(2) = 12\)
L37🚨 부호 있는 면적 vs 도형 넓이
\(y = x\)와 \(x\)축, \(x = -1, 2\)로 둘러싸인 도형 넓이 \(S\). \(I = \int_{-1}^{2}x\,dx\). \(S\)와 \(I\) 비교.
도형 넓이 \(S\): \([-1,0]\) 삼각형 \(\dfrac{1}{2}\), \([0,2]\) 삼각형 \(2\). \(S = \dfrac{5}{2}\). 정적분 \(I\): \(S_+ - S_- = 2 - \dfrac{1}{2} = \dfrac{3}{2}\). (검산: \(\big[\tfrac{1}{2}x^2\big]_{-1}^{2} = 2 - \dfrac{1}{2} = \dfrac{3}{2}\) ✓) \(S = \dfrac{5}{2} \neq I = \dfrac{3}{2}\). 다르다.
정답: \(S = \dfrac{5}{2}\), \(I = \dfrac{3}{2}\). 둘은 다름. 도형 넓이는 \(\int |f|\,dx\)로 구해야 한다.
발표 유도: "정적분과 도형 넓이가 왜 다른지" 한 문장으로. "정적분은 부호 있는 면적이어서 \(x\)축 아래는 음수로 계산하기 때문."
FC8형성평가 — 함수 결정형에서 \(f(a)\)
다항함수 \(f(x)\)가 \(\int_a^x f(t)\,dt = 6x^2 + x - 1\)을 만족. \(a \gt 0\)일 때 \(f(a)\).
① 미분: \(f(x) = 12x + 1\). ② \(x = a\): 좌변 \(= 0\), 우변 \(= 6a^2 + a - 1 = (3a-1)(2a+1) = 0\). \(a \gt 0 \Rightarrow a = \dfrac{1}{3}\). ③ \(f\!\left(\dfrac{1}{3}\right) = 12 \cdot \dfrac{1}{3} + 1 = 5\).
정답: \(f(a) = 5\)
CH1D1⭐ 수능특강 수학Ⅱ 부정적분과 정적분 유제 3
다항함수 \(f(x)\)가 \(\int_0^x tf(t)\,dt = x^4 + ax^2 + \int_0^x f(t)\,dt\)를 만족. \(f(a) = ?\)
STEP 1: 양변 미분 → \(xf(x) = 4x^3 + 2ax + f(x)\) → \((x-1)f(x) = 4x^3 + 2ax\). STEP 2: \(x = 1\) 대입 → \(0 = 4 + 2a \Rightarrow a = -2\). STEP 3: \((x-1)f(x) = 4x^3 - 4x = 4x(x-1)(x+1)\) → \(f(x) = 4x(x+1)\). STEP 4: \(f(-2) = 4(-2)(-1) = 8\).
정답: \(f(a) = 8\). 핵심: 양변 미분 → \(x=1\) 대입으로 \(a\) 결정 → \(f(x)\) 인수분해 복원.
CH2D2⭐ 2024년 5월 학력평가 공통 18번 [3점]
최고차항의 계수가 \(3\)인 이차함수 \(f(x)\)가 \(\int_0^x f(t)\,dt = 2x^3 + \int_0^{-x}f(t)\,dt\)를 만족. \(f(1)=5\)일 때 \(f(2)\).
STEP 1: 양변 미분 → \(f(x) = 6x^2 - f(-x)\) → \(f(x) + f(-x) = 6x^2\). STEP 2: \(f(x) = 3x^2 + bx + c\)로 두면 \(f(x) + f(-x) = 6x^2 + 2c = 6x^2 \Rightarrow c = 0\). STEP 3: \(f(1) = 3 + b = 5 \Rightarrow b = 2\). \(f(x) = 3x^2 + 2x\). STEP 4: \(f(2) = 12 + 4 = 16\).
정답: \(f(2) = 16\). 핵심: 위끝 \(-x\) 미분 시 합성함수 미분으로 \(-1\) 발생 → 대칭식 분리.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 "\(\int_0^2(-x)\,dx = -2\)인데 도형 넓이는 \(2\)인 게 처음엔 모순처럼 보였다. 정적분은 '\(x\)축 위쪽은 \(+\), 아래쪽은 \(-\)'로 부호를 살리는 약속이라 음수가 나온다는 걸 알았다. 도형 넓이를 구하려면 \(\int |f|\,dx\)가 필요하다."
유형 2: 결정적 실수와 극복 "\(\dfrac{d}{dx}\int_a^x f(t)\,dt\)에서 결과로 \(f(t)\)를 써서 틀렸다. '\(t\)를 \(x\)로 바꿔야 한다'는 걸 강조한 뒤, '위끝이 \(x\)이면 피적분함수의 변수를 \(x\)로 교체'라는 카드를 만들었다."
유형 3: 발전 가능성 "\(S(x) = \int_a^x f(t)\,dt\)를 미분하면 \(f(x)\)이므로 \(S\)도 결국 \(f\)의 부정적분. 다음 차시의 \(\int_a^b f\,dx = F(b)-F(a)\) 한 줄이 이것에서 나올 것 같다는 가설을 세웠다."
