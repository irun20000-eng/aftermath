00. 생각 열기 (Warm-up & Connection)

차시 14·15에서 우리는 증감표를 통해 도함수 부호 변화와 극값을 잡았다. 하지만 같은 증감 정보만 가지고도 그래프의 "높낮이"는 무한히 많다. 오늘은 증감과 극값에 좌표축과의 교점까지 더해 그래프 개형을 한 번에 결정하는 5단계 알고리즘을 배운다.

[개방형 발문] 두 학생의 대화: "도함수 \(y = f'(x)\)의 그래프가 \(x = -1,\ 1\)에서 \(x\)축을 가로지른다는 사실만 알아도 함수 \(y = f(x)\)의 그래프 모양은 하나로 정해질까?" 정해지지 않는다면, 무엇을 더 알아야 그래프 개형이 결정되는지 추측해 보자. (비상 미적분 I P.86 · 개념 탐구 참고)

[연결] 차시 14의 증감 + 차시 15의 극값만으로는 그래프가 평행이동된 채로 무수히 많다. 본 차시는 거기에 \(x\)절편·\(y\)절편(좌표축과의 교점)과 극값의 \(y\)좌표를 추가해 그래프 개형을 하나로 결정하는 5단계 표준 절차를 익힌다. 이는 차시 18(방정식의 실근 개수), 차시 19(운동 그래프 해석)의 기초 무기가 된다. (비상 미적분 I P.86 · 그래프의 개형 도입 참고)

01. 개념 구성 (Conceptual Foundation)

1 함수의 그래프의 개형 — 5단계 표준 알고리즘

미분가능한 함수 \(y = f(x)\)의 그래프의 개형은 다음 순서로 그린다.

  1. ❶ 도함수 \(f'(x)\)를 구한다.
  2. ❷ \(f'(x) = 0\)을 만족시키는 \(x\)의 값을 모두 구한다.
  3. ❸ 함수 \(f(x)\)의 증가와 감소를 표(증감표)로 나타내고, 극값을 구한다.
  4. ❹ 함수 \(y = f(x)\)의 그래프와 좌표축과의 교점(\(x\)절편, \(y\)절편)의 좌표를 구한다.
  5. ❺ 위 정보를 종합해 함수 \(y = f(x)\)의 그래프의 개형을 그린다.

핵심: 1·2·3단계는 차시 14·15에서 완성한 도구. 4단계의 \(x\)절편(\(f(x) = 0\)의 실근)과 \(y\)절편(\(f(0)\))이 그래프의 "위치"를 결정한다. 즉 차시 16의 새 도구는 "좌표축과의 교점" 한 가지뿐이다. (비상 미적분 I P.86 · 5단계 알고리즘 참고)

2 예제 — \(y = 2x^3 - 6x + 1\)의 그래프 개형

5단계를 한 번 완주해 보자.

STEP 1·2: \(f'(x) = 6x^2 - 6 = 6(x+1)(x-1)\). \(f'(x) = 0 \Rightarrow x = -1\) 또는 \(x = 1\).

STEP 3 (증감표):

\(x\)\(\cdots\)\(-1\)\(\cdots\)\(1\)\(\cdots\)
\(f'(x)\)\(+\)\(0\)\(-\)\(0\)\(+\)
\(f(x)\)↗\(5\)↘\(-3\)↗

STEP 4: \(y\)절편 \(f(0) = \) \(1\). 극댓값 \(f(-1) = 5\), 극솟값 \(f(1) = -3\). (3차 함수이므로 \(x\)절편은 1~3개, 정확한 값은 인수분해 어려움 — 개형엔 영향 없음.)

STEP 5: 좌하단에서 올라와 \((-1, 5)\) 극대 \(\to\) 내려가 \(y\)축 \(1\) 지나 \((1, -3)\) 극소 \(\to\) 우상으로 올라가는 모양.

주목: 삼차함수 그래프는 항상 좌→하 / 우→상 (양의 최고차) 또는 좌→상 / 우→하 (음의 최고차) 양 끝 방향이 정해진다. 가운데에 극값이 0개 또는 2개. (비상 미적분 I P.86~87 · 예제 1 참고)

3 예제 — \(y = x^4 + 4x^3 + 4x^2 - 1\)의 그래프 개형

사차함수는 극값 패턴이 더 풍부하다.

STEP 1·2: \(f'(x) = 4x^3 + 12x^2 + 8x = 4x(x+2)(x+1)\). \(f'(x) = 0 \Rightarrow x = -2,\ -1,\ 0\).

STEP 3 (증감표):

\(x\)\(\cdots\)\(-2\)\(\cdots\)\(-1\)\(\cdots\)\(0\)\(\cdots\)
\(f'(x)\)\(-\)\(0\)\(+\)\(0\)\(-\)\(0\)\(+\)
\(f(x)\)↘\(-1\)↗\(0\)↘\(-1\)↗

STEP 4: 극솟값 \(f(-2) = \) \(-1\), 극댓값 \(f(-1) = 0\), 극솟값 \(f(0) = \) \(-1\). \(y\)절편 \(f(0) = -1\).

STEP 5: 양끝 모두 \(+\infty\)로 올라가며 \(W\) 모양. 두 극솟값이 같아 그래프는 \(x = -\dfrac{3}{2}\)을 기준으로 거의 대칭처럼 보인다.

핵심: 사차함수(최고차 \(+\))의 양끝은 \(+\infty\). 가운데 극값은 1개(\(\cup\) 모양) 또는 3개(\(W\) 모양 — 극소·극대·극소). 극값 3개의 경우 두 극솟값의 비교가 형성평가·도전의 핵심 포인트. (비상 미적분 I P.87 · 예제 2 참고)

4 그래프 개형의 3대 정보 — 평행이동·대칭·범위

개형이 결정되면 다음 3가지를 즉시 읽을 수 있어야 한다.

  • ① 치역: 삼차함수는 \((-\infty, \infty)\). 사차함수(최고차 \(+\))는 \([\text{(가장 작은 극솟값)}, \infty)\).
  • ② \(x\)절편 개수 = 방정식 \(f(x) = 0\)의 실근 개수: 극값들의 \(y\)좌표 부호로 즉시 판정. 차시 18의 핵심 도구!
  • ③ \(y = k\)와의 교점 개수: 수평선 \(y = k\)가 극값들 사이 어디에 있느냐로 결정. 차시 18의 매개변수 분리.

예시: 위 예제 \(y = 2x^3 - 6x + 1\)에서 극댓값 \(5 \gt 0\), 극솟값 \(-3 \lt 0\). 따라서 \(x\)축과 세 점에서 교차 \(\Rightarrow\) 방정식 \(2x^3 - 6x + 1 = 0\)은 서로 다른 세 실근. 이것이 차시 18에서 본격적으로 쓰일 핵심 추론. (비상 미적분 I P.87 · 그래프 개형의 함의 참고)

Level 1

5단계 알고리즘으로 그래프 개형 그리기

기초-삼차함수 개형
1. 다음 함수의 그래프의 개형을 그리시오. (비상 미적분 I P.87 · 문제 01 참고)
(1) \( y = x^3 + 3x^2 - 2 \)    (2) \( y = -x^3 + 6x^2 - 12x + 4 \)
💡 힌트 보기
💡 힌트:
(1) \(f'(x) = 3x^2 + 6x = 3x(x+2)\). \(f' = 0 \Rightarrow x = -2,\ 0\). 부호 \((+,-,+)\). 극댓값 \(f(-2) = -8 + 12 - 2 = 2\), 극솟값 \(f(0) = -2\). \(y\)절편 \(-2\). 좌하 \(\to (-2, 2)\) 극대 \(\to (0, -2)\) 극소 \(\to\) 우상.
(2) \(f'(x) = -3x^2 + 12x - 12 = -3(x - 2)^2\). \(f' \leq 0\) 항상 (등호는 \(x = 2\)). 부호 변화 없음 \(\Rightarrow\) 극값 없이 단조감소. \(x = 2\)는 정체점(접선 수평)일 뿐. \(y\)절편 \(4\). 좌상 \(\to\) 단조감소 \(\to\) 우하 (\(x = 2\)에서 잠깐 평평).
기초-사차함수 개형
2. 다음 함수의 그래프의 개형을 그리시오. (비상 미적분 I P.87 · 문제 02 참고)
(1) \( y = x^4 - 2x^2 - 2 \)    (2) \( y = -\dfrac{3}{4}x^4 + 2x^3 + 1 \)
💡 힌트 보기
💡 힌트:
(1) \(f'(x) = 4x^3 - 4x = 4x(x-1)(x+1)\). \(f' = 0 \Rightarrow x = -1,\ 0,\ 1\). 부호 \((-,+,-,+)\). 극솟값 \(f(\pm 1) = 1 - 2 - 2 = -3\), 극댓값 \(f(0) = -2\). \(W\) 모양, 두 극솟값 동일 (\(y\)축 대칭).
(2) \(f'(x) = -3x^3 + 6x^2 = -3x^2(x - 2)\). \(f' = 0 \Rightarrow x = 0,\ 2\). 부호: \(x \lt 0\)에서 \(+\), \(0 \lt x \lt 2\)에서 \(+\), \(x \gt 2\)에서 \(-\). 즉 \(x = 0\) 좌우 부호 같음 \(\Rightarrow\) \(x = 0\)은 극값 아님 (정체점). \(x = 2\)에서만 부호 \(+ \to -\) \(\Rightarrow\) 극댓값 \(f(2) = -12 + 16 + 1 = 5\). 단조증가 후 극대 후 감소.
기초-좌표축 교점
3. 함수 \(y = x^3 - 3x^2 + 4\)의 그래프와 \(x\)축의 교점을 모두 구하고, 그래프의 개형을 그리시오. (자체 출제 · 비상 P.87 예제 1 변형)
💡 힌트 보기
💡 힌트: \(f'(x) = 3x^2 - 6x = 3x(x - 2)\). \(f' = 0 \Rightarrow x = 0,\ 2\). 부호 \((+,-,+)\). 극댓값 \(f(0) = 4\), 극솟값 \(f(2) = 8 - 12 + 4 = 0\). \(x\)축 교점: \(x^3 - 3x^2 + 4 = 0 \Rightarrow (x + 1)(x - 2)^2 = 0 \Rightarrow x = -1\) 또는 \(x = 2\) (중근). 극솟값이 \(0\)이라 그래프가 \(x = 2\)에서 \(x\)축에 접한다. 교점은 \((-1, 0),\ (2, 0)\) 두 곳, 단 \((2, 0)\)에서 접함.

Level 2

도함수 그래프 \(\to\) 원함수 개형, 매개변수 결정

응용-도함수 그래프 해석
4. 다음 그림은 삼차함수 \(y = f(x)\)의 도함수 \(y = f'(x)\)의 그래프이다. \(f(-1) = 2,\ f(1) = -2\)일 때, 함수 \(y = f(x)\)의 그래프의 개형을 그리시오. (비상 미적분 I P.86 · 개념 탐구 그림 참고)
💡 힌트 보기
💡 힌트: 도함수 그래프에서 \(f'(x) = 0\)인 \(x = -1, 1\). \(x \lt -1\)에서 \(f' \gt 0\), \(-1 \lt x \lt 1\)에서 \(f' \lt 0\), \(x \gt 1\)에서 \(f' \gt 0\) \(\Rightarrow\) 부호 \((+,-,+)\). 따라서 \(x = -1\)에서 극대 \(f(-1) = 2\), \(x = 1\)에서 극소 \(f(1) = -2\). 좌하 \(\to (-1, 2)\) 극대 \(\to (1, -2)\) 극소 \(\to\) 우상. 핵심: 도함수의 그래프 모양 + 두 함숫값만 알면 원함수 개형이 유일하게 결정된다(평행이동 자유도가 제거됨).
응용-매개변수 + 개형
5. 삼차함수 \(f(x) = x^3 + ax^2 + bx + c\)의 그래프가 점 \((0, 4)\)를 지나고, \(x = 1\)에서 극댓값, \(x = 3\)에서 극솟값을 가질 때, \(a, b, c\)와 극솟값을 구하고 그래프 개형을 설명하시오. (자체 출제 · 비상 P.85 예제 4 변형)
💡 힌트 보기
💡 힌트: \(f(0) = c = 4\). \(f'(x) = 3x^2 + 2ax + b\). 극값 조건 \(f'(1) = 0,\ f'(3) = 0\) \(\Rightarrow\) 근과 계수 \(1 + 3 = -\dfrac{2a}{3} = 4 \Rightarrow a = -6\), \(1 \cdot 3 = \dfrac{b}{3} = 3 \Rightarrow b = 9\). 따라서 \(f(x) = x^3 - 6x^2 + 9x + 4\). 극댓값 \(f(1) = 1 - 6 + 9 + 4 = 8\), 극솟값 \(f(3) = 27 - 54 + 27 + 4 = 4\). 두 극값 모두 양수 \(\Rightarrow\) 그래프는 \(x\)축과 한 점에서만 만남(\(x\)절편 1개).
응용-개형 + 실근 개수 (차시 18 미리보기)
6. 함수 \(f(x) = x^3 - 3x + 2\)의 그래프 개형을 그리고, 방정식 \(x^3 - 3x + 2 = 0\)의 서로 다른 실근의 개수를 그래프 추론으로 구하시오. (자체 출제 · 차시 18 P.90 예제 1 연계)
💡 힌트 보기
💡 힌트: \(f'(x) = 3x^2 - 3 = 3(x-1)(x+1)\). 부호 \((+,-,+)\). 극댓값 \(f(-1) = -1 + 3 + 2 = 4\), 극솟값 \(f(1) = 1 - 3 + 2 = 0\). 극솟값이 정확히 \(0\) \(\Rightarrow\) 그래프가 \(x = 1\)에서 \(x\)축에 접함. \(x\)축과의 교점은 \(x = 1\)(접점, 중근)과 \(x = -2\)(다른 한 실근). 따라서 방정식의 실근 개수는 2개(서로 다른 실근), 단 \(x = 1\)이 중근. (참고: 실제 인수분해 \((x-1)^2(x+2)=0\).) 핵심 규칙: 극값 중 하나가 \(0\) \(\Leftrightarrow\) 중근.

"틀려야 는다!" 극값 0개·1개·3개 분류 — 사차함수의 함정

오답률 1위 함정!
심화-사차함수 극값 개수 분류
7. 사차함수 \(f(x) = x^4 + ax^2 + b\) (\(a, b\)는 실수)의 그래프가 다음과 같을 때, \(a\)의 부호로 가능한 경우를 모두 말하시오. (1) 극값이 1개인 경우 (2) 극값이 3개인 경우 (비상 미적분 I P.87 · 사차함수 개형 분류 + 학생 오개념 함정)
🚨 선생님의 함정 경고 (클릭하여 펼치기)

"사차함수면 \(W\) 모양 (극값 3개)"이라는 고정관념을 깨는 차시 16의 핵심 함정.

STEP 1 — 도함수 분석: \(f'(x) = 4x^3 + 2ax = 2x(2x^2 + a)\). \(f'(x) = 0 \Rightarrow x = 0\) 또는 \(2x^2 + a = 0\), 즉 \(x^2 = -\dfrac{a}{2}\).

STEP 2 — 경우 분류:
① \(a \geq 0\): \(x^2 = -\dfrac{a}{2} \leq 0 \Rightarrow\) 해 없음(\(a = 0\)은 중근). \(f'(x) = 0\)인 실수는 \(x = 0\) 단 하나. 부호: \(x \lt 0\)에서 \(2x \lt 0\), \(2x^2 + a \gt 0\) \(\Rightarrow f' \lt 0\). \(x \gt 0\)에서 \(f' \gt 0\). 즉 부호 \((-,+)\) \(\Rightarrow\) \(x = 0\)에서 극소만 (극값 1개, \(\cup\) 모양).
② \(a \lt 0\): \(x^2 = -\dfrac{a}{2} \gt 0 \Rightarrow x = \pm\sqrt{-\dfrac{a}{2}}\). 세 점 \(x = -\sqrt{-\dfrac{a}{2}},\ 0,\ \sqrt{-\dfrac{a}{2}}\). 부호 \((-,+,-,+)\) \(\Rightarrow\) 극소·극대·극소 3개 (\(W\) 모양).

STEP 3 — 정답:
(1) 극값 1개 \(\Leftrightarrow a \geq 0\) (\(b\)는 \(y\)절편이므로 개형엔 무영향).
(2) 극값 3개 \(\Leftrightarrow a \lt 0\).

핵심 통찰: 사차함수 \(y = x^4 + ax^2 + b\)는 짝수 차수의 대칭형. 도함수가 인수분해될 때 "\(2x^2 + a = 0\)에 해가 있는가"가 극값 개수의 분기점. 단순히 "사차함수 = \(W\) 모양"이라는 도식적 사고를 버려야 한다. 일반적인 사차함수도 항상 극솟값 1개 or 극소·극대·극소 3개 두 패턴 중 하나. 극값 2개나 0개인 사차함수(최고차 양수)는 존재하지 않는다는 정리도 함께 이해해야 한다 — \(f'\)이 3차이므로 실근이 1개 or 3개(중근 포함 시 부호 패턴이 1개와 동치).

03. 형성평가 (Final Check)

8. 함수 \(f(x) = x^3 - 3x^2 + a\)의 그래프가 \(x\)축과 서로 다른 세 점에서 만나도록 하는 실수 \(a\)의 값의 범위를 구하시오. (비상 미적분 I P.97 · 도전 08 변형 · 차시 18 미리보기)
💡 풀이 전략
💡 전략 (4단계):
STEP 1 — 도함수와 극값: \(f'(x) = 3x^2 - 6x = 3x(x - 2)\). \(f' = 0 \Rightarrow x = 0, 2\). 부호 \((+,-,+)\). 극댓값 \(f(0) = a\), 극솟값 \(f(2) = 8 - 12 + a = a - 4\).
STEP 2 — \(x\)축과 세 점에서 만남의 조건: 삼차함수가 \(x\)축과 세 점에서 만나려면 극댓값과 극솟값의 부호가 반대여야 한다 (즉 극대 \(\gt 0 \gt\) 극소). 또는 동치 조건: (극댓값) × (극솟값) \(\lt 0\).
STEP 3 — 부등식: \(a(a - 4) \lt 0 \Rightarrow 0 \lt a \lt 4\).
STEP 4 — 검증: \(a = 2\)이면 극댓값 \(2 \gt 0\), 극솟값 \(-2 \lt 0\) ✓. \(a = 0\)이면 극솟값 \(-4\), 극댓값 \(0\) \(\Rightarrow\) 그래프가 \(x\)축에 접함(중근) \(\Rightarrow\) 서로 다른 실근 2개, 경계 제외. 정답: \(0 \lt a \lt 4\). (차시 18에서 이 사고를 일반화한다.)

⭐ 도전 문제 1

📖 출처: 수능특강 수학Ⅱ · 도함수의 활용 2 · 유제 2 [26009-0100]

난이도 ★★★ (학습지 max+2, 7분) · 핵심: 절댓값 포함 함수의 분기 + 닫힌구간에서의 최대·최소 — 구간을 \(x\lt -1\), \(x\geq -1\)로 나눠 각각 미분 후 양 끝값·극값·접점값 모두 비교

닫힌구간 \([-2,\ 2]\)에서 함수 \(f(x) = x^3 - 3|x + 1| + 3\)의 최댓값과 최솟값을 각각 \(M,\ m\)이라 할 때, \(M + m\)의 값은?
① \(-6\)    ② \(-4\)    ③ \(-2\)    ④ \(0\)    ⑤ \(2\)
📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 풀이 흐름 (4단계):
  1. STEP 1 — 구간 분기 (\(x \lt -1\)): \(|x+1| = -(x+1)\)이므로 \(f(x) = x^3 + 3x + 6\), \(f'(x) = 3x^2 + 3 \gt 0\). 이 구간에서 \(f\)는 단조증가.
  2. STEP 2 — 구간 분기 (\(x \geq -1\)): \(|x+1| = x+1\)이므로 \(f(x) = x^3 - 3x\), \(f'(x) = 3x^2 - 3 = 3(x-1)(x+1)\). \(x = 1\)에서 극소(부호 \(- \to +\)).
  3. STEP 3 — 후보 함숫값: \(f(-2) = -8\), \(f(-1) = 2\) (좌·우 일치), \(f(1) = -2\), \(f(2) = 2\).
  4. STEP 4 — 최종: \(M = 2\) (at \(x = -1\) 또는 \(x = 2\)), \(m = -8\) (at \(x = -2\)). \(M + m = -6\). 정답: ① \(-6\)

핵심: 절댓값이 끼어 있는 함수의 최대·최소는 ① 절댓값 분기점을 기준으로 구간을 나눠 부분 함수를 명시 ② 각 부분에서 미분으로 임계점 탐색 ③ 분기점에서의 좌우 함숫값 일치 확인(연속성) ④ 닫힌구간 양 끝값·분기점값·내부 임계점값을 모두 비교. 차시 16 시나리오의 절댓값·그래프 분석 패턴이 평가원형으로 어떻게 결합되는지 보여주는 정통 사례.

※ 자세한 풀이는 수능특강 수학Ⅱ 해설지 유제 2 참조.

⭐ 도전 문제 2

📖 출처: 2004년 10월 학력평가 수학영역 가형 6번 [3점]

난이도 ★★★ (학습지 max+2, 7분) · 핵심: 도함수 \(f'(x)\)의 그래프 \(\to\) 원함수 \(f(x)\)의 식 복원 \(\to\) 방정식 \(f(x) = kx\)의 실근 개수 (그래프 + 직선 교점 분석)

삼차함수 \(y = f(x)\)의 도함수 \(y = f'(x)\)의 그래프는 다음과 같다.
\(f(0) = 0\)일 때, \(x\)에 대한 방정식 \(f(x) = kx\)가 서로 다른 세 실근을 갖기 위한 실수 \(k\)의 값의 범위는?

① \(k \gt 2\)    ② \(k \gt 3\)    ③ \(k \lt 3\)

④ \(-4 \lt k \lt 4\)    ⑤ \(k \lt -2\) 또는 \(k \gt 2\)

📖 풀이 흐름 보기 (직접 풀어본 후 펼치기)
💡 풀이 흐름 (5단계):
  1. STEP 1 — 도함수 \(f'(x)\)의 식 복원: 그래프에서 \(f'(x)\)는 위로 볼록한 이차함수이고 \(x = \pm 2\)에서 \(0\), \(f'(0) = 3\) \(\Rightarrow f'(x) = -\dfrac{3}{4}(x + 2)(x - 2) = -\dfrac{3}{4}x^2 + 3\).
  2. STEP 2 — \(f(x)\) 적분 + 초기조건: \(f(x) = -\dfrac{1}{4}x^3 + 3x + C\). \(f(0) = 0 \Rightarrow C = 0\). 따라서 \(f(x) = -\dfrac{x^3}{4} + 3x\).
  3. STEP 3 — 방정식 정리: \(f(x) = kx \Leftrightarrow -\dfrac{x^3}{4} + 3x = kx \Leftrightarrow x\!\left(-\dfrac{x^2}{4} + 3 - k\right) = 0\). \(x = 0\)이 한 근이므로 나머지 두 근이 0이 아니고 서로 다른 실근이어야 한다.
  4. STEP 4 — 이차방정식 조건: \(-\dfrac{x^2}{4} + 3 - k = 0 \Leftrightarrow x^2 = 4(3 - k)\). 서로 다른 두 0이 아닌 실근 \(\Leftrightarrow 4(3 - k) \gt 0 \Leftrightarrow k \lt 3\).
  5. STEP 5 — 정답: \(k \lt 3\). 또한 \(k = 0\)일 때도 \(x^2 = 12\)는 두 실근이고 \(x = 0\)을 더해 세 실근이 되므로 \(k = 0\) 포함 OK. 정답: ③ \(k \lt 3\).

핵심: "도함수 그래프 \(\to\) 원함수 복원" 사고가 차시 16의 핵심. ①\(f'\)의 식을 정확히 읽고 ②적분상수를 초기조건으로 결정 ③원함수를 이용해 방정식 해석. 이는 차시 18(방정식·부등식 활용)의 핵심 전제 — 도함수가 보여주는 정보(극값·증감)로부터 원함수 그래프 개형을 추론하고 \(y = k\) 또는 \(y = kx\)와의 교점 개수를 통제한다.
