C 생각 열기 운영 가이드

발문: "미분하여 \(2x\)가 되는 함수를 모두 적어 보자. 공통점과 차이점은?"

예상 반응 ①"\(x^2\)이요."
피드백: 맞다. \(x^2 + 1\)도? \(x^2 - 7\)도? 공통점은 "\(x^2\)" 부분, 차이점은 상수. 그 상수를 \(C\)로 묶으면 \(x^2 + C\)로 표현된다.
예상 반응 ②"무한히 많다."
피드백: 정확하다! "무한히 많지만 모두 상수 차이만큼 다르다" — 이것이 \(+ C\)의 수학적 근거. 평균값 정리 따름정리로 뒷받침됨.
예상 반응 ③"거꾸로 미분?" (막연한 역방향 인식)
피드백: 딱 맞다. 적분 = "미분의 역연산". \(\dfrac{d}{dx}\int f(x)\,dx = f(x)\)가 이 한 줄을 수식으로 표현. 오늘 단원 전체의 출발점.

단원 II 연결: 단원 II에서 \((x^{n+1})' = (n+1)x^n\)이었다면, 이를 역방향으로 읽으면 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\). 공식 암기가 아니라 미분 공식의 역방향임.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE FORMULA — 부정적분의 정의와 거듭제곱 공식
\( \displaystyle \int f(x)\,dx = F(x) + C \quad (F'(x) = f(x)) \qquad \int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C \quad (n \neq -1) \)
📌 판서 시연 5단계:
  1. "\(\dfrac{d}{dx} \leftrightarrow \int\)" 양방향 화살표를 칠판 중앙에 크게 그린다.
  2. \((x^{n+1})' = (n+1)x^n\)을 역방향으로 읽어 \(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)를 유도.
  3. \(\int (3x^2 + 4x - 2)\,dx = x^3 + 2x^2 - 2x + C\) 항별 적분 시연. 적분상수 \(C\)는 항 수에 관계없이 하나.
  4. "접선의 기울기 \(= f'(x)\)" 선언 후 \(f'(x) = x - 3\), \((2, 4)\) 통과 문제 4단계 판서.
  5. 초기조건 결정: \(f(0) = k\) 또는 점 \((p, q)\) 통과 → \(C\) 확정. "\(C\)는 그래프의 \(y\)축 이동량"으로 기하 해석.
오개념 ①: "\(\int (3x^2)\,dx = 3x^2 + C\)" — 지수를 올리지 않고 그대로 두는 오류. \((x^3)' = 3x^2\)이므로 역방향은 \(x^3\).
오개념 ②: "항이 3개면 \(C\)도 3개" — 적분상수는 항상 하나로 묶인다. \(3C_1\)이 아닌 단순 \(C\).
오개념 ③: "초기조건이 없으면 \(C = 0\)" — 조건이 없으면 \(C\)는 임의 상수로 남겨둔다.

\(f'(x) = -8x^3 + 3x^2 - 4,\ f(0) = -1\)일 때 \(f(x)\).
적분: \(f(x) = -2x^4 + x^3 - 4x + C\). \(f(0) = C = -1\).

L24곱셈 전개 후 적분 + 두 적분의 차
(1) \(\int(x^2-2x+3)\,dx\)   (2) \(\int x(x+1)^2\,dx - \int x(x^2+1)\,dx\)
(1) \(\dfrac{1}{3}x^3 - x^2 + 3x + C\). (2) 피적분 차: \(x[(x+1)^2 - (x^2+1)] = x \cdot 2x = 2x^2\). \(\int 2x^2\,dx = \dfrac{2}{3}x^3 + C\).
정답: (1) \(\dfrac{1}{3}x^3 - x^2 + 3x + C\)   (2) \(\dfrac{2}{3}x^3 + C\)
L25접선의 기울기 + 통과점
기울기가 \(x - 3\), 점 \((2, 4)\) 통과. \(f(x)\) 구하기.
\(f'(x) = x - 3 \Rightarrow f(x) = \dfrac{1}{2}x^2 - 3x + C\). \(f(2) = 2 - 6 + C = 4 \Rightarrow C = 8\).
정답: \(f(x) = \dfrac{1}{2}x^2 - 3x + 8\)
L26도함수 + 두 점 조건 → \(f(-1)\)
기울기 \(6x^2 + 12x + 5\), 점 \((-2, 1)\) 통과. \(f(-1)\) 구하기.
\(f(x) = 2x^3 + 6x^2 + 5x + C\). \(f(-2) = -16 + 24 - 10 + C = -2 + C = 1 \Rightarrow C = 3\). \(f(-1) = -2 + 6 - 5 + 3 = 2\).
정답: \(f(-1) = 2\)
L37🚨 도함수 일반항 + 두 조건으로 자연수 \(n\) 결정
\(f'(x) = 1 + 2x + 3x^2 + \cdots + nx^{n-1}\), \(f(0) = 5\), \(f(1) = 11\). \(n\) 구하기. (\(n \geq 2\))
STEP 1: 항별 적분 \(\int kx^{k-1}\,dx = x^k\) → \(f(x) = x + x^2 + \cdots + x^n + C\). STEP 2: \(f(0) = C = 5\). STEP 3: \(f(1) = \underbrace{1 + 1 + \cdots + 1}_{n} + 5 = n + 5 = 11 \Rightarrow n = 6\).
정답: \(n = 6\). 핵심: "도함수 항 \(kx^{k-1}\)을 적분하면 \(x^k\)가 정확히 나온다"는 인식이 관문.
발표 포인트: "왜 항별로 \(x^k\)가 나오는가?"를 칠판에서 \((x^k)' = kx^{k-1}\)의 역방향으로 보여준다. 수열(단원 I) + 미분(단원 II) + 부정적분(단원 III) 통합 문제.
FC8형성평가 — 극값 조건 + 부정적분
\(f'(x) = ax^2 - ax\), 극댓값 \(6\), 극솟값 \(5\), \(a \gt 0\). \(f(x)\) 구하기.
\(f'(x) = ax(x-1) = 0 \Rightarrow x = 0\)(극대), \(x = 1\)(극소) (\(a \gt 0\)). \(f(x) = \dfrac{a}{3}x^3 - \dfrac{a}{2}x^2 + C\). 극댓값 \(f(0) = C = 6\). 극솟값 \(f(1) = \dfrac{a}{3} - \dfrac{a}{2} + 6 = -\dfrac{a}{6} + 6 = 5 \Rightarrow a = 6\).
정답: \(f(x) = 2x^3 - 3x^2 + 6\). 단원 II 극값 + 단원 III 부정적분의 결합.
CH1D1⭐ 수능특강 수학Ⅱ 부정적분과 정적분 유제 1
기울기 \(3x^2 - 2x + 1\), 점 \((1, 2)\) 통과. \(f(0)\)은? ① 1 ② 2 ③ 3 ④ 4 ⑤ 5
STEP 1: \(f'(x) = 3x^2 - 2x + 1\). STEP 2: \(f(x) = x^3 - x^2 + x + C\). STEP 3: \(f(1) = 1 - 1 + 1 + C = 1 + C = 2 \Rightarrow C = 1\). STEP 4: \(f(0) = 0 + 1 = 1\).
정답: ① 1. 핵심: "접선의 기울기 \(= f'(x)\)" → 부정적분 → 한 점으로 \(C\) 결정 → 요구 함숫값 계산.
CH2D2⭐ 2024년 3월 학력평가 공통 5번 [3점]
\(f'(x) = x(3x+2)\), \(f(1) = 6\). \(f(0)\)은? ① 1 ② 2 ③ 3 ④ 4 ⑤ 5
STEP 1: \(f'(x) = 3x^2 + 2x\). STEP 2: \(f(x) = x^3 + x^2 + C\). 검산: \((x^3+x^2+C)' = 3x^2 + 2x\) ✓. STEP 3: \(f(1) = 1 + 1 + C = 2 + C = 6 \Rightarrow C = 4\). STEP 4: \(f(0) = 0 + 0 + 4 = 4\).
정답: ④ 4. 핵심: 곱셈 형태 도함수는 전개부터, 항별 부정적분, 한 점으로 \(C\) 결정.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 "\(\int x^n\,dx = \dfrac{x^{n+1}}{n+1} + C\)를 공식으로 외웠는데, \((x^{n+1})' = (n+1)x^n\)을 역방향으로 읽으면 자연스럽게 나온다는 걸 알았다. \(+C\)가 '평균값 정리로 도함수가 같으면 상수 차이'라는 정리의 결과라는 게 인상 깊었다."
유형 2: 결정적 실수와 극복 "2번에서 \(\int(-x^4)\,dx = -x^4 + C\)로 지수를 올리지 않고 그대로 썼다. \((x^5)' = 5x^4\)를 역방향으로 적용하면 \(-\dfrac{1}{5}x^5\)가 됨을 알고, 앞으로 적분하면 지수를 반드시 하나 올린다는 체크를 먼저 한다."
유형 3: 발전 가능성 "부정적분은 함수족 \(F(x) + C\)를 반환하지만, 정적분 \(\int_a^b f\,dx\)는 한 개의 수라는 점이 궁금하다. \(F(b) - F(a)\) 계산 시 \(C\)가 사라질 것 같은데, 다음 차시에서 확인하고 싶다."
