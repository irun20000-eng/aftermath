C 단원 회고 발문 (Warm-up 대체)

발문: "26~28차시에서 가장 자주 막혔던 도구는? 절댓값·부호 처리가 어려웠다면 그 신호는 어떻게 보이는가?"

예상 반응 ①"넓이 공식에서 절댓값을 빼먹고 그냥 \(\int f\,dx\)로 적음."
피드백: 신호 = "f의 부호가 구간 안에서 바뀜". 교점·영점을 먼저 구해 부호 확인 → 구간 분리 → 절댓값 제거.
예상 반응 ②"변위 \(\int v\,dt\)와 거리 \(\int |v|\,dt\) 헷갈림."
피드백: 신호 = "돌아온다·방향 바뀐다". 변위는 부호 보존, 거리는 항상 양수 → v=0인 시각에서 분리 후 절댓값.
예상 반응 ③"두 곡선 사이 넓이에서 위·아래 구분 없이 그냥 뺌."
피드백: 신호 = "교점 2개 이상". 교점 사이에서 어느 함수가 위인지 확인, \(|f-g|\) 또는 구간 분리.

III-3 예고: 다음 단원 미분법에서는 정적분을 역으로 써 "넓이 함수 \(F(x) = \int_a^x f(t)\,dt\)" 를 미분하는 미적분 기본정리가 핵심 도구로 재등장.

D 개념 지도 노트 — 단원 III-2 종합 공식 트리

DECISION TREE — 문제 보고 공식 선택
곡선과 x축 넓이: \(S = \displaystyle\int_a^b |f(x)|\,dx\)  |  두 곡선 넓이: \(S = \displaystyle\int_a^b |f(x)-g(x)|\,dx\)  |  변위: \(\displaystyle\int_{t_1}^{t_2} v(t)\,dt\)  |  거리: \(\displaystyle\int_{t_1}^{t_2} |v(t)|\,dt\)
📌 판서 시연 — 단원 종합 4단계 (서술형 대비):
  1. 문제 유형 태그: 넓이(x축) / 넓이(두 곡선) / 변위 / 거리 중 하나 박스로 표시.
  2. 교점·영점·v=0 시각을 먼저 구해 구간 분리 기준 확인.
  3. 절댓값 처리: 구간마다 부호 확인 후 절댓값 제거하고 적분.
  4. 미적분 기본정리로 역검증: \(F(b)-F(a)\). 부호·단위 최종 확인.
가장 자주 틀리는 3가지: ① 넓이에서 절댓값 누락 (음수 넓이) ② 거리에서 \(v=0\) 구간 분리 안 함 (변위로 잘못 계산) ③ 두 곡선에서 위·아래 바꿔서 빼기 (부호 오류)

E 문항별 해설 카드 (14문항 종합)

기초1곡선과 x축 넓이
\(y = x^2 - 4x - 5\)와 \(x\)축 사이의 넓이.
영점: \(x^2-4x-5 = (x-5)(x+1) = 0\) → \(x = -1,\,5\). \([-1,\,5]\)에서 \(f \leq 0\): \(S = -\displaystyle\int_{-1}^{5}(x^2-4x-5)\,dx\). \(\displaystyle\int_{-1}^{5}(x^2-4x-5)\,dx = \left[\tfrac{x^3}{3}-2x^2-5x\right]_{-1}^{5}\) \(= \bigl(\tfrac{125}{3}-50-25\bigr)-\bigl(-\tfrac{1}{3}-2+5\bigr) = -\tfrac{100}{3} - \tfrac{8}{3} = -36\). 따라서 \(S = 36\).
정답: \(S = 36\)
기초2두 곡선 사이 넓이
두 곡선 \(y = x^2 - 6x + 9\), \(y = -2x^2 + 6x\)로 둘러싸인 도형의 넓이.
교점: \(x^2-6x+9 = -2x^2+6x\) → \(3x^2-12x+9=0\) → \(x^2-4x+3 = (x-1)(x-3)=0\) → \(x=1,\,3\). \([1,3]\)에서 \(-2x^2+6x \geq x^2-6x+9\). 위 - 아래 \(= -3x^2+12x-9 = -3(x-1)(x-3)\). 1/6 공식: \(S = -3\displaystyle\int_1^3 (x-1)(x-3)\,dx = -3 \cdot \left(-\dfrac{(3-1)^3}{6}\right) = -3 \cdot \left(-\dfrac{4}{3}\right) = 4\).
정답: \(S = 4\)
판서 포인트: 두 곡선 사이 정적분에서 \(-(x-\alpha)(x-\beta)\) 인수분해가 보이면 \(\int_\alpha^\beta (x-\alpha)(x-\beta)\,dx = -\dfrac{(\beta-\alpha)^3}{6}\) 1/6 공식 직행. 시간 단축의 핵심.
기초3속도·변위·거리 기본형
좌표 \(-1\)인 점에서 출발, \(v(t) = t^2 - 4t + 3\). (1) \(t=2\)에서 위치   (2) \(t=0\)~\(t=3\) 위치 변화량   (3) \(t=0\)~\(t=2\) 움직인 거리.
영점: \(v(t) = (t-1)(t-3)\). \([0,1]\)에서 \(v \geq 0\), \([1,3]\)에서 \(v \leq 0\). 부정적분 \(\int v\,dt = \tfrac{t^3}{3}-2t^2+3t + C\). (1) \(x(2) = x(0) + \displaystyle\int_0^2 v\,dt = -1 + \left[\tfrac{t^3}{3}-2t^2+3t\right]_0^2 = -1 + \left(\tfrac{8}{3}-8+6\right) = -1 + \tfrac{2}{3} = -\tfrac{1}{3}\). (2) 위치 변화량 \(= \displaystyle\int_0^3 v\,dt = \left[\tfrac{t^3}{3}-2t^2+3t\right]_0^3 = 9-18+9 = 0\). (3) 거리 \(= \displaystyle\int_0^2 |v|\,dt = \displaystyle\int_0^1 v\,dt + \left(-\displaystyle\int_1^2 v\,dt\right)\). \(\int_0^1 v = \tfrac{4}{3}\), \(\int_1^2 v = (\tfrac{8}{3}-8+6) - (\tfrac{1}{3}-2+3) = \tfrac{2}{3} - \tfrac{4}{3} = -\tfrac{2}{3}\) → 거리 \(= \tfrac{4}{3} + \tfrac{2}{3} = 2\).
정답: (1) \(-\dfrac{1}{3}\)   (2) \(0\)   (3) \(2\)
판서 포인트: (1) "위치"는 출발점 \(x(0) = -1\)에 변위 누적, "변화량"이 아님에 유의. (3) "거리"는 \(|v|\) 적분이라 부호 변화 구간 \([1,2]\)는 \(-\int v\)로 더해야 함.
기본4계수 역대입 — 넓이 조건
\(y = ax^2(x-3)\)과 \(x\)축 사이 넓이가 \(27\). \(a\)의 값.
영점: \(x=0,\,3\) (\(a \gt 0\)이면 \([0,3]\)에서 \(y \leq 0\)). \(S = -\displaystyle\int_0^3 ax^2(x-3)\,dx = -a\displaystyle\int_0^3(x^3-3x^2)\,dx\). \(= -a\left[\tfrac{x^4}{4}-x^3\right]_0^3 = -a\bigl(\tfrac{81}{4}-27\bigr) = -a \cdot (-\tfrac{27}{4}) = \tfrac{27a}{4}\). \(\tfrac{27a}{4} = 27\) → \(a = 4\).
정답: \(a = 4\)
기본5접선과 곡선 사이 넓이
\(y = x^3 + 1\)의 점 \((1,\,2)\)에서의 접선과 곡선으로 둘러싸인 넓이.
접선 기울기: \(f'(x)=3x^2\) → \(f'(1)=3\). 접선: \(y = 3(x-1)+2 = 3x-1\). 교점: \(x^3+1 = 3x-1\) → \(x^3-3x+2=0\) → \((x-1)^2(x+2)=0\) → \(x=-2,\,1\). \([-2,1]\)에서 \(3x-1 \geq x^3+1\): \(S = \displaystyle\int_{-2}^{1}(3x-1-x^3-1)\,dx = \displaystyle\int_{-2}^{1}(-x^3+3x-2)\,dx\). \(= \left[-\tfrac{x^4}{4}+\tfrac{3x^2}{2}-2x\right]_{-2}^{1} = (-\tfrac{1}{4}+\tfrac{3}{2}-2)-(- 4+6+4) = -\tfrac{3}{4}-6 = -\tfrac{27}{4}\). \(S = \dfrac{27}{4}\).
정답: \(S = \dfrac{27}{4}\)
기본6넓이 이등분 조건 — 그래프 의존
그래프 images/calc_29_q06.png. \(y = (x+a)^3\)와 \(x\)축 및 직선 \(x=2\)로 둘러싸인 넓이를 직선 \(x=b\)가 이등분. 조건 \((a+2)^3 = 16\)에서 \(a\)를 구하시오.
전체 넓이: \(S = \displaystyle\int_{-a}^{2}(x+a)^3\,dx = \left[\tfrac{(x+a)^4}{4}\right]_{-a}^{2} = \tfrac{(2+a)^4}{4}\). 이등분 조건: \(x=b\)가 넓이를 절반 \(\tfrac{(2+a)^4}{8}\)로 분리. \(\displaystyle\int_{-a}^{b}(x+a)^3\,dx = \tfrac{(b+a)^4}{4} = \tfrac{(2+a)^4}{8}\). \((b+a)^4 = \tfrac{(2+a)^4}{2}\) → \(b+a = \dfrac{2+a}{2^{1/4}}\). 학습지 조건 \((a+2)^3 = 16\): 이는 특정 그래프에서 도출된 식. 그림 확인 후 \(a\) 계산. \((a+2)^3 = 16\) → \(a+2 = 2\sqrt[3]{2}\) → \(a = 2\sqrt[3]{2}-2\).
정답: \((a+2)^3 = 16\) (그래프 조건), \(a = 2\sqrt[3]{2}-2\)
그림 확인 필수: calc_29_q06.png에서 이등분 기준선 \(x=b\)의 정확한 위치와 곡선의 영점 확인 후 판서.
기본7포물선과 \(|x-2|\) 넓이 — 그래프 의존
그래프 images/calc_29_q07.png. 포물선 \(y = f(x)\)와 \(y = |x-2|\) 사이의 넓이 \(\dfrac{20}{3}\).
\(|x-2|\)는 \(x=2\)에서 꺾이는 V자형. 교점을 기준으로 구간 분리. 그림에서 포물선과 교점 두 개 확인. \(x \leq 2\): \(|x-2| = 2-x\), \(x \geq 2\): \(|x-2| = x-2\). \(S = \displaystyle\int_{\alpha}^{\beta}|f(x)-|x-2||\,dx = \dfrac{20}{3}\) (학습지 정답). 구체적 포물선 식은 그림에서 읽음. 일반형: 교점 \(x_1, x_2\) 에서 분리 계산.
정답: \(S = \dfrac{20}{3}\)
그림 확인 필수: calc_29_q07.png에서 포물선 방정식·교점 좌표 확인 후 구간 분리 판서.

기본9역함수 대칭 넓이 — 그래프 의존
그래프 images/calc_29_q09.png. 함수 \(y=f(x)\)와 역함수 \(y=f^{-1}(x)\)로 둘러싸인 넓이. 정답 \(\dfrac{32}{3}\).
역함수 대칭 원리: \(y=f(x)\)와 \(y=f^{-1}(x)\)는 \(y=x\)에 대칭. 두 곡선이 만나는 점은 \(y=x\) 위 또는 \(f(x)=x\)의 해. 넓이 \(= 2\displaystyle\int_{x_1}^{x_2}|f(x)-x|\,dx\) (대칭성 이용). \(= \dfrac{32}{3}\) (학습지 정답). 그림에서 교점 좌표 읽어 계산.
정답: \(S = \dfrac{32}{3}\)
그림 확인 필수: calc_29_q09.png에서 교점·함수 식 확인. 대칭 넓이 \(= 2 \times (\text{f와 y=x 사이 넓이})\) 판서.
기본10드론 — 구간별 속도 적분
드론이 \(v(t)\)로 비행. 총 이동 거리 \(33\,\text{m}\).
학습지에서 \(v(t)\)는 구간별로 정의. 각 구간에서 속력 계산. 이동 거리 \(= \displaystyle\int |v(t)|\,dt = 33\,\text{m}\). 구간 분리: \(v \geq 0\) 구간은 \(\int v\,dt\), \(v \leq 0\) 구간은 \(-\int v\,dt\). 합산 \(= 33\). 학습지 단위: m (미터). 교사: 구간 경계값 칠판 판서.
정답: \(33\,\text{m}\)
주의: 이동 거리는 \(\int |v|\,dt\) (거리). 변위 \(\int v\,dt\)와 혼동하지 않도록. 드론 문항은 구간 속도값 확인 필수.
기본11원점 복귀 시각
원점에서 출발한 점이 \(v(t)\)로 운동. 다시 원점에 돌아오는 시각 \(t\).
초기 위치 \(x(0)=0\). 원점 복귀 조건: \(x(t) = \displaystyle\int_0^t v(s)\,ds = 0\). 즉 변위 \(= 0\)인 시각 \(t\)를 구함. 학습지 정답 \(t = \dfrac{8}{3}\). \(\displaystyle\int_0^{8/3} v\,dt = 0\) 검증.
정답: \(t = \dfrac{8}{3}\)
핵심: 원점 복귀 = 변위 0 = \(\int_0^t v\,ds = 0\). 이동 거리(= 총 경로)가 아님.
기본12넓이 비율 조건 \(A:B=1:2\) — 그래프 의존

정답: \(k = 6\)
그림 확인 필수: calc_29_q12.png에서 A, B 경계와 \(k\)의 역할 확인 후 판서.
도전13⭐ 서술형 — 두 점 만남 횟수
수직선 위 두 점 P, Q가 각각 속도 \(v_P(t)\), \(v_Q(t)\)로 출발. \(t \geq 0\)에서 두 점이 만나는 횟수. 서술 풀이. 정답 3회.
① 두 점이 만날 조건: 위치 \(x_P(t) = x_Q(t)\), 즉 \(x_P(t) - x_Q(t) = 0\). ② \(x_P(t) - x_Q(t) = \displaystyle\int_0^t \bigl(v_P(s) - v_Q(s)\bigr)\,ds\). ③ 상대 속도 \(w(t) = v_P(t) - v_Q(t)\)를 먼저 구함. \(w(t) = 0\)인 시각과 부호 변화 분석. ④ \(\displaystyle\int_0^t w(s)\,ds = 0\)이 되는 \(t\)의 개수 = 만남 횟수. 학습지 정답: 3회. 그래프에서 위치 차 \(h(t)\)가 0이 되는 횟수 = 3. 서술 채점 포인트: ① 위치 동치 조건 ② 상대 속도 정의 ③ 부호 변화 분석 ④ 만남 횟수 결론.
정답: 3회. 서술 채점: ① 위치 동치 식 ② 적분으로 위치 차 표현 ③ 영점 분석 ④ 횟수 결론. 4단계.
도전14⭐ 서술형 — 두 영역 넓이 동치 조건
\(y = f(x)\)와 \(y = g(x)\)로 둘러싸인 두 영역의 넓이가 같을 조건에서 \(\dfrac{b}{a}\)를 구하시오. 서술 풀이. 정답 \(\dfrac{b}{a} = 2\).
① 두 영역의 넓이 동치: \(S_1 = S_2\). ② 교점 \(a, b\) (또는 적분 경계)를 먼저 구함. ③ 넓이 동치 조건을 적분 방정식으로 세움: \(\displaystyle\int_{\alpha}^{a} |f-g|\,dx = \displaystyle\int_{a}^{\beta} |f-g|\,dx\). ④ 식 정리 → \(\dfrac{b}{a}\) 결정. 학습지 정답: \(\dfrac{b}{a} = 2\). 서술 채점 포인트: ① 교점 도출 ② 두 넓이 적분식 ③ 동치 방정식 ④ \(\dfrac{b}{a}\) 계산.
정답: \(\dfrac{b}{a} = 2\). 서술 채점: ① 교점·구간 ② 넓이 등식 ③ 방정식 풀기 ④ 비율 도출. 4단계.

G 중단원 성찰 — 3유형 우수 답안 예

유형 1: 단원 종합 통찰 "26~28차시 넓이·거리를 별개 공식으로 외웠는데, 마무리에서 보니 모두 '절댓값 → 구간 분리'라는 한 원리에서 출발했다. 그 절댓값이 단원 전체의 핵심이라는 걸 알게 됐다."
유형 2: 자주 틀린 유형 분석 "변위와 거리를 계속 헷갈렸다. 이제는 '돌아온다·방향 바뀐다'는 신호를 보면 자동으로 \(v=0\) 시각을 찾고 구간 분리 → \(\int|v|\,dt\)로 간다. 변위는 부호 그대로."
유형 3: 다음 단원 확장 "정적분으로 넓이·거리를 계산했는데, 다음 단원 미분법에서 \(F'(x) = f(x)\)로 역방향으로 쓰는 미적분 기본정리가 다시 나온다고 해서, 이번 단원의 적분 계산이 그 기반임을 알게 됐다."
