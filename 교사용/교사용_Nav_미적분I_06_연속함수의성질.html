C 생각 열기 운영 가이드

발문: 갈매기가 선착장(0 m) → 정상(98.6 m)으로 날 때, 해발 50 m 지점을 반드시 지나는가? 헬리콥터처럼 점프하면?

예상 반응 ①"당연히 지난다. 0에서 98까지 가려면."
피드백: 옳다. 그러나 그 직관을 수학적 가정으로 풀면 무엇이 필요할까? "끊김 없이" = "연속". 사잇값 정리의 핵심 가정.
예상 반응 ②"꼭 그렇지 않다. 갑자기 위로 점프할 수도 있다."
피드백: 좋은 비판적 시각. "점프 = 불연속"이고, 사잇값 정리는 연속이라는 가정이 있어야 성립한다. 가정의 중요성 자각.
예상 반응 ③"높이를 단계별로 측정하면 50 m를 놓칠 수도 있다."
피드백: 이산화의 한계 vs 연속의 강함. 수학에서는 "함수가 연속"이라는 가정 하나로 모든 중간 값을 보장. 사잇값 정리의 위력.

05차시 연결: 어제 배운 "연속 = 끊김 없음"이라는 정의에서 오늘은 그 정의가 만들어내는 두 가지 강력한 정리(EVT, IVT)를 만난다. 정리는 정의로부터 자동으로 따라온다.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

EVT & IVT
EVT: \(f\)가 \([a,b]\)에서 연속 \(\Rightarrow\) 최댓값·최솟값 존재.  |  IVT: \(f\)가 \([a,b]\)에서 연속, \(f(a) \neq f(b)\), \(k\)가 그 사이 값 \(\Rightarrow\) \(f(c) = k\)인 \(c \in (a, b)\) 적어도 하나.
📌 판서 시연 5단계:
  1. EVT 반례 — 열린구간 \((-1, 2)\)에서 \(f(x) = x^2\): \(x \to 2\)에 가까워지지만 도달 못 함. 최댓값 없음.
  2. EVT 반례 — \(\frac{1}{x}\)를 \((0, 1]\)에서: \(x \to 0^+\) 발산. 최댓값 없음. → "닫힘 + 연속" 둘 다 필수.
  3. IVT 시연 — \(f(x) = x^3 - x - 1\)이 \([-1, 2]\)에서 연속, \(f(-1) = -1 \lt 0\), \(f(2) = 5 \gt 0\) → \((-1, 2)\)에 실근 적어도 하나.
  4. "적어도 하나"의 의미 — \(f(x) = x^3 - x\), \([-2, 2]\)에서 부호 변화하지만 실근은 \(x = -1, 0, 1\) 3개. "정확히 하나"가 아님.
  5. 합성의 연속 — \(g(x) = x^2 - 3\)이 연속, \(f(t) = \sqrt{t}\)가 \(t \geq 0\)에서 연속 → \(\sqrt{x^2 - 3}\)이 정의역에서 연속.
오개념 ①: "IVT는 실근이 정확히 한 개임을 보장한다." → "적어도 하나"일 뿐. 출렁이는 그래프는 여러 실근 가능.
오개념 ②: "\(f(a) f(b) \geq 0\)이면 실근이 없다." → IVT 역명제는 거짓. 같은 부호여도 \(x^2 - 1\)처럼 실근 가능.
오개념 ③: "열린구간 \((a, b)\)에서도 EVT가 성립한다." → 닫힌이 깨지면 최댓값 미존재 가능 (반례 1·2).
오개념 ④: "방정식 \(f = g\)의 실근 증명에 사잇값 정리를 직접 쓴다." → 보조함수 \(h = f - g\)를 정의한 뒤 IVT 적용이 표준 절차.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11닫힌구간 최댓값·최솟값
(1) \(x^2 - 4x + 5\) on \([1, 4]\)  (2) \(\sqrt{x-2}\) on \([3, 6]\)
(1) 꼭짓점 \(x = 2 \in [1, 4]\), \(f(2) = 1\) (최솟). 양 끝 \(f(1) = 2, f(4) = 5\) → 최댓 \(5\). (2) 증가함수, \(f(3) = 1\) (최솟), \(f(6) = 2\) (최댓).
정답: (1) 최댓 \(5\), 최솟 \(1\)  (2) 최댓 \(2\), 최솟 \(1\)
L12IVT로 실근 존재 증명
\(x^3 - x - 1 = 0\)이 \((-1, 2)\)에 적어도 하나의 실근을 가짐을 보이시오.
① \(f(x) = x^3 - x - 1\)은 다항함수 → \([-1, 2]\)에서 연속. ② \(f(-1) = -1 - (-1) - 1 = -1 \lt 0\), \(f(2) = 8 - 2 - 1 = 5 \gt 0\). ③ 부호 변화 \(\Rightarrow\) IVT(\(k = 0\) 적용)에 의해 \(f(c) = 0\)인 \(c \in (-1, 2)\) 적어도 하나 존재.
정답: 위 ①②③을 답안으로 작성. 표준 4단계 (연속 + 두 끝 부호 + IVT 인용).
L23실근 구간 식별 — 부호표
\(2x^3 - 3x^2 - x + 1 = 0\)은 서로 다른 세 실근. \((-1,0), (0,1), (1,2), (2,3)\) 중?
\(f(-1) = -2 - 3 + 1 + 1 = -3\), \(f(0) = 1\), \(f(1) = 2 - 3 - 1 + 1 = -1\), \(f(2) = 16 - 12 - 2 + 1 = 3\), \(f(3) = 54 - 27 - 3 + 1 = 25\). 부호 변화: \((-1,0)\) − → +, \((0,1)\) + → −, \((1,2)\) − → + (3개). \((2,3)\) + → +(없음).
정답: ㄱ, ㄴ, ㄷ. 세 실근이 정확히 이 세 구간에 1개씩 분배됨을 명시.
L24분수함수 최댓값 — 절댓값 분기
\([1, 3]\)에서 \(f(x) = \frac{a}{|x-4|}\)의 최댓값 \(= 8\). 양수 \(a\)?
\([1, 3]\)에서 \(x \lt 4\) → \(|x-4| = 4 - x\). 따라서 \(f(x) = \frac{a}{4 - x}\). \(x\) 증가 → 분모 감소 → \(f\) 증가. 최댓값은 \(x = 3\)에서 \(\frac{a}{1} = a\). \(a = 8\).
정답: \(a = 8\).
L35🚨 오답률 1위 — IVT의 가정·결론 혼동
참인 명제 모두 + 거짓 명제는 반례. ㄱ "연속+부호반전 \(\Rightarrow\) 정확히 하나의 실근" ㄴ "연속+같은 부호 \(\Rightarrow\) 실근 없음" ㄷ "연속+\(k\) 사이 \(\Rightarrow\) \(f(c)=k\)인 \(c\) 존재".
ㄱ (거짓): IVT는 "적어도 하나". 반례 \(f(x) = x^3 - x\), \([-2, 2]\): \(f(-2) = -6, f(2) = 6\) → 부호 다름. 실근 \(x = -1, 0, 1\) 3개. ㄴ (거짓): 같은 부호여도 실근 가능. 반례 \(f(x) = x^2 - 1\), \([-2, 2]\): \(f(\pm 2) = 3 \gt 0\) 같은 부호. 실근 \(x = \pm 1\) 2개. ㄷ (참): IVT의 정확한 진술 그대로.
정답: 참 = ㄷ. ㄱ 반례 \(x^3 - x\), ㄴ 반례 \(x^2 - 1\).
발표 유도: "IVT가 '적어도 하나'라고 진술하는 이유는?"을 학생에게 직접 말하게 한다 — "그래프가 출렁이면 더 많이 만날 수 있기 때문". 추후 미분 단원의 "도함수 부호 → 실근 개수 정확 결정"의 동기.
FC6형성평가 — 사칙으로 연속 보장되는 함수 식별
\(f, g\)가 \(x=a\)에서 연속. ㄱ \(2f - g\)  ㄴ \(\{f\}^2\)  ㄷ \(\frac{f}{g}\)  ㄹ \(\frac{2}{\{f\}^2 + 1}\). 반드시 연속?
ㄱ 실수배 + 차 → 사칙에 의해 항상 연속 ✓. ㄴ \(f \cdot f\) 곱 → 항상 연속 ✓. ㄷ \(g(a) = 0\)일 수 있어 보장 X — 반례 \(g(x) = x\), \(a = 0\). ㄹ 분모 \(\{f\}^2 + 1 \geq 1 \gt 0\) (항상 양수) → 분모 \(\neq 0\) 보장 → 항상 연속 ✓.
정답: ㄱ, ㄴ, ㄹ.
CH1D1⭐ 수능특강 수학Ⅱ 함수의연속 예제 4 — 부호곱 + IVT
\(f(x) = x(x-a)(x-a^2)\), \(f(3)f(6) \lt 0\), \(f(3)f(20) \gt 0\). \(a \in (p, q)\). \(p^2 + q^2\)?
STEP 1 \(f(3), f(6)\) 부호 다름 + \(f(3), f(20)\) 같음 ⇒ \(f(6), f(20)\) 부호 다름 ⇒ \(f(6)f(20) \lt 0\). STEP 2 IVT: \([3,6]\)에 영점 \(c_1\), \([6,20]\)에 영점 \(c_2\) 존재. \(f\)의 실근은 \(\{0, a, a^2\}\). STEP 3 \(a \gt 3\)이면 \(a^2 \gt a\). 따라서 \(a \in (3, 6)\), \(a^2 \in (6, 20)\) — 두 영점이 자연스럽게 배치. STEP 4 \(6 \lt a^2 \lt 20\) ⇒ \(\sqrt{6} \lt a \lt 2\sqrt{5}\). 교집합 \(3 \lt a \lt 2\sqrt{5}\) ⇒ \(p = 3, q = 2\sqrt{5}\). STEP 5 \(p^2 + q^2 = 9 + 20 = 29\).
정답: \(29\). 핵심: 부호의 곱 부호 → 두 구간에 영점 → 삼차의 세 근 \(\{0,a,a^2\}\) 위치 결정.
CH2D2⭐ 1994학년도 수능 1차 11번 — 우함수 × IVT × EVT 종합
조건 (가) 연속 + 우함수, (나) \(|x| \gt 5\)에서 \(f = 0\), (다) \(|x| \lt 5\)에서 \(|f| \leq 10\) 및 \(f = 10\)인 \(x\)가 오직 한 개. 옳지 않은 것은?
① \(f(\pm 5) = 0\): 연속 + (나)에 의해 \(\lim_{x\to 5^+} f = 0 = f(5)\). 우함수로 \(f(-5) = 0\). 옳음. ② 최대 = \(f(0) = 10\): 우함수 + \(f(c) = 10\)이 오직 하나 ⇒ \(c = -c\) ⇒ \(c = 0\). (다)에 의해 \(|f| \leq 10\) ⇒ 최대 \(= 10 = f(0)\). 옳음. ③ \(f(x) = 5\)인 \(x\)가 두 개 이상: \([0,5]\)에서 \(f(0) = 10, f(5) = 0\), 연속 → IVT로 \(f(c_1) = 5\)인 \(c_1 \in (0, 5)\). 우함수 → \(f(-c_1) = 5\). 최소 2개. 옳음. ④ 최솟값을 갖는 \(x\)가 오직 하나? — 만약 최솟값 \(= 0\)이면 \(|x| \geq 5\) 전체에서 \(f = 0\) → 최솟점 무수히 많음. 음수 최솟이라도 우함수 → 짝으로 발생. 옳지 않음. ⑤ \(f(x+5)f(x-5) = 0\): \(x+5\)와 \(x-5\)는 거리 10. \(|x+5| \gt 5\) 또는 \(|x-5| \gt 5\) 중 적어도 하나 성립 → 한쪽이 0 → 곱 0. 옳음.
정답: ④. 핵심: ④는 EVT의 "최댓·최솟 존재"만 보장하지 "유일성"은 보장 안 함 + 우함수 대칭성.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 — 정리의 가정 분석 "EVT·IVT의 가정 '닫힌구간 + 연속'을 그냥 외우다가 직접 반례를 만들어보고 나서 이유가 보였다. \((-1, 2)\)에서 \(x^2\)은 끝점을 안 잡아서 최댓값이 없고, \((0, 1]\)에서 \(1/x\)는 발산해서 최댓값이 없다. '닫힘'은 끝점을 잡아주고 '연속'은 발산을 막는다 — 두 가정은 역할이 다르다는 점이 인상적이었다."
유형 2: 문제해결력 — 실근 존재 증명 루틴 "\(f(x) = g(x)\)의 실근 증명에서 처음에는 무엇을 적어야 할지 헤맸다. '① 보조함수 \(h = f - g\) 정의 → ② \([a, b]\)에서 연속 명시 → ③ \(h(a)h(b) \lt 0\) 보이기 → ④ IVT 인용' 4단계 루틴을 만들어 두니 어떤 형태의 실근 문제도 같은 틀로 풀린다. 7차시 도전 14번도 같은 패턴이었다."
유형 3: 발전 가능성 — 정리의 일반화 "같은 산을 첫째 날 10시에 올라가고 둘째 날 같은 시간에 내려오는 두 사람의 위치 함수가 같아지는 시각이 반드시 존재한다는 고전 문제를 떠올렸다. 두 위치의 차 \(h(t)\)에 IVT를 쓰면 풀린다. 정리가 일상 상황에서도 그대로 쓰인다는 점이 재미있었다."

H 수업 후 메모 (반 별 기록)
