01. 실전 훈련 (Textbook Mastery)

💡 비상 미적분 I 교과서 중단원 마무리(P.121~P.123) 14문항을 그대로 학습지화. 기초 1~3 · 기본 4~12 · 도전 13·14 (서술형 킬러)

[기초-부정적분 (다항식+곱셈전개)]
1. 다음 부정적분을 구하시오. (교과서 P.121 중단원 기초 01번)
(1) \( \displaystyle\int(x^2 - 2x + 3)\,dx \)    (2) \( \displaystyle\int x(x+1)^2\,dx - \int x(x^2 + 1)\,dx \)
🎯 풀이 전략 보기
💡 전략 힌트: (1) 항별 적분 \(\Rightarrow\) \(\tfrac{1}{3}x^3 - x^2 + 3x + C\).
(2) 두 부정적분을 합쳐 한 번에: 피적분 차 \(= x(x+1)^2 - x(x^2+1) = x\cdot[(x+1)^2 - (x^2+1)] = x\cdot 2x = 2x^2\). 답: \(\dfrac{2}{3}x^3 + C\).
[기초-정적분 기본 계산]
2. 다음 정적분의 값을 구하시오. (교과서 P.121 중단원 기초 02번 (1)(2))
(1) \( \displaystyle\int_0^2 (3x^2 + 5)\,dx \)    (2) \( \displaystyle\int_{-2}^{1}(x+2)^2\,dx - \int_{-2}^{1}(2x+1)\,dx \)
🎯 풀이 전략 보기
💡 전략 힌트: (1) \([x^3 + 5x]_0^2 = 8 + 10 = 18\).
(2) 같은 구간이므로 피적분 차로 정리: \((x+2)^2 - (2x + 1) = x^2 + 4x + 4 - 2x - 1 = x^2 + 2x + 3\). \([\tfrac{1}{3}x^3 + x^2 + 3x]_{-2}^{1} = (\tfrac{1}{3} + 1 + 3) - (-\tfrac{8}{3} + 4 - 6) = \tfrac{13}{3} - (-\tfrac{14}{3}) = 9\).
[기초-같은 피적분, 구간 분할 합치기]
3. 정적분 \(\displaystyle\int_{-1}^{1}(2x^3 - 6x)\,dx + \int_1^3 (2x^3 - 6x)\,dx\)의 값을 구하시오. (교과서 P.121 중단원 기초 02번 (3))
🎯 풀이 전략 보기
💡 전략 힌트: 구간 분할 \(\int_{-1}^1 + \int_1^3 = \int_{-1}^3\). \([\tfrac{1}{2}x^4 - 3x^2]_{-1}^{3} = (\tfrac{81}{2} - 27) - (\tfrac{1}{2} - 3) = \tfrac{27}{2} - (-\tfrac{5}{2}) = 16\).
[기본-도함수 일반항 + 두 조건]
4. 함수 \(f(x)\)가 \( f'(x) = 1 + 2x + 3x^2 + \cdots + nx^{n-1} \)을 만족시키고 \(f(0) = 5\), \(f(1) = 11\)일 때, 자연수 \(n\)의 값을 구하시오. (단, \(n \geq 2\)) (교과서 P.122 중단원 기본 03번)
🎯 풀이 전략 보기
💡 전략 힌트: 항별 적분 \(\int k x^{k-1}\,dx = x^k\) \(\Rightarrow\) \(f(x) = x + x^2 + \cdots + x^n + C\). \(f(0) = C = 5\). \(f(1) = n + 5 = 11 \Rightarrow n = 6\).
[기본-접선 기울기 + 통과점 → 함숫값]
5. 점 \((-2, 1)\)을 지나는 곡선 \(y = f(x)\) 위의 임의의 점 \((x, y)\)에서의 접선의 기울기가 \(6x^2 + 12x + 5\)일 때, \(f(-1)\)의 값을 구하시오. (교과서 P.122 중단원 기본 04번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(x) = 6x^2 + 12x + 5\) \(\Rightarrow\) \(f(x) = 2x^3 + 6x^2 + 5x + C\). \(f(-2) = -16 + 24 - 10 + C = 1 \Rightarrow C = 3\). \(f(-1) = -2 + 6 - 5 + 3 = 2\).
[기본-도함수 + 극값 조건으로 함수 결정]
6. 미분가능한 함수 \(f(x)\)에 대하여 \(f'(x) = ax^2 - ax\)이고 함수 \(f(x)\)의 극댓값이 \(6\), 극솟값이 \(5\)일 때, 함수 \(f(x)\)를 구하시오. (단, \(a \gt 0\)) (교과서 P.122 중단원 기본 05번)
🎯 풀이 전략 보기
💡 전략 힌트: \(f'(x) = ax(x-1) = 0 \Rightarrow x = 0\) 극대, \(x = 1\) 극소 (\(a \gt 0\)). \(f(x) = \tfrac{a}{3}x^3 - \tfrac{a}{2}x^2 + C\). \(f(0) = C = 6\). \(f(1) = \tfrac{a}{3} - \tfrac{a}{2} + 6 = -\tfrac{a}{6} + 6 = 5 \Rightarrow a = 6\). 답: \(f(x) = 2x^3 - 3x^2 + 6\).
[기본-부정적분 항등식 + 양변 미분]
7. 미분가능한 함수 \(f(x)\)의 한 부정적분을 \(F(x)\)라고 하면 \( F(x) = x f(x) - 3x^4 + 2x^3 \) 이 성립한다. \(f(1) = 6\)일 때, \(f(2)\)의 값을 구하시오. (교과서 P.122 중단원 기본 06번)
🎯 풀이 전략 보기
💡 전략 힌트: \(F'(x) = f(x)\). 양변 미분: \(f(x) = f(x) + xf'(x) - 12x^3 + 6x^2 \Rightarrow xf'(x) = 12x^3 - 6x^2 \Rightarrow f'(x) = 12x^2 - 6x\). 적분: \(f(x) = 4x^3 - 3x^2 + C\). \(f(1) = 4 - 3 + C = 6 \Rightarrow C = 5\). \(f(2) = 32 - 12 + 5 = 25\).
[기본-정적분 항등식 → \(f(a)\) 결정]
8. 모든 실수 \(x\)에 대하여 다항함수 \(f(x)\)가 \(\displaystyle\int_a^x f(t)\,dt = 6x^2 + x - 1\)을 만족시킬 때, \(f(a)\)의 값을 구하시오. (단, \(a \gt 0\)) (교과서 P.122 중단원 기본 07번)
🎯 풀이 전략 보기
💡 전략 힌트: ① 양변 미분 \(\Rightarrow\) \(f(x) = 12x + 1\). ② \(x = a\) 대입 \(\Rightarrow\) \(0 = 6a^2 + a - 1 = (3a - 1)(2a + 1)\). \(a \gt 0 \Rightarrow a = \tfrac{1}{3}\). \(f(a) = 12 \cdot \tfrac{1}{3} + 1 = 5\).
[기본-정적분의 최솟값 (이차함수)]
9. 정적분 \(\displaystyle\int_0^1 (6a^2 x^2 - 16ax + 5)\,dx\)의 값이 최소가 되도록 하는 실수 \(a\)의 값과 그때의 정적분의 값을 구하시오. (교과서 P.122 중단원 기본 08번)
🎯 풀이 전략 보기
💡 전략 힌트: 적분 \(= [2a^2 x^3 - 8a x^2 + 5x]_0^1 = 2a^2 - 8a + 5 = 2(a - 2)^2 - 3\). 최솟값: \(a = 2\)에서 \(-3\).
[기본-선형결합 연립으로 정적분]
10. 닫힌구간 \([-3, 5]\)에서 연속인 두 함수 \(f(x), g(x)\)가 \[ \int_{-3}^{5}\{f(x) - g(x)\}\,dx = -3,\quad \int_{-3}^{5}\{2f(x) + 3g(x)\}\,dx = 4 \] 를 만족시킬 때, \(\displaystyle\int_{-3}^{5}\{f(x) + g(x)\}\,dx\)의 값을 구하시오. (교과서 P.123 중단원 기본 09번)
🎯 풀이 전략 보기
💡 전략 힌트: \(p = \int_{-3}^{5} f\), \(q = \int_{-3}^{5} g\). \(p - q = -3\), \(2p + 3q = 4\). 연립: \(p = -1, q = 2\). 답: \(p + q = 1\).
[기본-구간 분할로 결합 정적분 계산]
11. 다항함수 \(f(x)\)가 \[ \int_0^1 f(x)\,dx = 5,\quad \int_0^4 f(x)\,dx = 9,\quad \int_{-1}^{1} f(x)\,dx = 3 \] 을 만족시킬 때, 정적분 \(\displaystyle\int_{-1}^{4} f(x)\,dx\)의 값을 구하시오. (교과서 P.123 중단원 기본 10번)
🎯 풀이 전략 보기
💡 전략 힌트: \(\int_{-1}^{4} = \int_{-1}^{1} + \int_{1}^{4}\). \(\int_1^4 = \int_0^4 - \int_0^1 = 9 - 5 = 4\). 합: \(3 + 4 = 7\).
[기본-절댓값 정적분 (구간 분할)]
12. 정적분 \(\displaystyle\int_{-1}^{2}(x^2 - 2|x|)\,dx\)의 값을 구하시오. (교과서 P.123 중단원 기본 11번)
🎯 풀이 전략 보기
💡 전략 힌트: \(x = 0\)에서 부호 변경. \(\int_{-1}^0 (x^2 + 2x)\,dx + \int_0^2 (x^2 - 2x)\,dx = [\tfrac{1}{3}x^3 + x^2]_{-1}^0 + [\tfrac{1}{3}x^3 - x^2]_0^2 = -\tfrac{2}{3} + (-\tfrac{4}{3}) = -2\).
[도전-정적분 항이 식 안에 등장 (서술형)]
13. 함수 \(f(x)\)가 \[ f(x) = 4x^2 + 3x + \int_0^1 t f(t)\,dt \] 를 만족시킬 때, \(f(1)\)의 값을 풀이 과정과 함께 구하시오. (교과서 P.123 중단원 도전 13번)
🎯 서술형 풀이 전략 보기
💡 전략 (5단계):
① 정적분 \(\int_0^1 t f(t)\,dt\)는 \(x\)와 무관한 상수이므로 \(k\)로 둠. \(f(x) = 4x^2 + 3x + k\).
② 정의식 대입: \(k = \int_0^1 t(4t^2 + 3t + k)\,dt = \int_0^1 (4t^3 + 3t^2 + kt)\,dt\).
③ 정적분 계산: \([t^4 + t^3 + \tfrac{k}{2}t^2]_0^1 = 1 + 1 + \tfrac{k}{2} = 2 + \tfrac{k}{2}\).
④ 방정식: \(k = 2 + \tfrac{k}{2} \Rightarrow k = 4\). \(f(x) = 4x^2 + 3x + 4\).
⑤ \(f(1) = 4 + 3 + 4 = 11\).
채점 포인트: 정적분 \(= k\) 치환 / \(k\) 정의식에 \(f(x)\) 대입 / 정적분 정확 계산 / \(k\) 결정 / 최종 \(f(1)\).
[도전-적분 항이 식 안 + 일차함수와 다항함수 (서술형)]
14. 일차함수 \(f(x)\)와 다항함수 \(g(x)\)가 모든 실수 \(x\)에 대하여 \[ x f(x) = g(x) + \int_0^x t f'(t)\,dt,\quad f(x) g(x) = 2x^3 + 9x^2 + 9x \] 를 만족시킬 때, \(f(0) g(1)\)의 값을 풀이 과정과 함께 구하시오. (교과서 P.123 중단원 도전 14번)
🎯 서술형 풀이 전략 보기
💡 전략 (5단계):
① 첫 식 양변을 \(x\)로 미분: 좌변 \((xf(x))' = f(x) + xf'(x)\). 우변 \(g'(x) + xf'(x)\) (적분의 미분 + 곱의 미분 결합). \(\Rightarrow\) \(g'(x) = f(x)\). 즉 \(f\)가 \(g'\).
② 일차함수 \(f(x) = ax + b\)이므로 \(g(x) = \int f(x)\,dx = \tfrac{a}{2}x^2 + bx + C\) (이차함수).
③ 첫 식 \(x = 0\) 대입: \(0 = g(0) + 0 \Rightarrow g(0) = C = 0\). 따라서 \(g(x) = \tfrac{a}{2}x^2 + bx\).
④ 둘째 식 \(f(x)g(x) = (ax + b)\cdot x(\tfrac{a}{2}x + b) = x(ax + b)(\tfrac{a}{2}x + b)\)를 \(2x^3 + 9x^2 + 9x = x(2x^2 + 9x + 9) = x(2x + 3)(x + 3)\)과 일치시키기. 두 인수 \((ax + b), (\tfrac{a}{2}x + b)\)가 \((2x + 3), (x + 3)\)에 대응 \(\Rightarrow\) \(a = 2, b = 3\) (또는 \(a = 1, b = 3\)는 \(\tfrac{a}{2}\) 일치 안 함). 검증: \(a = 2, b = 3\) ⇒ \(f = 2x + 3, g = x^2 + 3x\), \(fg = (2x+3)(x^2+3x) = 2x^3 + 6x^2 + 3x^2 + 9x = 2x^3 + 9x^2 + 9x\) ✓.
⑤ \(f(0) = 3\), \(g(1) = 1 + 3 = 4\). \(f(0)g(1) = 12\).
채점 포인트: 첫 식 양변 미분으로 \(g' = f\) 도출 / 일차함수 가정 + \(g(0) = 0\) 결정 / 둘째 식에서 \(a, b\) 결정 / 검증 / 최종 \(f(0)g(1) = 12\).
