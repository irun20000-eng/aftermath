C 생각 열기 운영 가이드

발문: \(f(x) = x^3 - 3x^2 + 1\)의 그래프를 머릿속에 그려 본 후, 어디서 증가하고 어디서 감소하는가? 도함수 \(f'(x)\)의 부호를 어떻게 활용할 수 있을지 추측.

예상 반응 ①"\(f'(x) = 3x^2 - 6x\)을 계산해 부호를 보면 되겠다."
피드백: 정확한 직관. 한 걸음 더 — "왜 도함수 부호가 본함수 증감을 결정?"을 묻고 평균값 정리(차시 13)와 연결.
예상 반응 ②"증가는 그래프가 올라가는 거고, 감소는 내려가는 거."
피드백: 직관 인정. 그러나 수학적 정의 "\(x_1 \lt x_2 \Rightarrow f(x_1) \lt f(x_2)\)"로 표현해 보자. "구간 단위" 강조.
예상 반응 ③"이차함수 판별식 같은 걸 쓰면 되는 거 같다."
피드백: 좋은 직감. L2 4번에서 정확히 그 도구를 사용한다. 삼차의 도함수는 이차 → "이차의 모든 \(x\)에서 \(\geq 0\)" 조건이 판별식 \(D \leq 0\).

차시 13 → 14 연결: 평균값 정리 \(\dfrac{f(x_2) - f(x_1)}{x_2 - x_1} = f'(c)\)에서 분모 \(x_2 - x_1 \gt 0\). 따라서 분자 부호 = \(f'(c)\) 부호. "\(f'(c) \gt 0\) 모든 \(c\) \(\Rightarrow\) 분자 \(\gt 0\) \(\Rightarrow\) 증가"가 직접 따름.

D 개념 지도 노트 — 핵심 공식 · 판서 시연 · 흔한 오개념

CORE CRITERIA
\( f'(x) \gt 0 \;\Rightarrow\; f \) 증가    \( f'(x) \lt 0 \;\Rightarrow\; f \) 감소     (역은 \(\geq 0\)·\(\leq 0\)까지만 보장, 등호는 고립점에서만)
📌 판서 시연 4단계 — 증감표 작성 루틴 (\(f(x) = x^3 - 3x^2 + 1\)):
  1. \(f'(x) = 3x^2 - 6x = 3x(x - 2)\). \(f'(x) = 0 \Rightarrow x = 0, 2\).
  2. 표 3열 (\(\cdots, 0, \cdots, 2, \cdots\)) 작성. \(f'\) 부호 행 채우기 (\(+, 0, -, 0, +\)).
  3. \(f\) 행에 ↗·↘ 표시. 임계점 함숫값 \(f(0) = 1\), \(f(2) = -3\) 계산.
  4. 증가구간 \((-\infty, 0]\), \([2, \infty)\); 감소구간 \([0, 2]\) 결론.
오개념 ①: "\(f' \gt 0\)이면 증가, \(f \gt 0\)이면 증가" — \(f\)와 \(f'\)을 혼동. \(f'\) 부호가 증감을 결정한다는 점을 빨간 화살표로 칠판에 표시.
오개념 ②: "증가 \(\Leftrightarrow f'(x) \gt 0\) 모든 \(x\)에서" — 거짓. \(f(x) = x^3\)에서 \(f'(0) = 0\)이지만 증가. L3 7번 함정의 핵심.
오개념 ③: "증가·감소가 한 점에서 결정된다." → 한 점이 아니라 구간 단위로 말한다. "구간 안의 임의의 \(x_1, x_2\)" 정의 강조.

E 문항별 해설 카드 (Level 1·2·3 + 형성평가 + 도전 1·2)

L11증감 조사 (삼차·사차 4문항)
(1) \(f = -2x^3 + 3x^2 + 12x - 3\)   (2) \(f = 2x^3 + 6x^2 + 6x + 7\)   (3) \(f = -x^3 - 6x + 1\)   (4) \(f = x^4 - 2x^2 + 5\)
(1) \(f' = -6x^2 + 6x + 12 = -6(x - 2)(x + 1)\). 부호 (외/내/외) \((-,+,-)\). 증가 \([-1, 2]\), 감소 \((-\infty, -1] \cup [2, \infty)\). (2) \(f' = 6x^2 + 12x + 6 = 6(x + 1)^2 \geq 0\) (등호 \(x = -1\)만). 실수 전체 증가. (3) \(f' = -3x^2 - 6 \lt 0\) (항상). 실수 전체 감소. (4) \(f' = 4x^3 - 4x = 4x(x - 1)(x + 1)\). 부호 \((-,+,-,+)\). 증가 \([-1, 0] \cup [1, \infty)\), 감소 \((-\infty, -1] \cup [0, 1]\).
정답: 각 함수의 증감 구간 위와 같음. 판서 포인트: (2)·(3)은 도함수가 단일 부호 → 항상 증가/감소; 도함수 그래프와 \(x\)축의 관계를 시각화.
L12구간 한정 증감
\(f(x) = x^3 - 12x + 5\), 구간 \([-2, 2]\)에서 증감.
\(f'(x) = 3x^2 - 12 = 3(x - 2)(x + 2)\). \(f' = 0 \Rightarrow x = \pm 2\). 구간 내부 \(-2 \lt x \lt 2\)에서 \(f'(x) \lt 0\) 일관 → 감소.
정답: \([-2, 2]\)에서 \(f\)는 감소. 증가 구간 없음.
L13전하량 감소 시각 (수학+과학)
\(Q(t) = t^3 - 6t^2 + 9t + 2\), \(t \geq 0\). 감소하는 \(t\) 범위.
\(Q'(t) = 3t^2 - 12t + 9 = 3(t - 1)(t - 3)\). 감소 \(\Leftrightarrow Q'(t) \lt 0 \Leftrightarrow 1 \lt t \lt 3\).
정답: \(1 \lt t \lt 3\). 해석: 전하량이 감소하는 동안 전류 방향이 반대.
L24실수 전체에서 증가 — 판별식
\(f(x) = x^3 + ax^2 + 3x + 1\)이 실수 전체에서 증가할 \(a\) 범위.
\(f'(x) = 3x^2 + 2ax + 3\). 실수 전체 증가 \(\Leftrightarrow f'(x) \geq 0\) 모든 \(x\). \(f'\)의 최고차계수 \(3 \gt 0\) (아래로 볼록) → 판별식 \(D \leq 0\). \(D/4 = a^2 - 9 \leq 0 \Rightarrow -3 \leq a \leq 3\).
정답: \(-3 \leq a \leq 3\). 핵심: 삼차의 \(f'\)은 이차 → "모든 \(x\)에서 \(\geq 0\)"은 최고차 \(\gt 0\) + \(D \leq 0\) 결합.
L25감소 구간 존재하지 않을 조건 함정
\(f(x) = 2x^3 - 9x^2 + 12x + k\)가 \([0, 3]\)에서 감소 구간 없도록 하는 \(k\).
\(f'(x) = 6x^2 - 18x + 12 = 6(x - 1)(x - 2)\). \(k\)와 무관. \([0, 3]\) 내부 \(1 \lt x \lt 2\)에서 \(f'(x) \lt 0\) → 감소 구간 반드시 존재. \(k\) 값과 무관.
정답: 그러한 \(k\)는 존재하지 않는다. 함정: 학생들은 "\(k\) 범위가 있다"고 가정. 도함수 부호는 평행이동 \(k\)에 무관함을 강조.
L26도함수 그래프 → 본함수 증감
그림에서 \(y = f'(x)\) (우상향 직선, \(x = a\)에서 \(x\)축 통과), \(y = g'(x)\) (위로 볼록 포물선, \(x = b, c\)에서 \(x\)축 통과)의 증감.
\(f'\): \(x \lt a\)에서 \(-\), \(x \gt a\)에서 \(+\). → \(f\)는 \((-\infty, a]\) 감소, \([a, \infty)\) 증가. \(g'\): 위로 볼록 포물선, \(b \lt x \lt c\)에서 \(+\), 그 밖에서 \(-\). → \(g\)는 \([b, c]\) 증가, 밖에서 감소.
정답: 위와 같음. 핵심: 함수 그래프가 아니라 도함수 그래프 — \(y = 0\)인 점과 \(y\)의 부호만 읽으면 본함수 증감 즉시 결정.
L37🚨 오답률 1위 — "증가 \(\Rightarrow f'(x) \gt 0\)?" 학생 주장
📢 학생: "\((a, b)\)에서 \(f\)가 증가하면 \(f'(x) \gt 0\)이야!" 참/거짓 판정.
STEP 1: 결론 — 거짓. 반례 \(f(x) = x^3\)은 실수 전체에서 증가하지만 \(f'(0) = 0\). STEP 2: 정확한 명제 ① \(f'(x) \gt 0 \Rightarrow\) 증가 (참, 충분조건) ② 증가 \(\Rightarrow f'(x) \geq 0\) (참, 필요조건) ③ 증가 \(\Rightarrow f'(x) \gt 0\) 모든 \(x\) (거짓). STEP 3: 정확한 동치: "증가 \(\Leftrightarrow f'(x) \geq 0\)이고 등호는 고립점에서만 성립".
정답: 학생의 말은 거짓. 정확히는 "\(f'(x) \geq 0\)이고 \(f'(x) = 0\)은 고립점". 핵심: 충분조건 vs 필요조건 구분 — 차시 15에서 "\(f'(a) = 0\)이지만 극값 아님(\(x^3\))" 함정으로 직결.
발표 유도: "학생 말이 맞다고 생각한 학생 손!" → \(x^3\) 그래프를 칠판에 그려 \(f'(0) = 0\)을 직접 보인 후 "그래도 증가한다"를 한 학생이 정의로 증명하게 한다.
FC8형성평가 — 매개변수 함정
\(f(x) = x^3 - 3ax^2 + 3(a^2 - 1)x + 1\)이 실수 전체에서 증가할 \(a\) 범위.
\(f'(x) = 3x^2 - 6ax + 3(a^2 - 1) = 3(x^2 - 2ax + a^2 - 1)\). 실수 전체 증가 \(\Leftrightarrow D \leq 0\). \(D/4 = a^2 - (a^2 - 1) = 1 \gt 0\) — 모든 \(a\)에 대해 양수. 결론: 어떤 \(a\)에 대해서도 \(f'(x) \geq 0\)이 모든 \(x\)에서 성립하지 않음. 따라서 그러한 \(a\)는 존재하지 않음.
정답: 존재하지 않는다. 의미: 매개변수 형태가 항상 "유효 \(a\) 범위"를 주는 건 아님. 판별식이 양수 일정이면 도함수에 항상 음의 구간 존재.
CH1D1⭐ 수능특강 도함수의 활용 1 유제 5 [26009-0079]
\(f(x) = x^3 - ax^2 + 4ax\), \(x_1 \lt x_2\)인 모든 실수에 대해 \((x_1 - x_2)(f(x_1) - f(x_2)) \gt 0\)을 만족시키는 정수 \(a\)의 개수. ① 11 ② 12 ③ 13 ④ 14 ⑤ 15.
STEP 1: 조건 \((x_1 - x_2)(f(x_1) - f(x_2)) \gt 0\) → \(x_1 \lt x_2\)에서 \(x_1 - x_2 \lt 0\) → \(f(x_1) - f(x_2) \lt 0\) → \(f\) 실수 전체 증가. STEP 2: \(f'(x) = 3x^2 - 2ax + 4a \geq 0\) 모든 \(x\) \(\Leftrightarrow\) 판별식 \(\leq 0\). STEP 3: \(D/4 = a^2 - 12a \leq 0 \Rightarrow a(a - 12) \leq 0 \Rightarrow 0 \leq a \leq 12\). STEP 4: 정수 \(a \in \{0, 1, 2, \ldots, 12\}\) → 13개.
정답: ③ 13. 핵심: \((x_1 - x_2)(f(x_1) - f(x_2)) \gt 0\)는 "증가함수"의 동치 표현. 다항함수 실수 전체 증가 \(\Leftrightarrow f'(x) \geq 0\) \(\Leftrightarrow\) 이차 \(f'\)이면 \(D \leq 0\).
CH2D2⭐ 2003년 6월 모평 가/나형 11번 [3점]
\(f(x) = x^3 + (a - 1)x^2 + (a - 1)x + 1\)이 극값을 갖지 않을 \(a\) 범위. ① \(-1 \leq a \leq 2\) ② \(0 \leq a \leq 3\) ③ \(1 \leq a \leq 4\) ④ \(2 \leq a \leq 5\) ⑤ \(3 \leq a \leq 6\).
STEP 1: "극값 없음" \(\Leftrightarrow\) \(f'(x)\)가 부호 변화 없음 \(\Leftrightarrow f'(x) \geq 0\) 모든 \(x\). STEP 2: \(f'(x) = 3x^2 + 2(a - 1)x + (a - 1)\). STEP 3: \(D/4 = (a - 1)^2 - 3(a - 1) = (a - 1)\{(a - 1) - 3\} = (a - 1)(a - 4) \leq 0\) \(\Rightarrow 1 \leq a \leq 4\). STEP 4 검증: \(a = 1\)에서 \(f'(x) = 3x^2 \geq 0\) (등호 \(x = 0\)만), 극값 없음 ✓. \(a = 4\)에서 \(f'(x) = 3(x + 1)^2 \geq 0\) (등호 \(x = -1\)만), 극값 없음 ✓. 경계 포함.
정답: ③ \(1 \leq a \leq 4\). 핵심: 본 차시의 표준 매개변수 응용. "극값 없음" 등호 포함 처리(중근에서 부호 변화 없음)가 함정. 차시 15에서 "극값 있음 \(\Leftrightarrow D \gt 0\)" (등호 제외)과 정확히 반대.

G 성찰지 운영 — 3유형 우수 답안 예

유형 1: 지식의 내면화 (평균값 정리 → 증감 판정) "'\(f' \gt 0\)이면 증가'를 공식으로 외웠는데, 평균값 정리 \(\dfrac{f(x_2) - f(x_1)}{x_2 - x_1} = f'(c)\)를 보니 분모가 양수라 분자 부호와 \(f'(c)\) 부호가 같다는 게 보였다. 증감 판정이 평균값 정리의 자연스러운 따름이라는 걸 알고 단원 II-2의 정리들이 한 줄기로 이어진다는 게 느껴졌다."
유형 2: 문제해결력 (충분조건 vs 필요조건) "'증가 \(\Leftrightarrow f' \gt 0\)' 인 줄 알았다가 \(f(x) = x^3\) 반례에서 깨졌다. 정확히는 '증가 \(\Leftrightarrow f' \geq 0\)이고 등호는 고립점만'. 다음 차시 '\(f' = 0\)이라고 다 극값은 아니다'로 자연스럽게 이어질 것 같다."
유형 3: 발전 가능성 (증감표 로드맵) "증감표가 단순한 표가 아니라 단원 II-2 전체에서 공통으로 쓰는 작업판이라는 게 보였다. '증감표 → 극값 → 그래프 → 실근의 개수·부등식 증명'으로 이어지는 로드맵을 노트 첫 장에 그려두니, 다음 차시들이 새로운 도구가 아니라 같은 표를 다른 각도에서 읽는 일일 것 같았다."
